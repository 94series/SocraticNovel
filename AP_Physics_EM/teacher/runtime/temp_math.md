### 6. 数学公式处理

Claude Code 运行在终端中，无法渲染 LaTeX。采用混合方案：

- 简单公式直接用 ASCII/Unicode 写在对话里：`F = kq₁q₂/r²`、`E = kQ/r²`、`∮E·dA = Q/ε₀`
  - 上下标用 Unicode：²³⁴ⁿ ₀₁₂₃ₙ
  - 希腊字母直接写：ε₀、μ₀、π、Φ、θ
  - 向量用箭头或粗体标注：E⃗ 或 **E**
  - 积分、求和等用文字符号：∫、∮、Σ、∂、∇
- 复杂推导（多步推导、矩阵、长公式链）写入 `teacher/runtime/temp_math.md`，用 LaTeX 语法（`$...$` 和 `$$...$$`），在对话中提示学习者"推导写在 temp_math.md 里了，用 VS Code 打开看（Cmd+Shift+V 预览）"
- **严禁在对话中直接输出 LaTeX 语法**（`$...$`、`$$...$$`、`\frac`、`\int` 等），终端无法渲染，只会显示为乱码。所有公式必须用上述 ASCII/Unicode 方式或写入 temp_math.md
- 临时文件在下次课开头自动清理
