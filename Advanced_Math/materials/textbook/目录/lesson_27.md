# 第27课：二阶常系数齐次ODE——振荡与衰减

---

> *在膜科学家的日常里，有两个无处不在的物理过程：扩散和反应。扩散让物质从高浓度走向低浓度，反应让物质消失或变成别的东西。当这两件事同时发生在一张膜里时，浓度分布 $c(x)$ 满足的不再是一阶方程，而是一个二阶方程——因为扩散本身就涉及浓度的二阶导数。*
>
> *今天，我们要学会解这类方程。而你将看到一个令人目眩的魔术：一个关于导数的微分方程，被变成了一个关于数字的代数方程。微积分被还原为初中数学。这个魔术的名字叫做"特征方程"，它之所以能成功，完全是因为一个字——**线性**。*

## 一、两个物理场景：从膜里走出来的二阶ODE

### 1.1 场景一：稳态扩散——最简单的二阶方程

让我们从你最熟悉的情境开始。一张平面膜，左侧浓度 $c(0) = c_L$，右侧浓度 $c(L) = c_R$。没有化学反应，只有扩散。在稳态（浓度不再随时间变化）下，Fick 第二定律退化为：

$$
D\frac{d^2c}{dx^2} = 0
$$

$D > 0$ 是扩散系数，可以除掉。你面对的是：

$$
\frac{d^2c}{dx^2} = 0
$$

这是一个二阶ODE——但极其简单。它说的是：$c$ 的二阶导数为零。什么函数的二阶导数恒为零？

二阶导数为零意味着一阶导数是常数。一阶导数是常数意味着函数是……**线性函数**。

$$
c(x) = Ax + B
$$

代入边界条件 $c(0) = c_L$ 和 $c(L) = c_R$：

$$
B = c_L, \quad AL + c_L = c_R \implies A = \frac{c_R - c_L}{L}
$$

所以：

$$
c(x) = c_L + \frac{c_R - c_L}{L}\,x
$$

线性浓度分布——这就是第3课里 Fick 第一定律的老朋友。没有反应、纯扩散的膜内浓度分布就是一条直线。

到此为止，没什么新鲜的。但现在让我们把事情弄得更有趣。

### 1.2 场景二：扩散-反应方程——当膜里有化学反应

假设膜不仅允许物质通过，还在膜内发生化学反应——比如催化膜中的催化降解、酶膜中的酶促反应、或生物膜中的微生物代谢。如果反应是一级的（反应速率正比于浓度），那么稳态时：

$$
D\frac{d^2c}{dx^2} - kc = 0
$$

左边第一项是扩散引起的浓度变化（Fick 第二定律），第二项是反应引起的消耗（一级反应速率 $kc$，$k > 0$）。稳态意味着扩散的"补给"和反应的"消耗"精确平衡。

把它整理一下，除以 $D$：

$$
\frac{d^2c}{dx^2} - \frac{k}{D}\,c = 0
$$

令 $\phi^2 = k/D$（这个 $\phi$ 叫做 **Thiele 模数**，在催化和膜反应工程中你会反复遇到它）：

$$
\frac{d^2c}{dx^2} - \phi^2 c = 0
$$

这就不是 $d^2c/dx^2 = 0$ 那样的"空方程"了。它有一个非零的 $c$ 项。**二阶导数等于函数本身乘以一个正常数。** 什么函数有这种性质？

你可能想到了指数函数：$(e^{\phi x})'' = \phi^2 e^{\phi x}$。对。但 $(e^{-\phi x})'' = \phi^2 e^{-\phi x}$ 也行。所以有两个解。事实上，通解是：

$$
c(x) = C_1 e^{\phi x} + C_2 e^{-\phi x}
$$

（或者等价地写成 $c(x) = A\cosh(\phi x) + B\sinh(\phi x)$，用双曲函数有时更方便处理对称边界条件。）

但我们不是靠"猜"得到这个结论的——至少不应该仅仅靠猜。下面，我将带你看到一个系统性的方法，它能处理所有二阶常系数齐次ODE，不管你是否猜得到。

## 二、"如果指数函数是万能的，那就让我们试试看"

### 2.1 一般形式

我们要解的方程的一般形式是：

$$
ay'' + by' + cy = 0
$$

其中 $a$、$b$、$c$ 是常数，$a \neq 0$。自变量可以是 $x$（空间）或 $t$（时间），为了统一，我们暂时用 $y$ 和 $x$。

关键观察：这是一个**线性**方程。"线性"意味着：如果 $y_1$ 和 $y_2$ 都是解，那么 $c_1 y_1 + c_2 y_2$ 也是解（叠加原理）。这一条性质将贯穿整节课——请把它刻在脑子里。

> **叠加原理的证明**（极短）：设 $y_1$ 和 $y_2$ 都满足 $ay'' + by' + cy = 0$。令 $y = c_1 y_1 + c_2 y_2$，则：
>
> $$
> ay'' + by' + cy = a(c_1 y_1'' + c_2 y_2'') + b(c_1 y_1' + c_2 y_2') + c(c_1 y_1 + c_2 y_2)
> $$
>
> $$
> = c_1(ay_1'' + by_1' + cy_1) + c_2(ay_2'' + by_2' + cy_2) = c_1 \cdot 0 + c_2 \cdot 0 = 0 \quad \blacksquare
> $$
>
> 证明中唯一用到的性质是：导数的线性性（$(c_1 f + c_2 g)' = c_1 f' + c_2 g'$）和方程右端为零。

### 2.2 大胆的猜测

现在，让我们像一个物理学家那样思考。我们要找一个函数 $y(x)$，它的二阶导数、一阶导数和自身的某个线性组合恰好为零。

什么函数求导之后"形式不变"？**指数函数 $e^{rx}$。**

$$
y = e^{rx} \implies y' = re^{rx} \implies y'' = r^2 e^{rx}
$$

每次求导，都只是在前面多乘一个 $r$。函数的"骨架"$e^{rx}$ 始终不变。

这就是关键：如果我们把 $y = e^{rx}$ 代入 $ay'' + by' + cy = 0$，所有项都有公因子 $e^{rx}$，可以消掉——留下来的只是一个关于 $r$ 的代数方程。

让我们实际做一遍。代入：

$$
ar^2 e^{rx} + bre^{rx} + ce^{rx} = 0
$$

提取公因子 $e^{rx}$：

$$
e^{rx}(ar^2 + br + c) = 0
$$

因为 $e^{rx} > 0$ 对所有 $x$ 和所有实数 $r$ 成立，所以 $e^{rx}$ **永远不为零**。唯一的可能是：

$$
\boxed{ar^2 + br + c = 0}
$$

这就是**特征方程**（characteristic equation，也叫辅助方程 auxiliary equation）。

停下来感受一秒。

你刚才做了一件了不起的事情：**把一个微分方程变成了一个二次方程。** 微积分消失了。无穷小消失了。极限消失了。剩下的只是 $a$、$b$、$c$ 和一元二次方程的求根公式：

$$
r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

这正是"线性"的威力。因为方程是线性的，指数函数代入后完美地"分解"了微分运算，把导数翻译成了乘法——$d/dx$ 变成了 $\times r$，$d^2/dx^2$ 变成了 $\times r^2$。整个微分方程变成了一个多项式方程。

> **一个深刻的类比**：你也许在线性代数里学过特征值问题 $A\mathbf{v} = \lambda\mathbf{v}$——一个矩阵乘以一个向量等于一个标量乘以那个向量。在那里，你也是把一个"运算"（矩阵乘法）转化为一个"缩放"（标量乘法），然后解一个多项式方程 $\det(A - \lambda I) = 0$。这不是巧合。ODE的特征方程和线性代数的特征方程是**同一个思想**在不同语境下的表达。微分算子 $aD^2 + bD + cI$ 作用于 $e^{rx}$，就像矩阵 $A$ 作用于特征向量——结果只是函数/向量的标量倍。当这个"标量倍"恰好为零时，你就找到了齐次方程的解。

### 2.3 三种情况

一元二次方程有三种根的情况，取决于判别式 $\Delta = b^2 - 4ac$：

| 判别式 | 根的类型 | ODE通解的形式 |
|---|---|---|
| $\Delta > 0$ | 两个不相等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $\Delta = 0$ | 一个重根 $r$ | $y = C_1 e^{rx} + C_2 x e^{rx}$ |
| $\Delta < 0$ | 共轭复根 $\alpha \pm i\beta$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

每一种情况都有自己独特的物理意义和图形性格。让我们逐一深入。

## 三、情况一：两个不相等实根——指数的两面

### 3.1 通解结构

当 $\Delta = b^2 - 4ac > 0$ 时，特征方程有两个不相等的实根 $r_1$ 和 $r_2$。那么 $y_1 = e^{r_1 x}$ 和 $y_2 = e^{r_2 x}$ 都是解，而且它们**线性无关**——$e^{r_2 x}$ 不是 $e^{r_1 x}$ 的常数倍（因为 $r_1 \neq r_2$）。

由叠加原理和通解定理（二阶线性齐次ODE的通解恰好包含两个任意常数），通解为：

$$
y = C_1 e^{r_1 x} + C_2 e^{r_2 x}
$$

### 3.2 回到扩散-反应方程

我们的扩散-反应方程 $c'' - \phi^2 c = 0$ 属于这种情况。这里 $a = 1$，$b = 0$，$c = -\phi^2$。特征方程：

$$
r^2 - \phi^2 = 0 \implies r = \pm\phi
$$

两个不相等实根：$r_1 = \phi$，$r_2 = -\phi$。通解：

$$
c(x) = C_1 e^{\phi x} + C_2 e^{-\phi x}
$$

物理含义：$e^{\phi x}$ 是一个从左到右指数增长的模式，$e^{-\phi x}$ 是一个指数衰减的模式。实际的浓度分布是这两个模式的叠加，具体的 $C_1$、$C_2$ 由边界条件决定。

**具体例子**：设 $c(0) = c_0$，$c(L) = 0$（膜左侧浓度为 $c_0$，右侧浓度为零——完全被反应消耗或被冲走）。

代入边界条件：

$$
c(0) = C_1 + C_2 = c_0
$$

$$
c(L) = C_1 e^{\phi L} + C_2 e^{-\phi L} = 0
$$

从第二个方程：$C_1 = -C_2 e^{-2\phi L}$。代入第一个方程解出 $C_1$ 和 $C_2$——代数有点繁琐，但用双曲函数会优雅得多。令 $c(x) = A\sinh(\phi(L-x))/\sinh(\phi L)$（你可以验证这满足两个边界条件），其中 $A = c_0$。

$$
c(x) = c_0 \frac{\sinh[\phi(L-x)]}{\sinh(\phi L)}
$$

### 3.3 又一个精讲例题

**例题 1.** 解 $y'' + y' - 6y = 0$。

**解.** 特征方程：

$$
r^2 + r - 6 = 0
$$

因式分解：$(r+3)(r-2) = 0$，根为 $r_1 = 2$，$r_2 = -3$。

通解：

$$
y = C_1 e^{2x} + C_2 e^{-3x}
$$

两个指数项一增一减。如果配上初始条件 $y(0) = 2$，$y'(0) = 3$：

$$
y(0) = C_1 + C_2 = 2
$$

$$
y'(0) = 2C_1 - 3C_2 = 3
$$

第一个方程给出 $C_2 = 2 - C_1$，代入第二个：

$$
2C_1 - 3(2 - C_1) = 3 \implies 5C_1 = 9 \implies C_1 = \frac{9}{5}, \quad C_2 = \frac{1}{5}
$$

$$
y = \frac{9}{5}e^{2x} + \frac{1}{5}e^{-3x}
$$

长期行为：$e^{2x}$ 项增长远快于 $e^{-3x}$ 项衰减，所以 $y \to +\infty$（$x \to \infty$）。$e^{-3x}$ 项在短时间内迅速"蒸发"，之后解几乎只剩下 $\frac{9}{5}e^{2x}$。

## 四、情况二：重根——临界阻尼的微妙

### 4.1 重根的出现

当 $\Delta = b^2 - 4ac = 0$ 时，特征方程只有一个根：

$$
r = \frac{-b}{2a}
$$

我们找到了一个解 $y_1 = e^{rx}$。但二阶ODE的通解需要**两个**线性无关的解。$e^{rx}$ 只给了我们一个。我们需要第二个。

再试一次 $e^{rx}$？不行——那只是 $y_1$ 的常数倍。

试 $xe^{rx}$ 如何？

### 4.2 验证 $xe^{rx}$ 是第二个解

令 $y_2 = xe^{rx}$。计算：

$$
y_2' = e^{rx} + rxe^{rx} = (1 + rx)e^{rx}
$$

$$
y_2'' = re^{rx} + r(1+rx)e^{rx} = (2r + r^2 x)e^{rx}
$$

代入 $ay'' + by' + cy = 0$：

$$
a(2r + r^2 x)e^{rx} + b(1 + rx)e^{rx} + cxe^{rx} = 0
$$

提取 $e^{rx}$：

$$
e^{rx}[(ar^2 + br + c)x + (2ar + b)] = 0
$$

现在用我们的条件。因为 $r$ 是特征方程 $ar^2 + br + c = 0$ 的根，所以第一个括号 $ar^2 + br + c = 0$。又因为 $r$ 是重根，$\Delta = 0$，此时 $r = -b/(2a)$，所以 $2ar + b = 2a \cdot \frac{-b}{2a} + b = 0$。

两项都为零！$y_2 = xe^{rx}$ 确实是解。

而 $y_1 = e^{rx}$ 和 $y_2 = xe^{rx}$ 是线性无关的——$xe^{rx}$ 不是 $e^{rx}$ 的常数倍（它们的比值 $x$ 不是常数）。

通解：

$$
\boxed{y = C_1 e^{rx} + C_2 xe^{rx} = (C_1 + C_2 x)e^{rx}}
$$

### 4.3 物理对应：临界阻尼

在弹簧-阻尼器系统中（后面的课会详细讲），重根对应**临界阻尼**——阻尼力恰好使得系统"刚好"不振荡就回到平衡位置。过阻尼（两个不等实根，阻尼太强）回得慢，欠阻尼（复根，阻尼不够）会振荡。临界阻尼是回到平衡最快的非振荡方式。

### 4.4 精讲例题

**例题 2.** 解 $y'' + 6y' + 9y = 0$，$y(0) = 2$，$y'(0) = 1$。

**解.** 特征方程：

$$
r^2 + 6r + 9 = 0 \implies (r+3)^2 = 0 \implies r = -3 \text{（重根）}
$$

通解：$y = (C_1 + C_2 x)e^{-3x}$。

代入初始条件。$y(0) = C_1 = 2$。

$y' = C_2 e^{-3x} + (C_1 + C_2 x)(-3)e^{-3x} = (C_2 - 3C_1 - 3C_2 x)e^{-3x}$

$y'(0) = C_2 - 3C_1 = C_2 - 6 = 1 \implies C_2 = 7$

$$
y = (2 + 7x)e^{-3x}
$$

这个解在 $x = 0$ 时为 $2$，初始增长率为 $1$（正的——解先上升！），然后指数衰减把它拉回来。因子 $(2 + 7x)$ 始终为正（对 $x \geq 0$），所以解从不穿越零轴——它慢慢抬头，然后被 $e^{-3x}$ 压回零。

## 五、情况三：共轭复根——振荡来了

### 5.1 复根的出现

当 $\Delta = b^2 - 4ac < 0$ 时，求根公式中出现了负数的平方根。根变成了**共轭复数**：

$$
r_1 = \alpha + i\beta, \quad r_2 = \alpha - i\beta
$$

其中

$$
\alpha = \frac{-b}{2a}, \quad \beta = \frac{\sqrt{4ac - b^2}}{2a}
$$

### 5.2 从复指数到实函数：Euler 公式登场

如果我们天真地写下通解 $y = C_1 e^{(\alpha+i\beta)x} + C_2 e^{(\alpha-i\beta)x}$，它包含复数——但我们的物理量（浓度、位移、温度）是实数。我们需要把复指数转化为实函数。

这正是**Euler 公式**大显身手的地方。回忆第23课：

$$
e^{i\theta} = \cos\theta + i\sin\theta
$$

因此：

$$
e^{(\alpha + i\beta)x} = e^{\alpha x} \cdot e^{i\beta x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)
$$

$$
e^{(\alpha - i\beta)x} = e^{\alpha x} \cdot e^{-i\beta x} = e^{\alpha x}(\cos\beta x - i\sin\beta x)
$$

将这两个表达式相加和相减，就能分离出实部和虚部：

$$
e^{(\alpha+i\beta)x} + e^{(\alpha-i\beta)x} = 2e^{\alpha x}\cos\beta x
$$

$$
e^{(\alpha+i\beta)x} - e^{(\alpha-i\beta)x} = 2ie^{\alpha x}\sin\beta x
$$

因为原方程是线性的，解的任意线性组合还是解。所以 $e^{\alpha x}\cos\beta x$ 和 $e^{\alpha x}\sin\beta x$ 都是实值解，而且它们线性无关。

通解（以实函数形式）：

$$
\boxed{y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)}
$$

请你停下来欣赏一秒。这个结果说的是：解是一个**振荡函数**（$\cos$ 和 $\sin$ 的线性组合）乘以一个**指数包络**（$e^{\alpha x}$）。

- 如果 $\alpha < 0$：振荡逐渐衰减——**衰减振荡**（damped oscillation）。
- 如果 $\alpha = 0$：永恒的等幅振荡——**简谐运动**。
- 如果 $\alpha > 0$：振荡幅度逐渐增大——**发散振荡**。

### 5.4 精讲例题

**例题 3.** 解 $y'' - 6y' + 13y = 0$。

**解.** 特征方程：

$$
r^2 - 6r + 13 = 0
$$

判别式 $\Delta = 36 - 52 = -16 < 0$。用求根公式：

$$
r = \frac{6 \pm \sqrt{-16}}{2} = \frac{6 \pm 4i}{2} = 3 \pm 2i
$$

所以 $\alpha = 3$，$\beta = 2$。通解：

$$
y = e^{3x}(C_1 \cos 2x + C_2 \sin 2x)
$$

注意这里 $\alpha = 3 > 0$——振荡幅度随 $x$ 指数**增大**。这在物理上对应着一个不稳定系统。

**例题 4.** 解 $y'' + 4y = 0$，$y(0) = 3$，$y'(0) = -2$。

**解.** 特征方程：$r^2 + 4 = 0$，根为 $r = \pm 2i$。

这里 $\alpha = 0$，$\beta = 2$。通解：

$$
y = C_1 \cos 2x + C_2 \sin 2x
$$

没有指数包络——纯振荡。这是**简谐运动**。

代入初始条件：$y(0) = C_1 = 3$。

$y' = -2C_1 \sin 2x + 2C_2 \cos 2x$，$y'(0) = 2C_2 = -2$，所以 $C_2 = -1$。

$$
y = 3\cos 2x - \sin 2x
$$

这是一个周期函数，周期 $T = 2\pi/\beta = \pi$，永远不衰减。

### 5.5 膜科学中的"振荡"在哪里？

你可能会问：膜里的浓度会振荡吗？

在纯稳态扩散-反应问题中，通常不会——因为边界条件和物理约束会选择出非振荡的解。但在以下情况中，你确实会遇到复根和振荡行为：

1. **膜过程的动态响应**：当你突然改变操作条件（如进料浓度阶跃变化），膜系统的暂态响应可以表现为衰减振荡——通量先冲过稳态值，再回落，如此反复，最终稳定。

2. **耦合传质-传热问题**：在某些热驱动膜过程（如膜蒸馏）中，温度和浓度的耦合方程可以产生振荡模式。

3. **反应-扩散系统中的 Turing 模式**：在特定条件下，反应-扩散方程的解可以呈现空间周期性——这就是著名的 **Turing 不稳定性**，图灵用它解释了生物体表面斑纹的形成。膜表面的某些周期性沉积模式也被怀疑与此有关。

所以，即使你现在的研究只涉及指数型解，理解振荡型解仍然是必要的——因为你不知道什么时候判别式会落到零的另一边。

## 六、三种情况的统一视角

让我们从高处回望这三种情况。它们真的是"三种不同的东西"吗？

不。它们是同一个结构的三副面孔。

回到特征方程 $ar^2 + br + c = 0$。无论判别式的符号如何，根**始终存在**（在复数域中）。区别只在于：

- $\Delta > 0$：两个实根分开站着。解的空间被两个独立的指数方向张开。
- $\Delta = 0$：两个实根合并了。一个指数方向"塌缩"了。系统用 $xe^{rx}$ 来弥补缺失的维度——就像一个被压扁的坐标轴需要另一种基底来重新张开二维空间。
- $\Delta < 0$：根跑进了复数平面。但 Euler 公式把复指数翻译成了正弦和余弦——实数世界的振荡就是复数世界的圆周运动的"投影"。

三种情况不是三个无关的配方。它们是一个连续谱系的三个区段，由判别式 $\Delta$ 的符号划分。

## 七、边界值问题 vs. 初值问题

在到目前为止的例题中，我们用的都是**初值条件**：在同一个点（通常是 $x = 0$ 或 $t = 0$）给出 $y$ 和 $y'$ 的值。

但在膜科学中，更常见的是**边界值问题**（BVP）：在空间的两个端点给出条件。比如扩散-反应方程中的 $c(0) = c_0$，$c(L) = 0$。

关键区别：**初值问题在（合理条件下）总有唯一解。边界值问题不一定。**

这一点值得你记住。初值问题就像推一个球——给定起点和初始速度，轨迹唯一确定。边界值问题像射箭——给定起点和终点，中间的路径不一定存在，或者可能有多条。

**例题 5.** 解边界值问题 $y'' + \pi^2 y = 0$，$y(0) = 0$，$y(1) = 0$。

**解.** 特征方程 $r^2 + \pi^2 = 0$，根为 $r = \pm i\pi$。$\alpha = 0$，$\beta = \pi$。

通解：$y = C_1 \cos\pi x + C_2 \sin\pi x$。

$y(0) = C_1 = 0$。$y(1) = C_2 \sin\pi = 0$。

但 $\sin\pi = 0$，所以 $C_2$ **可以是任意值**！这意味着方程有无穷多个解：

$$
y = C_2 \sin\pi x, \quad C_2 \text{ 任意}
$$

这不是"出错了"——这是边界值问题的一个深刻特征。在某些特殊的参数值（这里是 $\pi^2$）下，齐次BVP有非平凡解。这些特殊值叫做**特征值**，对应的非平凡解叫做**特征函数**。你将在偏微分方程（PDE）和 Sturm-Liouville 理论中大量遇到它们——那是膜传质建模中高级方法的基石。

## 八、本课核心总结

让我把今天走过的路浓缩成一张表。

**解 $ay'' + by' + cy = 0$ 的步骤：**

1. 写出**特征方程** $ar^2 + br + c = 0$。
2. 计算**判别式** $\Delta = b^2 - 4ac$。
3. 根据 $\Delta$ 的符号选择通解形式：

| $\Delta$ | 根 | 通解 | 图形性格 |
|---|---|---|---|
| $> 0$ | $r_1 \neq r_2$（实） | $C_1 e^{r_1 x} + C_2 e^{r_2 x}$ | 两个指数的叠加 |
| $= 0$ | $r$（重根） | $(C_1 + C_2 x)e^{rx}$ | 临界——恰好不振荡 |
| $< 0$ | $\alpha \pm i\beta$ | $e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ | 指数包络内的振荡 |

4. 用**初始条件或边界条件**确定 $C_1$、$C_2$。

**核心思想**：特征方程把微分方程变成了代数方程。这是"线性"赋予我们的礼物——因为线性方程允许指数函数的"形式不变"代入（$e^{rx}$ 的导数仍然是 $e^{rx}$ 的常数倍），微分算子被约化为乘以 $r$ 的幂次。

## 练习

### 练习 1：特征方程与通解（基础功）

写出下列各方程的特征方程，求根，写出通解。

(a) $y'' - 5y' + 6y = 0$

(b) $y'' + 4y' + 4y = 0$

(c) $y'' + 2y' + 5y = 0$

(d) $y'' - 9y = 0$

**答案提示：**
(a) $r^2 - 5r + 6 = 0 \implies (r-2)(r-3) = 0$。$r_1 = 2$，$r_2 = 3$。$y = C_1 e^{2x} + C_2 e^{3x}$。
(b) $r^2 + 4r + 4 = 0 \implies (r+2)^2 = 0$。重根 $r = -2$。$y = (C_1 + C_2 x)e^{-2x}$。
(c) $r^2 + 2r + 5 = 0$。$\Delta = 4 - 20 = -16$。$r = -1 \pm 2i$。$y = e^{-x}(C_1\cos 2x + C_2\sin 2x)$。
(d) $r^2 - 9 = 0 \implies r = \pm 3$。$y = C_1 e^{3x} + C_2 e^{-3x}$。（这正是扩散-反应方程 $c'' - 9c = 0$ 的形式！也可以写成 $y = A\cosh 3x + B\sinh 3x$。）

### 练习 2：初值问题（完整求解）

解以下初值问题并描述解的长期行为。

(a) $y'' + y' - 2y = 0$，$y(0) = 1$，$y'(0) = 0$

(b) $y'' + 6y' + 9y = 0$，$y(0) = 1$，$y'(0) = 0$

(c) $y'' + 2y' + 2y = 0$，$y(0) = 1$，$y'(0) = -1$

**答案提示：**
(a) 特征根 $r = 1, -2$。$y = C_1 e^x + C_2 e^{-2x}$。代入得 $C_1 = 2/3$，$C_2 = 1/3$。长期：$y \to +\infty$（$e^x$ 项主导）。
(b) 重根 $r = -3$。$y = (1 + 3x)e^{-3x}$。长期：$y \to 0$（$(1+3x)$ 增长远慢于 $e^{-3x}$ 的衰减）。
(c) $r = -1 \pm i$。$y = e^{-x}(\cos x + C_2 \sin x)$。$y'(0) = -1 + C_2 - 1 = -1$ 得 $C_2 = 0$（仔细算！$y' = e^{-x}(-\cos x - \sin x + C_2(\cos x - \sin x))$，$y'(0) = -1 + C_2 = -1$，$C_2 = 0$）。$y = e^{-x}\cos x$——衰减振荡，在 $x = \pi/2, 3\pi/2, \ldots$ 处过零，振幅按 $e^{-x}$ 衰减。

### 练习 3：扩散-反应方程的边界值问题（膜科学核心）

催化膜内溶质浓度满足：

$$
D\frac{d^2c}{dx^2} - kc = 0, \quad c(0) = c_0, \quad c'(L) = 0
$$

第一个条件是膜入口处浓度为 $c_0$。第二个条件 $c'(L) = 0$ 表示膜的另一端是**不可渗透壁**——溶质无法穿过（"零通量"边界条件，在死端膜和负载型催化膜中很常见）。

(a) 写出特征方程并求根。令 $\phi = \sqrt{k/D}$。

(b) 写出通解（用双曲函数或指数函数形式）。

(c) 利用两个边界条件确定常数。最终结果是：

$$
c(x) = c_0 \frac{\cosh[\phi(L-x)]}{\cosh(\phi L)}
$$

请你验证这个结果确实满足方程和两个边界条件。

(d) 讨论：当 $\phi L \to 0$（反应极慢或膜极薄），$c(x)$ 趋向什么？当 $\phi L \to \infty$（反应极快或膜极厚），$c(x)$ 在 $x = 0$ 附近的行为如何？

**答案提示：**
(a) $r^2 - \phi^2 = 0$，$r = \pm\phi$。
(b) $c(x) = A\cosh(\phi x) + B\sinh(\phi x)$，或等价地 $c = C_1 e^{\phi x} + C_2 e^{-\phi x}$。
(c) $c(0) = A = c_0$（如果用 $\cosh$-$\sinh$ 形式起点在 $x=L$ 端更方便——改写为 $c = \tilde{A}\cosh[\phi(L-x)] + \tilde{B}\sinh[\phi(L-x)]$）。$c'(L) = 0$ 给出 $\tilde{B} = 0$。$c(0) = c_0$ 给出 $\tilde{A} = c_0/\cosh(\phi L)$。
(d) $\phi L \to 0$ 时 $\cosh \to 1$，$c(x) \to c_0$——几乎均匀，反应来不及消耗什么。$\phi L \to \infty$ 时，$c(x) \approx c_0 e^{-\phi x}$ 在入口附近指数衰减——深处几乎没有溶质。

### 练习 4：判别式的物理意义（概念思考）

考虑一般的二阶方程 $y'' + 2\gamma y' + \omega_0^2 y = 0$，其中 $\gamma > 0$（阻尼参数），$\omega_0 > 0$（固有频率）。

(a) 写出特征方程和判别式。

(b) 证明：三种情况分别对应 $\gamma > \omega_0$（过阻尼）、$\gamma = \omega_0$（临界阻尼）、$\gamma < \omega_0$（欠阻尼）。

(c) 在欠阻尼情况下，求振荡频率 $\omega_d$（叫做"阻尼频率"）与 $\gamma$ 和 $\omega_0$ 的关系。解释为什么阻尼越大，振荡频率越低。

(d) 膜联想：在一个膜过滤系统中，操作压力突然从 $P_1$ 阶跃到 $P_2$。如果通量的动态响应可以近似为二阶系统（这在某些复杂的多层膜系统中是合理的），那么三种阻尼情况分别对应什么样的通量响应曲线？画出草图。

**答案提示：**
(a) $r^2 + 2\gamma r + \omega_0^2 = 0$。$\Delta = 4\gamma^2 - 4\omega_0^2 = 4(\gamma^2 - \omega_0^2)$。
(b) $\Delta > 0 \Leftrightarrow \gamma > \omega_0$；$\Delta = 0 \Leftrightarrow \gamma = \omega_0$；$\Delta < 0 \Leftrightarrow \gamma < \omega_0$。
(c) $r = -\gamma \pm i\sqrt{\omega_0^2 - \gamma^2}$。阻尼频率 $\omega_d = \sqrt{\omega_0^2 - \gamma^2}$。随着 $\gamma$ 增大，$\omega_d$ 减小——阻尼"拖慢"了振荡，直到 $\gamma = \omega_0$ 时振荡完全消失。
(d) 过阻尼：通量缓慢、单调地趋向新稳态，像老人上楼梯。临界阻尼：最快地趋向新稳态，不冲过头。欠阻尼：通量冲过新稳态，回落，再冲过，反复振荡，逐渐收敛——像弹簧秤上刚放上一个砝码时指针的跳动。

### 练习 5：挑战题——当特征方程遇到超越方程（选做）

在某些膜过程中，边界条件不是简单的"给定浓度值"，而是涉及传质系数。例如：

$$
c'' - \phi^2 c = 0, \quad c(0) = c_0, \quad -Dc'(L) = h_m \cdot c(L)
$$

第二个条件说的是：膜右端面的扩散通量 $-Dc'(L)$ 等于外部传质系数 $h_m$ 乘以该处浓度——这是**Robin 边界条件**（也叫第三类边界条件），在膜传质建模中非常常见。

(a) 写出通解 $c(x) = A\cosh(\phi x) + B\sinh(\phi x)$，代入两个边界条件，推导 $A$ 和 $B$。

(b) 令无量纲参数 $\text{Bi} = h_m L / D$（Biot 数）和 $\Phi = \phi L$（Thiele 模数）。把最终结果写成无量纲形式。

(c) 讨论两个极限：$\text{Bi} \to \infty$（外部传质阻力为零，退化为 $c(L) = 0$）和 $\text{Bi} \to 0$（外部传质极慢，退化为 $c'(L) = 0$——练习3的情况）。验证你的公式在这两个极限下确实回到了已知的结果。

**提示**：$c(0) = A = c_0$。第二个条件给出 $D\phi[A\sinh(\phi L) + B\cosh(\phi L)] = h_m[A\cosh(\phi L) + B\sinh(\phi L)]$。解出 $B/A$，化简。最终结果中出现的 $\cosh$ 和 $\sinh$ 的线性组合不能化简为单个双曲函数——这是 Robin 条件比 Dirichlet/Neumann 条件更复杂的代价。但表达式仍然是闭合的、可以直接数值计算的。

## 后记：代数与微积分的握手

让我们站远一步，回望这堂课的全貌。

你学会了一个方法：面对 $ay'' + by' + cy = 0$，猜 $y = e^{rx}$，代入，微分方程坍缩成代数方程 $ar^2 + br + c = 0$。根据根的形态——两个实根、重根、复根——你得到了三种通解。

这个方法之所以有效，有两个根本原因。第一个是**指数函数的形式不变性**：$e^{rx}$ 求导后仍然是 $e^{rx}$ 的常数倍。第二个是**方程的线性性**：只有线性方程才能在代入 $e^{rx}$ 后"完美消去"，只留下一个关于 $r$ 的多项式。

如果方程是非线性的——比如 $y'' + (y')^2 = 0$——代入 $e^{rx}$ 后得到的不是多项式，而是一团乱麻。特征方程法瞬间失效。线性就像一座桥——它允许你在微分方程和代数方程之间自由穿梭。一旦桥断了（方程变成非线性），你就被困在了微分方程那一侧，不得不寻找其他出路。

但好消息是：在膜科学和大多数工程应用中，你遇到的二阶ODE绝大多数是线性常系数的——至少在首次近似中是。扩散-反应方程、弹簧质量阻尼器、RLC 电路、浓度极化模型——它们都在这座桥的通行范围内。

下一课，我们将学习当方程右端不为零时（非齐次方程 $ay'' + by' + cy = g(x)$）如何求解。那时候，你今天学的齐次通解将扮演"基底"的角色，而特解的寻找将需要新的技巧——待定系数法和参数变易法。齐次解不会退场，它只是从独唱变成了合唱的一部分。
