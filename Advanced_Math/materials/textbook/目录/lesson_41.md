```markdown
## 1. 一个反直觉的起点：无知的精确表达

年轻的研究者往往害怕误差。在他们的潜意识里，误差代表失败——实验不够精密、操作不够仔细、仪器不够昂贵。他们恨不得把误差藏起来，只报告那个"最好看的"数值。

但请想一想：如果有人告诉你"这根膜纤维的通量是 $32.7\;\text{L}\,\text{m}^{-2}\,\text{h}^{-1}$"，你满意吗？你立刻会追问——

> "这个数字你有多确信？32.7 是什么意思？是 $32.7 \pm 0.1$，还是 $32.7 \pm 15$？如果是后者，你其实几乎什么都没告诉我。"

**一个不带不确定度的测量值，就像一句没有语调的话——你无法判断说话人是在陈述事实、提出猜测、还是在讽刺。** 误差分析就是科学家给自己的话添上"语调"的方法。它不是你的敌人；它是你诚实地报告"你知道什么和不知道什么"的语言。

这种语言一旦掌握，你会发现它赋予你惊人的权力：你可以量化自己的无知，并且——这才是关键——找出无知的来源，然后有针对性地消灭它。

让我们从两种根本不同的"无知"开始。

## 2. 两种误差：偏了的秤和颤抖的手

### 2.1 类比

想象你有一台电子秤。你把一个标准砝码（真实质量 $100.000\;\text{g}$）放上去，反复称量十次，得到如下读数：

```

102.3, 102.1, 102.5, 102.2, 102.4,
102.3, 102.0, 102.6, 102.1, 102.3

```

你注意到两件事：

1. **所有读数都偏大了约 $2.3\;\text{g}$。** 不管你称多少次，中心都在 $102.3$ 附近，而不是 $100.0$。——这台秤偏了。
2. **每次读数之间有微小的波动。** 不完全相同，分散在 $102.0$ 到 $102.6$ 之间。

第一件事叫**系统误差（systematic error）**，第二件事叫**随机误差（random error）**。

### 2.2 系统误差：偏了的秤

系统误差有一个令人不安的特征：**它不会因为你多做几次实验就消失。** 你称一百次、一千次，平均值仍然在 $102.3$ 附近，而不是真值 $100.0$。重复实验对系统误差无能为力。

系统误差的来源极其多样：

- 仪器未经校准（"你的秤偏了2克"）。
- 环境因素的持续偏移（温度计放在了空调出风口旁边）。
- 方法本身的固有偏差（你测量膜的有效面积时，忽略了密封圈遮挡的环形区域）。
- 操作者的个人习惯（你读液位时总是从上方俯视，而不是平视，于是系统性地读高了——这就是"视差"）。

系统误差的可怕之处在于：**你往往不知道它的存在。** 它是隐形的。你怎么发现一台秤偏了？你必须用一个已知的标准来检验它。这就是为什么实验室里需要标准品、标准曲线、仪器校准——它们的存在理由之一就是揪出系统误差。

> **膜科学情境**：你在测试一张新合成的纳滤膜对 $\text{NaCl}$ 的截留率。你用电导率仪测量透过液浓度。但如果这台电导率仪最后一次校准是三个月前，而探头上已经积了一层薄薄的水垢——它给出的电导率就会系统性地偏低，于是你计算出的截留率就会系统性地偏高。你重复十次实验，十次都偏高。你满心欢喜以为自己合成了一张超级好膜，但真相是——你的"秤"偏了。

### 2.3 随机误差：颤抖的手

随机误差完全不同。它没有固定的方向，时正时负，服从某种概率分布（在许多情况下近似正态分布——回忆第32课）。

随机误差的来源：

- 电子噪声（仪器内部的热扰动）。
- 环境的微小波动（实验室温度在一天中微微漂移）。
- 操作的不可完全重复性（你每次剪裁膜样品，面积不可能完全一样）。
- 样品本身的不均匀性（同一张膜的不同位置，孔径分布可能有微小差异）。

随机误差有一个可爱的特性：**它可以通过多次测量来驯服。** 如果你称量 $n$ 次取平均值，随机误差对平均值的影响大约以 $1/\sqrt{n}$ 的速率缩小（回忆第33课的标准误差概念）。这就是重复实验的意义——不是为了消灭系统误差（那是校准的事），而是为了压缩随机误差。

### 2.4 一张对比表

```

┌────────────────┬──────────────────────┬─────────────────────────┐
│                │  系统误差             │  随机误差                │
│                │ (Systematic Error)    │ (Random Error)           │
├────────────────┼──────────────────────┼─────────────────────────┤
│ 方向           │ 固定（总是偏大或偏小）│ 随机（时正时负）          │
│ 多次测量能解决？│ 不能                 │ 能（以 1/√n 缩小）       │
│ 如何发现？      │ 与标准品对照/校准     │ 计算标准差/观察散布       │
│ 如何减少？      │ 改进方法/校准仪器     │ 增加重复次数/改进精密度   │
│ 影响的是…      │ 准确度 (accuracy)     │ 精密度 (precision)       │
└────────────────┴──────────────────────┴─────────────────────────┘

```

**准确度（accuracy）** 是说你射的箭有没有集中靶心附近。**精密度（precision）** 是说你射的箭有没有彼此靠近。一个弓箭手可以非常精密但不准确（所有箭都射到了左上角的同一个点），也可以准确但不精密（箭的平均位置在靶心，但散得到处都是）。理想当然是两者兼得。

```

准确但不精密         精密但不准确          准确又精密
·                     · ·                   ·
·               · · ·                  · ·
⊕     ·              · ·    ⊕              ⊕·
·                   · ·                    ·
·

（⊕ = 靶心）

```

如果你只看平均值，第一和第三种情况可能给出类似的结果；但如果你看标准差，第一种情况的标准差远大于第三种。这就是为什么**光报平均值不够，你必须同时报告不确定度**。

## 3. 误差传播：全微分法的荣耀时刻

### 3.1 问题的提出

实验室里，你几乎不会只直接测量你最终关心的那个量。你关心的是膜的通量 $J$，但你直接测量的是透过液体积 $V$、收集时间 $t$、膜面积 $A$。然后你用公式

$$
J = \frac{V}{A \cdot t}
$$

来计算 $J$。

问题来了：$V$、$A$、$t$ 各自都有测量不确定度 $\delta V$、$\delta A$、$\delta t$。这些不确定度如何"传播"到最终结果 $J$ 的不确定度 $\delta J$ 中？

这正是第11课全微分法大显身手的地方。

### 3.2 从全微分到误差传播

回忆第11课。如果 $f = f(x_1, x_2, \ldots, x_n)$，那么 $f$ 的全微分是

$$
df = \sum_{i=1}^{n} \frac{\partial f}{\partial x_i}\,dx_i
$$

现在做一个关键的概念跳跃：把 $dx_i$ 替换为 $\delta x_i$——第 $i$ 个测量变量的不确定度。

但是，直接把所有 $\delta x_i$ 加起来会怎样？

$$
\delta f_{\max} = \sum_{i=1}^{n} \left|\frac{\partial f}{\partial x_i}\right| \cdot \delta x_i
$$

这给出了**最大可能误差**——所有误差恰好在最坏情况下同时叠加。这在实际中极不可能发生。这就像说："所有噪声源恰好在同一个瞬间全都往同一个方向推了你一把。"概率上，这几乎不会发生。

更合理的做法是使用**均方根合成**（Root Sum of Squares，简称 RSS）。如果各个测量变量的随机误差是**独立的**（这个条件很重要！），那么

$$
\delta f = \sqrt{\sum_{i=1}^{n} \left(\frac{\partial f}{\partial x_i} \cdot \delta x_i\right)^2}
$$

这就是**误差传播公式**，也叫**不确定度传播的RSS公式**。

### 3.3 为什么是均方根？——一个直觉

让我用一个物理直觉来解释为什么是"平方和再开根号"，而不是简单相加。

想象你在二维平面上随机走步。每一步的长度是 $\delta x_i$，方向随机。走了 $n$ 步之后，你离原点多远？

如果所有步都朝同一个方向（最坏情况），你走了 $\delta x_1 + \delta x_2 + \cdots + \delta x_n$。但如果方向是随机的，步之间会互相部分抵消。统计力学告诉我们（这也是布朗运动的核心结论——回忆第34课），$n$ 步随机行走的净位移大约是

$$
\text{净位移} \sim \sqrt{\delta x_1^2 + \delta x_2^2 + \cdots + \delta x_n^2}
$$

独立随机误差的合成遵循完全相同的逻辑。每个误差源是一个"随机的步"，总误差是这些"步"的RSS合成。

这就是为什么 RSS 合成总是小于或等于简单相加——**独立的随机误差会互相部分抵消**。

### 3.4 特殊情况速查

在推导一般公式之前，让我给你几个最常用的结论。对于一些简单的函数形式，误差传播公式可以直接写出。

**加减法：** 若 $f = x + y$ 或 $f = x - y$，则

$$
\delta f = \sqrt{(\delta x)^2 + (\delta y)^2}
$$

注意：减法时误差也是加的！这就是为什么"两个大数相减得到一个小数"是实验中最危险的操作之一——绝对误差不变，但相对误差（$\delta f / f$）可以暴涨。

> **膜科学例子**：你在做正渗透实验。进料液浓度 $c_{\text{feed}} = 35.0 \pm 0.5\;\text{g/L}$，透过液浓度 $c_{\text{perm}} = 0.3 \pm 0.5\;\text{g/L}$。截留率 $R = 1 - c_{\text{perm}}/c_{\text{feed}}$ 中的 $c_{\text{feed}} - c_{\text{perm}} = 34.7 \pm \sqrt{0.5^2 + 0.5^2} = 34.7 \pm 0.71\;\text{g/L}$，相对误差约 $2\%$。但如果你关心的是 $c_{\text{perm}}$ 本身——$0.3 \pm 0.5\;\text{g/L}$——它的相对误差是 $167\%$！这个数字基本上毫无意义。

**乘除法：** 若 $f = x \cdot y$ 或 $f = x / y$，则相对误差

$$
\frac{\delta f}{|f|} = \sqrt{\left(\frac{\delta x}{x}\right)^2 + \left(\frac{\delta y}{y}\right)^2}
$$

这是一个极其优雅的结果：**乘除法传播的是相对误差**。

**幂函数：** 若 $f = x^n$，则

$$
\frac{\delta f}{|f|} = |n| \cdot \frac{\delta x}{|x|}
$$

幂次 $n$ 充当了"误差放大器"。如果你用 $r^2$ 来计算面积（$A = \pi r^2$），半径的 $1\%$ 不确定度会变成面积的 $2\%$ 不确定度。如果你计算 $r^4$（比如 Hagen-Poiseuille 方程中流量依赖于 $r^4$），$1\%$ 的半径误差就爆炸成 $4\%$ 的流量误差。

让我把这些列成一个小表，方便你将来速查：

```

┌────────────────────────┬─────────────────────────────────────┐
│ 函数形式               │  RSS误差传播                         │
├────────────────────────┼─────────────────────────────────────┤
│ f = x ± y              │  δf = √(δx² + δy²)                 │
│ f = x · y  或  f = x/y │  δf/|f| = √((δx/x)² + (δy/y)²)    │
│ f = xⁿ                 │  δf/|f| = |n| · δx/|x|             │
│ f = ln(x)              │  δf = δx / |x|                     │
│ f = eˣ                 │  δf/|f| = δx                       │
└────────────────────────┴─────────────────────────────────────┘

```

## 4. 完整例题：从温度和压力计算膜通量的不确定度

### 4.1 问题设定

假设你在研究一种气体分离膜。根据一个简化的渗透模型，气体透过膜的通量 $J$ 可以表示为

$$
J = \frac{P_0}{L} \cdot \Delta p \cdot \exp\!\left(-\frac{E_a}{RT}\right)
$$

其中：
- $P_0 = 2.5 \times 10^{-10}\;\text{mol}\,\text{m}^{-1}\,\text{s}^{-1}\,\text{Pa}^{-1}$（渗透系数预指数因子，视为精确常数）
- $L = 50 \times 10^{-6}\;\text{m}$（膜厚度，视为精确常数——在本例中假设已精确测量）
- $E_a = 20.0\;\text{kJ/mol}$（活化能，视为精确常数）
- $R = 8.314\;\text{J}\,\text{mol}^{-1}\,\text{K}^{-1}$（气体常数，精确）
- $\Delta p = 300 \pm 5\;\text{kPa}$（压差，有不确定度）
- $T = 353 \pm 2\;\text{K}$（温度，有不确定度）

**问：$J$ 的不确定度 $\delta J$ 是多少？每个测量变量各贡献了多少？**

### 4.2 第一步：确认函数形式

$J$ 是 $\Delta p$ 和 $T$ 的函数（其他量视为常数）。写紧凑一些：

$$
J = C \cdot \Delta p \cdot \exp\!\left(-\frac{\alpha}{T}\right)
$$

其中 $C = P_0 / L$ 是常数，$\alpha = E_a / R = 20000 / 8.314 = 2405.5\;\text{K}$ 也是常数。

### 4.3 第二步：求偏导数

**对 $\Delta p$ 求偏导：**

$$
\frac{\partial J}{\partial (\Delta p)} = C \cdot \exp\!\left(-\frac{\alpha}{T}\right)
$$

注意到 $J = C \cdot \Delta p \cdot \exp(-\alpha/T)$，所以

$$
\frac{\partial J}{\partial (\Delta p)} = \frac{J}{\Delta p}
$$

这正是"乘法关系传播相对误差"的体现。

**对 $T$ 求偏导：**

$$
\frac{\partial J}{\partial T} = C \cdot \Delta p \cdot \exp\!\left(-\frac{\alpha}{T}\right) \cdot \frac{\alpha}{T^2} = J \cdot \frac{\alpha}{T^2}
$$

### 4.4 第三步：写出RSS误差传播公式

$$
\delta J = \sqrt{\left(\frac{\partial J}{\partial (\Delta p)} \cdot \delta(\Delta p)\right)^2 + \left(\frac{\partial J}{\partial T} \cdot \delta T\right)^2}
$$

代入偏导数：

$$
\delta J = \sqrt{\left(\frac{J}{\Delta p} \cdot \delta(\Delta p)\right)^2 + \left(J \cdot \frac{\alpha}{T^2} \cdot \delta T\right)^2}
$$

提出 $J$：

$$
\frac{\delta J}{J} = \sqrt{\left(\frac{\delta(\Delta p)}{\Delta p}\right)^2 + \left(\frac{\alpha}{T^2} \cdot \delta T\right)^2}
$$

### 4.5 第四步：代入数值

**压差项的贡献：**

$$
\frac{\delta(\Delta p)}{\Delta p} = \frac{5}{300} = 0.0167 \quad (1.67\%)
$$

**温度项的贡献：**

$$
\frac{\alpha}{T^2} \cdot \delta T = \frac{2405.5}{353^2} \times 2 = \frac{2405.5}{124609} \times 2 = 0.01930 \times 2 = 0.03861 \quad (3.86\%)
$$

**总的相对不确定度：**

$$
\frac{\delta J}{J} = \sqrt{0.0167^2 + 0.0386^2} = \sqrt{0.000279 + 0.001490} = \sqrt{0.001769} = 0.0421 \quad (4.21\%)
$$

### 4.6 第五步：分析——谁是罪魁祸首？

来看看各项的贡献：

- 压差项：$0.0167^2 = 0.000279$，占总方差 $0.001769$ 的 $\mathbf{15.8\%}$
- 温度项：$0.0386^2 = 0.001490$，占总方差 $0.001769$ 的 $\mathbf{84.2\%}$

**温度不确定度是通量不确定度的主要来源！** 它贡献了总不确定度方差的 $84\%$。

这个发现至关重要。它告诉你：**如果你想改进实验精度，应该优先提高温度测量的精度**，而不是去花钱买更好的压力计。把 $\delta T$ 从 $2\;\text{K}$ 降到 $1\;\text{K}$，温度项贡献会降为原来的 $1/4$，总相对不确定度会降到约 $2.5\%$。而把 $\delta(\Delta p)$ 从 $5\;\text{kPa}$ 降到 $1\;\text{kPa}$（花一大笔钱），总不确定度也不过从 $4.21\%$ 降到 $3.89\%$——改善微乎其微。

**这就是误差分析的权力：它不仅告诉你"你不知道多少"，还告诉你"你应该把钱和精力花在哪里"。**

### 4.7 计算通量的绝对值和绝对不确定度

为了完整，算一下通量本身：

$$
C = \frac{P_0}{L} = \frac{2.5 \times 10^{-10}}{50 \times 10^{-6}} = 5.0 \times 10^{-6}\;\text{mol}\,\text{m}^{-2}\,\text{s}^{-1}\,\text{Pa}^{-1}
$$

$$
J = 5.0 \times 10^{-6} \times 300 \times 10^{3} \times \exp\!\left(-\frac{2405.5}{353}\right)
$$

$$
= 5.0 \times 10^{-6} \times 3 \times 10^{5} \times \exp(-6.814)
$$

$$
= 1.5 \times \exp(-6.814)
$$

$$
= 1.5 \times 1.097 \times 10^{-3}
$$

$$
= 1.645 \times 10^{-3}\;\text{mol}\,\text{m}^{-2}\,\text{s}^{-1}
$$

绝对不确定度：

$$
\delta J = 0.0421 \times 1.645 \times 10^{-3} = 6.9 \times 10^{-5}\;\text{mol}\,\text{m}^{-2}\,\text{s}^{-1}
$$

**最终报告：** $J = (1.65 \pm 0.07) \times 10^{-3}\;\text{mol}\,\text{m}^{-2}\,\text{s}^{-1}$

注意我在报告时做了什么：不确定度 $\delta J$ 保留了一位有效数字（$0.07$），然后把 $J$ 的有效数字对齐到相同的小数位。这引出了我们的下一个话题。

## 5. RSS合成与有效数字规则

### 5.1 有效数字：被高中教坏的概念

几乎每个学生都在高中化学课上学过"有效数字规则"：乘除法保留最少有效数字数，加减法对齐最后一位小数。这些规则是真正的不确定度分析的一种**粗糙近似**。它们在80%的日常情况下大致可用，但在你成为研究者之后，你应该知道它们的局限。

真正的规则是这样的：

> **规则：不确定度通常保留1至2位有效数字。然后将最终结果的有效数字对齐到不确定度所在的位数。**

例如：
- 计算得 $J = 1.6452837 \times 10^{-3}$，$\delta J = 6.93 \times 10^{-5}$
- 不确定度保留1位有效数字：$\delta J \approx 7 \times 10^{-5} = 0.00007$
- 结果对齐：$J = 0.00165$，即 $J = 1.65 \times 10^{-3}$
- 报告：$(1.65 \pm 0.07) \times 10^{-3}$

写成 $1.6452837 \times 10^{-3}$ 是**虚假精度**——你的不确定度在第四位小数上，后面的数字全是噪声伪装成信号。把它们写下来不是严谨，是欺骗（骗别人，也骗自己）。

### 5.2 中间计算：保留全部精度

但是要注意一个重要的实践要点：**在计算过程中，你应该保留所有数位**（或至少多保留2-3位），只在最终报告结果时才做有效数字的截断。过早的四舍五入会导致"舍入误差的累积"，这是一种完全可以避免的人为误差。

用一个比喻：在厨房里做菜时，你可以使用精确的量杯。但端上桌时，你把边缘整理一下，让菜品看起来漂亮。没有人在做菜过程中就开始"修饰摆盘"——那只会让你手忙脚乱。

### 5.3 RSS合成的一般步骤总结

当你的最终量 $f$ 由多个独立测量量 $x_1, x_2, \ldots, x_n$ 通过某个函数关系 $f = f(x_1, x_2, \ldots, x_n)$ 决定时：

1. **写出函数关系。**
2. **对每个 $x_i$ 求偏导 $\partial f / \partial x_i$。**（这一步用到了第11课的技术。）
3. **写出RSS公式：** $\delta f = \sqrt{\sum_i \left(\frac{\partial f}{\partial x_i} \cdot \delta x_i\right)^2}$。
4. **代入数值。**
5. **分析各项贡献**——找出"最大罪犯"。
6. **正确报告结果**——不确定度1-2位有效数字，结果对齐。

这个步骤如此标准化，以至于你可以（也应该）把它编成一个模板，在你的每篇论文、每次实验报告中反复使用。

### 5.4 一个微妙但重要的条件：独立性

RSS公式成立的关键前提是：**各测量变量的误差是互相独立的。** 如果 $x$ 和 $y$ 的误差是相关的（比如你用同一台仪器同时测量了 $x$ 和 $y$，仪器的零点漂移同时影响了两者），那么完整的误差传播公式需要包含协方差项：

$$
\delta f = \sqrt{\sum_{i} \left(\frac{\partial f}{\partial x_i}\right)^2 (\delta x_i)^2 + 2\sum_{i<j} \frac{\partial f}{\partial x_i}\frac{\partial f}{\partial x_j}\,\text{cov}(x_i, x_j)}
$$

如果 $x_i$ 和 $x_j$ 正相关，协方差项为正，总误差比RSS大。如果负相关，总误差比RSS小。在许多实验情况下，研究者会通过仔细的实验设计来保证测量的独立性，从而合法地使用简单的RSS公式。这就引出了我们最后一个主题。

## 6. 实验设计初步：纪律的艺术

### 6.1 为什么科学实验需要纪律？

你可能会想：做实验不就是"改一个东西，看看结果怎么变"吗？原则上是的，但魔鬼在细节里。

假设你想研究"操作压力对反渗透膜通量的影响"。你做了如下实验：

- 第一天：压力 $5\;\text{bar}$，温度 $25°\text{C}$，进料浓度 $1000\;\text{ppm}$ $\text{NaCl}$，新膜——通量 $30\;\text{LMH}$。
- 第二天：压力 $10\;\text{bar}$，温度 $28°\text{C}$（那天空调坏了），进料浓度 $1200\;\text{ppm}$（前一天忘了换进料液），膜用了一天已有轻微污染——通量 $50\;\text{LMH}$。

你能自信地说"通量随压力增加而增加"吗？不能。因为从第一天到第二天，变化的不仅仅是压力。温度也变了，浓度也变了，膜的状态也变了。你无法分离压力本身的效应——它和其他因素的效应纠缠在一起了。

这就像在一锅汤里同时加了盐、糖、醋三种调料，然后说"变咸了是因为加了盐"——你怎么知道不是其他调料也有影响？

### 6.2 核心概念

实验设计中有三个最基本的角色：

**独立变量（independent variable）**：你主动改变的量。在上面的例子里，压力是你打算研究的独立变量。

**响应变量（dependent variable / response variable）**：你测量的结果。在上面的例子里，通量 $J$ 是响应变量。

**控制变量（controlled variable）**：所有你不想改变、需要保持恒定的量。温度、进料浓度、膜的状态——这些必须在所有实验组中保持一致（或者至少被监测和记录）。

**核心原则：一次只改变一个变量。** 其他所有可能影响结果的因素都必须保持不变。只有这样，你观察到的响应变化才能归因于你改变的那个变量。

### 6.3 重复实验的意义

你做了一次实验，测得通量 $J = 32.7\;\text{LMH}$。你应该报告这个结果吗？

不。你应该至少重复几次。为什么？

**第一个理由：** 估计随机误差。如果只有一个数据点，你完全不知道 $32.7$ 的随机不确定度有多大。是 $32.7 \pm 0.1$ 还是 $32.7 \pm 15$？一次实验无法告诉你。至少做三次，你才能计算标准差。

**第二个理由：** 发现异常值。假设你做了五次：$32.7, 32.5, 32.9, 32.6, 45.1$。最后那个 $45.1$ 明显异常——也许那次实验中有气泡堵住了管路，或者温度突然飙升。只做一次实验，如果恰好遇上了那个异常情况，你就会得到一个完全错误的结论，而且你永远不会知道它是错的。

**第三个理由：** 统计推断的基础。如果你想比较"压力 $5\;\text{bar}$ 和 $10\;\text{bar}$ 的通量是否有统计学上的显著差异"，你需要每组至少有几个数据点，才能做有意义的统计检验（如 $t$ 检验——这会在第43课详细讨论）。

> **经验法则：** 对于关键数据点，至少重复 $3$ 次（获得标准差的粗略估计）。如果资源允许，$5$ 次更好（因为 $n=5$ 的 $t$ 分布已经相当接近正态了）。如果你计划做统计检验来发表论文，请在实验前做统计功效分析（power analysis），确定所需的最小样本量。这不是奢侈品——这是科学的基本卫生。

### 6.4 一个完整的实验设计例子

**研究目标：** 考察操作压力对新型聚酰胺纳滤膜截留率的影响。

**独立变量：** 跨膜压差 $\Delta p$。取值：$3, 5, 7, 9, 11\;\text{bar}$（5个水平）。

**响应变量：** 对 $\text{MgSO}_4$ 的截留率 $R$。

**控制变量（保持恒定）：**
- 进料浓度：$2000\;\text{ppm}$ $\text{MgSO}_4$
- 温度：$25.0 \pm 0.5\;°\text{C}$（恒温水浴控制）
- 错流速度：$0.5\;\text{m/s}$
- 膜样品：同一批次切割的膜片
- 预处理方案：每片膜先在 $10\;\text{bar}$ 纯水下压密 $30\;\text{min}$

**重复次数：** 每个压力水平使用 $3$ 片独立的膜片（生物学重复），每片膜测量 $2$ 次（技术重复），取平均值。

**随机化：** 压力水平的测试顺序随机排列（不是从低到高——因为膜可能随着实验进程而老化，如果你总是从低压到高压，你就无法区分"压力效应"和"时间效应"）。

**数据收集清单：**

```

┌──────────────────────────────┐
│  每次实验必须记录：          │
│  □  日期和时间               │
│  □  膜片编号                 │
│  □  实际跨膜压差             │
│  □  进料液温度               │
│  □  错流流量                 │
│  □  进料液电导率             │
│  □  透过液电导率             │
│  □  透过液收集体积和时间     │
│  □  任何异常情况的文字记录   │
└──────────────────────────────┘

````

注意最后一项——"任何异常情况的文字记录"。在实验过程中，可能发生各种意想不到的事情：管路里出现气泡、进料液浑浊了、隔壁实验室有人开了大功率离心机导致电压波动……这些"不在计划内"的事件如果不被记录下来，事后你将无法解释那些异常数据点的来源。

> **一条被低估的建议：** 养成写实验日志的习惯——不是只记数字，而是用完整的句子描述你观察到的一切。"14:37，测量第三组数据时发现透过液管里有一个小气泡，敲了一下管壁后消失"——这样的记录在两个月后分析数据时可能价值连城。

### 6.5 再谈系统误差与实验设计的关系

仔细设计实验不仅仅是为了降低随机误差和分离变量效应——它也是发现和消除系统误差的手段。

**空白实验（blank）**：在没有膜的情况下做一次实验（或者用一张已知性能的标准膜），看看你的系统本身是否引入偏差。

**对照实验（control）**：用一张已知性能的商用膜（比如 Dow Filmtec 的 NF90）在你的装置上测量，看看结果是否与文献报道一致。如果不一致——你的装置可能存在系统误差。

**随机化（randomization）**：打乱实验顺序，防止时间趋势被错误地归因于你研究的变量。

**盲法（blinding）**：在某些情况下，让测量者不知道样品的身份。这在膜科学中不太常见，但在药学和临床试验中是核心要求。它存在的原因是人类观察者有无意识的偏见——你希望自己的新膜很好，这个期望可能会无意识地影响你的读数判断。

## 7. 关键教学时刻：误差是光，不是阴影

让我用一段比较长的话来结束这一课的正文部分，因为我觉得这件事值得你认真读一遍。

在科学研究的训练中，有一个痛苦的过渡期。在本科阶段，你被训练成"给出正确答案"的人。考试有标准答案，作业有标准解法，你的目标是尽可能接近那个"正确值"。误差被视为扣分项——偏离标准答案越远，你的成绩越低。

但在研究中，情况完全反过来了。你面对的是**没有人知道答案**的问题。你是第一个测量这个量的人。不存在"标准值"可以对照。在这种情况下，你的测量值和不确定度具有同等的重要性——前者告诉世界"我们最好的猜测是多少"，后者告诉世界"这个猜测有多可靠"。

想象一个探险家在浓雾中行走。他的位置（测量值）很重要，但他能看到多远（不确定度）同样重要。一个宣称自己精确知道位置但其实什么也看不见的探险家，比一个坦率承认"我大概在这里，但上下五十米都有可能"的探险家危险得多。

**没有不确定度的测量不是"更干净"——它是不完整的。** 报告不确定度不是在承认失败，而是在展示你的专业素养和科学诚信。审稿人看到合理的误差棒（error bars）时，不会觉得你的实验差——恰恰相反，他们会觉得你知道自己在做什么。

反过来，那些不报告误差、数据点悬浮在图表上像没有根的浮萍一样的论文——它们才真正令人不安。你无法判断两个数据点之间的差异是真实的物理效应，还是仅仅是噪声。这样的数据不是科学——它是自欺。

所以请记住：**误差不是你的敌人。它是你诚实地与世界对话的语言。学会说这种语言的人，他的数据才真正有说服力。**

## 精讲例题

### 例题 1：加法的误差传播

你在膜分离实验中，需要计算总固体溶解物（TDS）中两种离子的贡献之和。测量得：

$$
c_{\text{Na}^+} = 230 \pm 8\;\text{mg/L}, \quad c_{\text{Cl}^-} = 355 \pm 12\;\text{mg/L}
$$

求：$c_{\text{total}} = c_{\text{Na}^+} + c_{\text{Cl}^-}$ 的不确定度。

**解：** 加法使用绝对误差的RSS合成：

$$
\delta c_{\text{total}} = \sqrt{8^2 + 12^2} = \sqrt{64 + 144} = \sqrt{208} = 14.4\;\text{mg/L}
$$

$$
c_{\text{total}} = 585 \pm 14\;\text{mg/L}
$$

注意：14.4 保留到 14（不确定度取1位有效数字），结果对齐到个位。

### 例题 2：对数函数的误差传播

在吸附实验中，你测量了溶质的平衡浓度 $c_e = 0.045 \pm 0.003\;\text{mol/L}$，需要在 Langmuir 等温线的线性化形式中使用 $\ln c_e$。

求：$\ln c_e$ 的不确定度。

**解：** 对于 $f = \ln x$，有 $\delta f = \delta x / |x|$。

$$
\delta(\ln c_e) = \frac{0.003}{0.045} = 0.067
$$

$$
\ln c_e = \ln(0.045) = -3.101
$$

报告：$\ln c_e = -3.10 \pm 0.07$

这意味着在 $\ln c_e$ vs $1/c_e$ 的线性回归图上，每个数据点的水平误差棒是 $\pm 0.07$ 个单位。这个信息对于判断线性拟合的可靠性至关重要——如果你的误差棒比数据点之间的间距还大，那你的拟合结果就需要非常谨慎地解读了。

### 例题 3：组合函数与主导误差源识别

反渗透膜的水通量可以用以下模型描述：

$$
J_w = A \cdot (\Delta p - \Delta \pi)
$$

其中 $A = 3.5 \pm 0.2\;\text{LMH/bar}$ 是膜的水力透过系数，$\Delta p = 15.0 \pm 0.3\;\text{bar}$ 是施加压力，$\Delta \pi = 5.0 \pm 0.5\;\text{bar}$ 是渗透压差。

求 $J_w$ 的不确定度，并识别主导误差源。

**解：**

先计算净驱动压力：$\Delta p - \Delta \pi = 15.0 - 5.0 = 10.0\;\text{bar}$

净驱动压力的不确定度（减法用绝对误差RSS）：

$$
\delta(\Delta p - \Delta \pi) = \sqrt{0.3^2 + 0.5^2} = \sqrt{0.09 + 0.25} = \sqrt{0.34} = 0.583\;\text{bar}
$$

现在 $J_w = A \cdot (\Delta p - \Delta \pi)$ 是乘法关系，使用相对误差的RSS：

$$
\frac{\delta J_w}{J_w} = \sqrt{\left(\frac{\delta A}{A}\right)^2 + \left(\frac{\delta(\Delta p - \Delta \pi)}{\Delta p - \Delta \pi}\right)^2}
$$

$$
= \sqrt{\left(\frac{0.2}{3.5}\right)^2 + \left(\frac{0.583}{10.0}\right)^2}
$$

$$
= \sqrt{0.05714^2 + 0.05830^2}
$$

$$
= \sqrt{0.003265 + 0.003399} = \sqrt{0.006664} = 0.0816 \quad (8.16\%)
$$

$J_w = 3.5 \times 10.0 = 35.0\;\text{LMH}$

$\delta J_w = 0.0816 \times 35.0 = 2.86\;\text{LMH}$

报告：$J_w = 35 \pm 3\;\text{LMH}$

**误差源分析：**
- $A$ 的贡献：$0.003265 / 0.006664 = 49.0\%$
- $(\Delta p - \Delta \pi)$ 的贡献：$0.003399 / 0.006664 = 51.0\%$

两个来源几乎均分！但细看 $(\Delta p - \Delta \pi)$ 内部：

- $\Delta p$ 贡献了 $0.09$（方差），$\Delta \pi$ 贡献了 $0.25$（方差）。
- 在净驱动压力的不确定度中，渗透压差 $\Delta \pi$ 占了 $0.25 / 0.34 = 73.5\%$。

所以最终的"头号嫌犯"是**渗透压差 $\Delta \pi$ 的测量不确定度**。这在实验中是有物理道理的——渗透压往往是通过间接方法（比如从浓度推算）得到的，精度本来就不如直接读取的压力表。

## 练习

### 练习 1：两个大数相减的危险

你测量了膜进料侧和透过侧的 TOC（总有机碳）浓度：

$$
c_{\text{feed}} = 520 \pm 15\;\text{mg/L}, \quad c_{\text{perm}} = 490 \pm 15\;\text{mg/L}
$$

(a) 计算去除量 $\Delta c = c_{\text{feed}} - c_{\text{perm}}$ 及其不确定度。

(b) 计算 $\Delta c$ 的相对不确定度。

(c) 这个测量结果有意义吗？你会怎么建议改进？

**答案提示：** $\delta(\Delta c) = \sqrt{15^2 + 15^2} = 21.2\;\text{mg/L}$。$\Delta c = 30 \pm 21\;\text{mg/L}$。相对不确定度约 $70\%$——这个数字几乎不能说明膜有任何去除效果。改进方向：要么提高 TOC 测量精度，要么选择更高浓度的进料液使去除量更大。

### 练习 2：幂函数的误差放大

Hagen-Poiseuille 方程描述了膜孔中的粘性流动：

$$
Q = \frac{\pi r^4 \Delta p}{8 \mu L}
$$

假设膜孔半径 $r = 5.0 \pm 0.3\;\text{nm}$，其他量视为精确已知。

(a) 求 $Q$ 的相对不确定度。

(b) 如果你将孔径测量精度提高一倍（$\delta r$ 减半），$Q$ 的不确定度减少多少？

(c) 讨论：为什么膜科学家对孔径的精确测量如此执着？

**答案提示：** 因为 $Q \propto r^4$，所以 $\delta Q / Q = 4 \times \delta r / r = 4 \times 0.3/5.0 = 0.24 = 24\%$。如果 $\delta r$ 减半，$\delta Q / Q = 12\%$——改善了一半。这解释了为什么膜表征文献中对孔径分布的讨论篇幅总是那么长。

### 练习 3：设计你自己的实验

你计划研究"进料液 pH 对壳聚糖/聚砜复合膜重金属离子截留率的影响"。

(a) 明确指出：独立变量、响应变量、控制变量各是什么。

(b) 你会取几个 pH 水平？如何选择范围？

(c) 每个 pH 水平需要多少次重复实验？为什么？

(d) 你会如何随机化实验顺序？为什么这很重要？

(e) 列出你会记录的所有测量量，包括那些不直接出现在最终结果中的。

**答案提示：** 独立变量：进料液 pH；响应变量：截留率 $R$（或透过液中重金属浓度）；控制变量：进料浓度、温度、跨膜压差、错流速度、膜样品批次等。pH 范围建议 $3\text{–}9$，每隔 $1$ 取一个水平（$7$ 个水平），或根据文献选择更密的关键区域。每个水平至少 $3$ 次重复。随机化顺序以避免膜老化/污染的时间趋势与 pH 效应混淆。记录量应包括实际 pH 值（不仅仅是目标 pH）、温度、压力、进料液和透过液浓度（用 ICP 测）、通量、实验时间、以及任何异常。

### 练习 4：误差传播的Python实践

编写一个 Python 函数，输入一个函数 $f(x_1, x_2, \ldots, x_n)$、各变量的值和各自的不确定度，用数值微分方法自动计算 $\delta f$ 和各变量的贡献比例。

**提示代码框架：**

```python
import numpy as np

def error_propagation(f, values, uncertainties, h=1e-8):
    """
    f: 函数, 接受一个列表/数组作为输入
    values: 各变量的中心值 [x1, x2, ..., xn]
    uncertainties: 各变量的不确定度 [dx1, dx2, ..., dn]
    h: 数值微分步长
    """
    n = len(values)
    values = np.array(values, dtype=float)
    uncertainties = np.array(uncertainties, dtype=float)
    
    partials = np.zeros(n)
    for i in range(n):
        v_plus = values.copy()
        v_minus = values.copy()
        v_plus[i] += h
        v_minus[i] -= h
        partials[i] = (f(v_plus) - f(v_minus)) / (2 * h)
    
    contributions = (partials * uncertainties) ** 2
    delta_f = np.sqrt(np.sum(contributions))
    fractions = contributions / np.sum(contributions)
    
    return delta_f, fractions

# 测试：用例题中的通量公式
def flux(params):
    dp, T = params  # 压差 (Pa), 温度 (K)
    alpha = 20000 / 8.314
    C = 2.5e-10 / 50e-6
    return C * dp * np.exp(-alpha / T)

delta_J, fracs = error_propagation(
    flux,
    [300e3, 353],     # 值
    [5e3, 2]          # 不确定度
)

J_val = flux([300e3, 353])
print(f"J = {J_val:.4e}")
print(f"delta_J = {delta_J:.4e}")
print(f"相对不确定度 = {delta_J/J_val*100:.2f}%")
print(f"压差贡献 = {fracs[0]*100:.1f}%")
print(f"温度贡献 = {fracs[1]*100:.1f}%")
````

### 练习 5：思考题

一位同学声称："我的实验误差很小，因为我每次实验只差 $0.1%$ 。" 但你发现他从来只做一次实验，然后重复读数三次来声称"高重复性"。

(a) 他的做法有什么问题？

(b) "重复读数"和"重复实验"有什么区别？

(c) 他的做法能发现系统误差吗？能充分估计随机误差吗？

**答案提示：** 他混淆了"仪器重复性"（同一样品的重复读数）和"实验重复性"（独立的重复实验）。重复读数只能估计仪器的读数噪声，不能捕捉样品制备、环境变化、膜片差异等更重要的随机误差来源。更不能发现系统误差。这是初学者最常犯的错误之一——三次"重复"读数给人以虚假的信心。

---

## 本课要点回顾

1. **系统误差**像一台偏了的秤——它使你的所有测量值系统性地偏离真值，且不会因重复实验而消除。只有校准和改进方法才能解决。

2. **随机误差**像你颤抖的手——它使每次测量值在真值附近随机波动。它可以通过多次测量取平均来压缩（$\propto 1/\sqrt{n}$）。

3. **误差传播的RSS公式** $\delta f = \sqrt{\sum_i (\partial f / \partial x_i \cdot \delta x_i)^2}$ 是全微分法（第11课）在实验科学中最重要的应用之一。它要求各误差源独立。

4. **分析各项贡献比例**才是误差分析的真正价值所在——它告诉你把改进精力投在哪里。

5. **有效数字规则**是误差分析的简化版。正确做法是：中间过程保留全部精度，最终结果的有效数字由不确定度决定。

6. **实验设计三纪律**：区分独立/响应/控制变量，一次只改一个变量，每个条件至少重复三次。

7. **误差不是敌人，而是诚实的语言。** 一个不带误差棒的数据点不是更干净的数据——它是不完整的数据。
