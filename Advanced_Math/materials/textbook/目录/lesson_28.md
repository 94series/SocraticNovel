## 前置知识

为了跟上这节课，你至少需要熟悉：

- **齐次二阶常系数ODE的解法**（第26–27课）：你应该能熟练求解 $ay'' + by' + cy = 0$，包括特征根为两个不同实根、重根、以及共轭复根三种情况。

- **线性代数中的 $A\mathbf{x} = \mathbf{b}$**（第1–8课）：非齐次ODE的解的结构和非齐次线性方程组 $A\mathbf{x} = \mathbf{b}$ 的解的结构在深层是同一件事——通解 = 齐次解 + 特解。如果你还记得这个结构，这节课会格外通透。

## 1. 从密闭到开放：一个膜科学家的日常

在前几课中，我们研究的是齐次方程 $ay'' + by' + cy = 0$，右边恒为零。用物理的语言说：系统是**封闭**的，没有外部输入。弹簧被拨动后自由振动，逐渐归于静止。

但在真实的实验室里，事情几乎从来不是这样。

想象一张致密膜将一个反应器分隔成两侧。膜内有一种溶质，浓度为 $c(x)$，其中 $x$ 是膜厚方向的坐标。膜内同时发生着三件事：

1. **扩散**——溶质沿浓度梯度扩散，贡献一个 $D \, c''$ 的项（Fick第二定律）。
2. **一级反应消耗**——溶质在膜内被化学反应消耗，速率正比于浓度：$-kc$。
3. **外部持续供给**——膜的上游侧不断有新的溶质分子渗入，提供一个持续的"源"，记为 $S(x)$。

把这三件事写成一个方程：

$$
D \, c''(x) - k \, c(x) = -S(x)
$$

整理一下形式：

$$
c'' - \frac{k}{D} \, c = -\frac{S(x)}{D}
$$

右边不再是零。它是一个已知的函数——外部供给。

这就是**非齐次ODE**。

$$
ay'' + by' + cy = G(x)
$$

其中 $G(x) \neq 0$ 是给定的函数，被称为**驱动项**、**强迫项**或**非齐次项**。它代表了来自系统外部的输入——可以是外力、电压源、浓度源、热源。

如果说齐次方程描述的是系统"被扰动后的自由响应"，那非齐次方程描述的是系统"在持续外部驱动下的行为"。前者是拨了一下钟摆看它怎么摆，后者是有人一直在推钟摆。

这个区别至关重要。因为真实的世界几乎总是在被什么东西驱动着。

## 2. 非齐次方程解的结构：最深刻的洞察

在冲进具体的求解技术之前，我要先告诉你一件事。这件事是本课最重要的内容——比任何具体的计算技巧都重要。

> **定理.** 非齐次方程 $ay'' + by' + cy = G(x)$ 的通解为：
>
> $$
> y = y_h + y_p
> $$
>
> 其中 $y_h$ 是**齐次方程** $ay'' + by' + cy = 0$ 的通解（又叫**互补解**），$y_p$ 是非齐次方程的**任意一个特解**。

翻译成中文：

**通解 = 齐次通解 + 一个特解。**

这和你在线性代数里学过的东西一模一样。$A\mathbf{x} = \mathbf{b}$ 的通解是什么？是 $\mathbf{x} = \mathbf{x}_h + \mathbf{x}_p$，其中 $\mathbf{x}_h$ 是零空间 $\text{Nul}(A)$ 中的任意向量（即 $A\mathbf{x} = \mathbf{0}$ 的解），$\mathbf{x}_p$ 是 $A\mathbf{x} = \mathbf{b}$ 的一个特解。

这不是巧合。二阶线性ODE和线性方程组具有完全相同的代数结构——因为"线性"这个词在两个语境中意味着同一件事。

**为什么 $y = y_h + y_p$ 是对的？** 证明极为简洁。

设 $y$ 是非齐次方程的任意一个解，$y_p$ 是某个已知的特解。考察它们的差 $w = y - y_p$。由于线性运算的性质：

$$
aw'' + bw' + cw = a(y'' - y_p'') + b(y' - y_p') + c(y - y_p)
$$

$$
= \big(ay'' + by' + cy\big) - \big(ay_p'' + by_p' + cy_p\big) = G(x) - G(x) = 0
$$

所以 $w = y - y_p$ 满足齐次方程！也就是说，$w$ 属于 $y_h$。

因此 $y = y_p + w = y_p + y_h$。

反过来，如果 $y_h$ 是齐次解，$y_p$ 是特解，那么 $y = y_h + y_p$ 满足：

$$
a(y_h + y_p)'' + b(y_h + y_p)' + c(y_h + y_p) = \underbrace{(ay_h'' + by_h' + cy_h)}_{=\,0} + \underbrace{(ay_p'' + by_p' + cy_p)}_{=\,G(x)} = G(x) \quad \checkmark
$$

证毕。

**这个结构的物理意义.** 齐次解 $y_h$ 描述系统的"自由响应"——它包含两个任意常数（由初始条件决定），代表系统的内在振荡、衰减或增长模式。特解 $y_p$ 描述系统对外部驱动的"被迫响应"——它完全由驱动函数 $G(x)$ 决定，不包含任何任意常数。

总响应 = 自由响应 + 被迫响应。

在有阻尼的系统中，自由响应 $y_h$ 通常会随时间衰减（因为特征根有负实部）。当 $t \to \infty$ 后，自由响应消失，系统的行为完全由被迫响应 $y_p$ 主导。工程师把 $y_h$ 称为**瞬态**（transient），把 $y_p$ 称为**稳态**（steady state）。你给一个电路接上交流电源，一开始电流乱跳（瞬态），过一会儿后电流稳定在一个固定的振荡模式上（稳态）——那个稳态就是特解 $y_p$。

所以：**我们只需要两样东西。** 齐次解我们已经会求了（第26–27课）。剩下的全部问题归结为：**如何找到一个特解 $y_p$？**

## 3. 待定系数法：像侦探拼图一样猜特解

我们先学习最直觉、最好用的方法——**待定系数法**（method of undetermined coefficients）。

核心思想极其简单：**先猜特解的形式，再把待定的系数代入方程，让等式成立来确定系数。**

这听起来不够"系统"，但它之所以管用，是因为有一个强有力的观察：**某些类型的函数在求导后不会改变自身的"类型"。** 多项式求导后还是多项式（低一阶）。$e^{kx}$ 求导后还是 $e^{kx}$ 的倍数。$\cos kx$ 和 $\sin kx$ 求导后互变但始终在 $\{\cos kx, \sin kx\}$ 家族内。

这意味着，如果驱动项 $G(x)$ 属于这些"封闭"的函数族，那么特解 $y_p$ 很可能也属于同一个族——因为把 $y_p$ 及其导数代入左边 $ay_p'' + by_p' + cy_p$ 后，结果必须等于 $G(x)$，而如果所有运算都在同一个族内进行，等式就有可能成立。

让我们通过三个逐步升级的例子来掌握这个方法。

---

### 例题 1：多项式驱动

**求解** $y'' + y' - 2y = x^2$。

**第一步：求齐次解 $y_h$。** 特征方程 $r^2 + r - 2 = 0$，因式分解为 $(r+2)(r-1) = 0$，根为 $r = 1, -2$。

$$
y_h = c_1 e^x + c_2 e^{-2x}
$$

**第二步：猜特解形式。** 右边 $G(x) = x^2$ 是二次多项式。猜 $y_p$ 也是二次多项式：

$$
y_p = Ax^2 + Bx + C
$$

（三个待定系数——$A$、$B$、$C$。）

**第三步：代入方程，匹配系数。** 计算导数：

$$
y_p' = 2Ax + B, \qquad y_p'' = 2A
$$

代入 $y'' + y' - 2y = x^2$：

$$
2A + (2Ax + B) - 2(Ax^2 + Bx + C) = x^2
$$

展开，按 $x$ 的幂次整理：

$$
-2Ax^2 + (2A - 2B)x + (2A + B - 2C) = x^2
$$

现在让左右两边对应幂次的系数相等：

$$
x^2: \quad -2A = 1 \implies A = -\frac{1}{2}
$$

$$
x^1: \quad 2A - 2B = 0 \implies B = A = -\frac{1}{2}
$$

$$
x^0: \quad 2A + B - 2C = 0 \implies -1 - \frac{1}{2} - 2C = 0 \implies C = -\frac{3}{4}
$$

**特解：**

$$
y_p = -\frac{1}{2}x^2 - \frac{1}{2}x - \frac{3}{4}
$$

**通解：**

$$
y = c_1 e^x + c_2 e^{-2x} - \frac{1}{2}x^2 - \frac{1}{2}x - \frac{3}{4}
$$

注意发生了什么：我们把"解ODE"这件看似困难的事，转化为了"解一个关于系数的线性方程组"——一件你在第1课就已经学会的事情。这就是待定系数法的魔力。

---

### 例题 2：指数函数驱动

**求解** $y'' + 4y = e^{3x}$。

**齐次解：** $r^2 + 4 = 0$，$r = \pm 2i$。

$$
y_h = c_1 \cos 2x + c_2 \sin 2x
$$

**猜特解：** 右边是 $e^{3x}$，所以猜 $y_p = Ae^{3x}$。

**代入：** $y_p'' = 9Ae^{3x}$，所以

$$
9Ae^{3x} + 4Ae^{3x} = e^{3x} \implies 13A = 1 \implies A = \frac{1}{13}
$$

**通解：**

$$
y = c_1\cos 2x + c_2 \sin 2x + \frac{1}{13}e^{3x}
$$

这个例子干净到几乎不需要思考。但别让它的简洁蒙蔽了你——这里有一个隐藏的陷阱，我们马上就会遇到。

---

### 例题 3：三角函数驱动

**求解** $y'' + y' - 2y = \sin x$。

**齐次解：** 和例题1一样，$y_h = c_1 e^x + c_2 e^{-2x}$。

**猜特解：** 右边是 $\sin x$。这里有一个关键细节——你不能只猜 $y_p = A\sin x$，因为 $\sin x$ 求导后会变出 $\cos x$，而 $\cos x$ 也需要被"吸收"。所以必须猜：

$$
y_p = A\cos x + B\sin x
$$

**代入：** 计算导数：

$$
y_p' = -A\sin x + B\cos x, \qquad y_p'' = -A\cos x - B\sin x
$$

代入方程：

$$
(-A\cos x - B\sin x) + (-A\sin x + B\cos x) - 2(A\cos x + B\sin x) = \sin x
$$

整理——收集 $\cos x$ 和 $\sin x$ 的系数：

$$
\cos x: \quad -A + B - 2A = -3A + B = 0
$$

$$
\sin x: \quad -B - A - 2B = -A - 3B = 1
$$

从第一个方程：$B = 3A$。代入第二个：$-A - 9A = 1$，所以 $A = -\frac{1}{10}$，$B = -\frac{3}{10}$。

**通解：**

$$
y = c_1 e^x + c_2 e^{-2x} - \frac{1}{10}\cos x - \frac{3}{10}\sin x
$$

**理解这个结果。** 注意特解 $y_p = -\frac{1}{10}\cos x - \frac{3}{10}\sin x$ 的振幅是 $\sqrt{(1/10)^2 + (3/10)^2} = \sqrt{1/10} \approx 0.316$。驱动力 $\sin x$ 的振幅是 1。系统对驱动力的响应被"缩小"了——这是因为驱动频率和系统固有频率不匹配，系统没有产生共振。

## 4. 陷阱与修复：当猜测撞上齐次解

现在来看待定系数法的一个经典陷阱。

**例题 4.** 求解 $y'' + y = \sin x$。

**齐次解：** $r^2 + 1 = 0$，$r = \pm i$。

$$
y_h = c_1\cos x + c_2 \sin x
$$

**按照正常流程猜特解：** $y_p = A\cos x + B\sin x$。

但是等等——你猜的 $y_p$ 长得和 $y_h$ 一模一样！$A\cos x + B\sin x$ 已经是齐次方程的解了，它代入左边 $y'' + y$ 一定等于零，绝不可能等于 $\sin x$。

**我们的猜测失败了。**

这就像侦探根据线索锁定了一个嫌疑人，却发现此人有完美的不在场证明。怎么办？不是放弃，而是修正猜测。

**修复方法：把猜测的特解乘以 $x$。**

改猜：

$$
y_p = x(A\cos x + B\sin x) = Ax\cos x + Bx\sin x
$$

现在来算。$y_p' = A\cos x - Ax\sin x + B\sin x + Bx\cos x$。

$$
y_p'' = -2A\sin x - Ax\cos x + 2B\cos x - Bx\sin x
$$

代入 $y'' + y$：

$$
(-2A\sin x - Ax\cos x + 2B\cos x - Bx\sin x) + (Ax\cos x + Bx\sin x) = \sin x
$$

含 $x\cos x$ 和 $x\sin x$ 的项互相抵消（这正是乘以 $x$ 的精妙之处），剩下：

$$
-2A\sin x + 2B\cos x = \sin x
$$

所以 $-2A = 1$，$2B = 0$，即 $A = -\frac{1}{2}$，$B = 0$。

$$
y_p = -\frac{1}{2}x\cos x
$$

**通解：**

$$
y = c_1\cos x + c_2\sin x - \frac{1}{2}x\cos x
$$

**理解这个结果的物理意义。** 特解 $y_p = -\frac{1}{2}x\cos x$ 的振幅随 $x$（时间）**线性增长**。这就是**共振**！驱动频率（$\sin x$ 的频率 $\omega = 1$）恰好等于系统的固有频率（特征根 $\pm i$ 对应 $\omega_0 = 1$）。每一次推都恰好在系统运动的同一个相位施加，能量不断累积，振幅无限增大。

共振的标志是特解中出现了 $x$ 乘以三角函数。而 $x$ 的出现，数学上正是因为我们不得不把原始猜测乘以 $x$ 来避开齐次解。所以**共振不是一个额外的物理概念——它自然地从待定系数法的结构中涌现出来。**

> **规则总结.** 如果你猜的 $y_p$ 的任何一项恰好是齐次解的一部分，就把整个 $y_p$ 乘以 $x$。如果乘一次 $x$ 后仍然和齐次解重叠（比如齐次解中有重根对应的 $xe^{rx}$），就再乘一次 $x$，用 $x^2$。

## 5. 叠加原理：线性的终极武器

如果驱动项不是单一类型，而是几种的叠加呢？比如：

$$
y'' - 4y = xe^x + \cos 2x
$$

线性的魔力在这里大放异彩。

> **叠加原理.** 如果 $y_{p_1}$ 是 $ay'' + by' + cy = G_1(x)$ 的特解，$y_{p_2}$ 是 $ay'' + by' + cy = G_2(x)$ 的特解，那么 $y_{p_1} + y_{p_2}$ 是 $ay'' + by' + cy = G_1(x) + G_2(x)$ 的特解。

证明只有一行——直接把 $y_{p_1} + y_{p_2}$ 代入左边，利用线性运算的分配律，两部分分别产生 $G_1$ 和 $G_2$，加起来得到 $G_1 + G_2$。

这意味着什么？**你可以把一个复杂的驱动力拆解为简单的成分，分别求解，再把结果加起来。** 这就像把一束白光分解为光谱——每种颜色单独分析，结果叠加就是完整的响应。

在工程中，这个原理无处不在。一个电路同时接收多个频率的信号？对每个频率分别计算响应，然后叠加。一张膜同时受到多种溶质的驱动？对每种溶质分别求解，然后叠加。

**叠加原理只对线性方程成立。** 对于非线性方程 $y'' + y^2 = G(x)$，两个特解之和一般不再是特解。这就是线性理论如此珍贵的原因——它让你能把复杂问题拆解为简单部分。

## 6. 待定系数法速查表

让我把所有情况整理成一张表，供你随时查阅。

设方程为 $ay'' + by' + cy = G(x)$。

| 驱动项 $G(x)$ 的类型 | 猜测的特解形式 $y_p$ |
|:-:|:-:|
| $P_n(x)$（$n$次多项式） | $x^s(A_n x^n + A_{n-1}x^{n-1} + \cdots + A_0)$ |
| $e^{\alpha x}$ | $x^s A e^{\alpha x}$ |
| $\cos\beta x$ 或 $\sin\beta x$ | $x^s(A\cos\beta x + B\sin\beta x)$ |
| $e^{\alpha x}P_n(x)$ | $x^s e^{\alpha x}(A_n x^n + \cdots + A_0)$ |
| $e^{\alpha x}\cos\beta x$ 或 $e^{\alpha x}\sin\beta x$ | $x^s e^{\alpha x}(A\cos\beta x + B\sin\beta x)$ |
| $e^{\alpha x}P_n(x)\cos\beta x$ 或同类 | $x^s e^{\alpha x}(Q_n\cos\beta x + R_n\sin\beta x)$ |

其中 $s$ 的取值规则：$s = 0$ 如果猜测的形式不包含齐次解的任何项；$s = 1$ 如果有重叠（乘一个 $x$）；$s = 2$ 如果乘一个 $x$ 后仍然重叠（对应重根情况）。

**使用流程：**
1. 求齐次通解 $y_h$。
2. 看 $G(x)$ 属于表中哪种类型，写出猜测的 $y_p$。
3. 检查 $y_p$ 是否与 $y_h$ 的任何项重叠——如果重叠，乘以 $x$（或 $x^2$）。
4. 代入方程，匹配系数，解出待定常数。
5. 通解 = $y_h + y_p$。

## 7. 精讲例题：膜内扩散-反应方程

**例题 5.** 回到开篇的膜问题。设膜内稳态浓度分布满足：

$$
c'' - 4c = -8
$$

其中 $c(x)$ 是膜内溶质浓度，$x \in [0, L]$。右边 $-8$ 代表均匀的外部供给。边界条件 $c(0) = 5$，$c(L) = 1$（我们先取 $L = 1$）。

**第一步：齐次解。** $r^2 - 4 = 0$，$r = \pm 2$。

$$
c_h = c_1 e^{2x} + c_2 e^{-2x}
$$

**第二步：猜特解。** $G(x) = -8$ 是常数（零次多项式）。猜 $c_p = A$（常数）。

代入：$0 - 4A = -8$，所以 $A = 2$。

$$
c_p = 2
$$

**第三步：通解。**

$$
c(x) = c_1 e^{2x} + c_2 e^{-2x} + 2
$$

**第四步：用边界条件确定常数。**

$$
c(0) = c_1 + c_2 + 2 = 5 \implies c_1 + c_2 = 3
$$

$$
c(1) = c_1 e^2 + c_2 e^{-2} + 2 = 1 \implies c_1 e^2 + c_2 e^{-2} = -1
$$

这是关于 $c_1, c_2$ 的 $2 \times 2$ 线性方程组。解之（数值上 $e^2 \approx 7.389$，$e^{-2} \approx 0.135$）：

$$
c_1 \approx \frac{-1 - 3 \times 0.135}{7.389 - 0.135} \approx \frac{-1.405}{7.254} \approx -0.194
$$

$$
c_2 \approx 3 - (-0.194) = 3.194
$$

**理解结果。** 特解 $c_p = 2$ 就是"如果膜无限长、且没有边界效应时的平衡浓度"——外部供给的速率恰好被反应消耗的速率平衡。齐次解部分 $c_h$ 是边界条件引起的偏离——在两端浓度被"钉住"的约束下，浓度分布偏离均匀值 2，形成一条向两端弯曲的曲线。

## 8. 常数变易法：更通用的武器

待定系数法强大但有局限：它只在 $G(x)$ 是多项式、指数函数、正弦余弦（以及它们的乘积组合）时管用。如果 $G(x) = \tan x$，或 $G(x) = \frac{1}{x}$，或 $G(x) = \sec x$，待定系数法就无能为力——因为 $\tan x$ 的导数 $\sec^2 x$ 不属于 $\tan x$ 的"家族"，你永远猜不出一个封闭的形式。

这时你需要一把万能钥匙：**常数变易法**（variation of parameters）。

**核心思想.** 我们已经有了齐次解 $y_h = c_1 y_1 + c_2 y_2$，其中 $c_1, c_2$ 是常数，$y_1, y_2$ 是两个线性无关的齐次解。

大胆的想法：把常数 $c_1, c_2$ **替换为未知函数** $u_1(x), u_2(x)$，猜测非齐次方程的特解形式为：

$$
y_p = u_1(x)\,y_1(x) + u_2(x)\,y_2(x)
$$

这个"变"的是"参数"——从常数变成函数，所以叫"常数变易法"。

**推导.** 对 $y_p$ 求导：

$$
y_p' = u_1' y_1 + u_1 y_1' + u_2' y_2 + u_2 y_2'
$$

这里有四项，但 $u_1, u_2$ 是两个未知函数，而我们的约束只有一个（$y_p$ 必须满足方程）。两个未知函数、一个约束——自由度太多了。所以我们可以额外施加一个条件来简化计算。选择：

$$
u_1' y_1 + u_2' y_2 = 0 \qquad \text{（附加条件）}
$$

有了这个条件，$y_p' = u_1 y_1' + u_2 y_2'$，再求导：

$$
y_p'' = u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2''
$$

代入方程 $ay_p'' + by_p' + cy_p = G(x)$，利用 $y_1, y_2$ 各自满足齐次方程（即 $ay_i'' + by_i' + cy_i = 0$），大量项会消去，最终简化为：

$$
a(u_1' y_1' + u_2' y_2') = G(x)
$$

连同附加条件，我们得到一个关于 $u_1', u_2'$ 的 $2 \times 2$ 线性方程组：

$$
\begin{cases} u_1' y_1 + u_2' y_2 = 0 \\ u_1' y_1' + u_2' y_2' = \dfrac{G(x)}{a} \end{cases}
$$

用 Cramer 法则或直接消元解出 $u_1'$ 和 $u_2'$，再积分得到 $u_1$ 和 $u_2$。

**关键公式.** 设 $W = y_1 y_2' - y_2 y_1'$ 是 $y_1, y_2$ 的 **Wronskian 行列式**（它不为零，因为 $y_1, y_2$ 线性无关）。则：

$$
u_1' = -\frac{y_2 \, G(x)}{a \, W}, \qquad u_2' = \frac{y_1 \, G(x)}{a \, W}
$$

$$
u_1 = -\int \frac{y_2 \, G(x)}{a \, W}\,dx, \qquad u_2 = \int \frac{y_1 \, G(x)}{a \, W}\,dx
$$

**例题 6.** 求解 $y'' + y = \tan x$，$0 < x < \pi/2$。

**齐次解：** $y_1 = \sin x$，$y_2 = \cos x$。

**Wronskian：**

$$
W = \sin x \cdot (-\sin x) - \cos x \cdot \cos x = -\sin^2 x - \cos^2 x = -1
$$

**计算 $u_1'$ 和 $u_2'$：** （这里 $a = 1$，$G(x) = \tan x$）

$$
u_1' = -\frac{\cos x \cdot \tan x}{1 \cdot (-1)} = \frac{\cos x \cdot \tan x}{1} = \sin x
$$

$$
u_2' = \frac{\sin x \cdot \tan x}{1 \cdot (-1)} = -\sin x \cdot \tan x = -\frac{\sin^2 x}{\cos x}
$$

**积分：**

$$
u_1 = \int \sin x \, dx = -\cos x
$$

$$
u_2 = -\int \frac{\sin^2 x}{\cos x}\,dx = -\int \frac{1 - \cos^2 x}{\cos x}\,dx = -\int (\sec x - \cos x)\,dx
$$

$$
= -\ln|\sec x + \tan x| + \sin x
$$

**特解：**

$$
y_p = u_1 y_1 + u_2 y_2 = (-\cos x)\sin x + (-\ln|\sec x + \tan x| + \sin x)\cos x
$$

$$
= -\sin x\cos x + \sin x \cos x - \cos x \ln|\sec x + \tan x|
$$

$$
= -\cos x \ln|\sec x + \tan x|
$$

**通解：**

$$
y = c_1 \sin x + c_2 \cos x - \cos x \ln|\sec x + \tan x|
$$

注意：这个 $\tan x$ 的驱动，待定系数法完全无法处理——你猜什么形式都不行。但常数变易法把问题归结为积分，而积分（不管多丑陋）至少在原理上总是可以做的。

**常数变易法 vs 待定系数法：何时用哪个？**

- 如果 $G(x)$ 是多项式、指数、正弦余弦（及其乘积）→ 用待定系数法。更快，更不容易出错。
- 如果 $G(x)$ 是其他任何东西 → 用常数变易法。更通用，但计算量更大。
- 如果你不确定 → 先试待定系数法。如果猜不出形式，就退回常数变易法。

## 9. 更深一层：为什么这一切都和线性有关

让我退后一步，帮你看清更大的图景。

整节课的每一个结论——$y = y_h + y_p$ 的结构、叠加原理、待定系数法的可行性——都建立在一个基础之上：**方程是线性的。**

"线性"在这里意味着什么？意味着我们可以定义一个**算子** $L$：

$$
L[y] = ay'' + by' + cy
$$

然后你能验证 $L$ 满足：

$$
L[y_1 + y_2] = L[y_1] + L[y_2], \qquad L[\alpha y] = \alpha L[y]
$$

这和线性变换 $T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v})$ 是完全一样的性质。

齐次方程 $L[y] = 0$ 的解集就是 $L$ 的**核**（null space / kernel）——它是一个向量空间（二阶时维度为2）。

非齐次方程 $L[y] = G$ 的解集就是这个核的一个**平移**——取任一特解 $y_p$，加上核中的任意元素 $y_h$。

这和 $A\mathbf{x} = \mathbf{b}$ 的解 = $\text{Nul}(A)$ 的平移是同一个几何图像。

= 过原点的子空间                   = 过零函数的函数空间

= x_p + Nul(A)                    = y_p + {y_h}

## 10. 总结

让我把这节课的核心内容提炼出来。

**非齐次方程 $ay'' + by' + cy = G(x)$ 的通解 = 齐次通解 $y_h$ + 一个特解 $y_p$。** 这个结构是线性叠加原理的直接推论，和线性代数中 $A\mathbf{x} = \mathbf{b}$ 的解结构完全平行。

**待定系数法**是找 $y_p$ 的快速工具——猜一个和 $G(x)$ 同类型的函数形式，代入方程匹配系数。适用于 $G(x)$ 是多项式、指数函数、正弦余弦及其组合的情况。如果猜测和齐次解重叠，乘以 $x$。

**叠加原理**说：如果驱动力是几个成分的和，可以分别求解再相加。这是线性系统分析的基石。

**常数变易法**是更通用的武器——把齐次解中的常数替换为未知函数，通过求解一个 $2 \times 2$ 方程组和两个积分来找特解。适用于任何连续的 $G(x)$，但计算量通常更大。

**物理图景**：齐次解 = 自由响应（瞬态），特解 = 被迫响应（稳态）。共振发生在驱动频率等于固有频率时——数学上表现为特解猜测与齐次解重叠，需要乘以 $x$，导致振幅随时间线性增长。

## 练习

**练习1.** 求解 $y'' - 3y' + 2y = e^{5x}$。

**提示：** 齐次解：$r^2 - 3r + 2 = 0$，$(r-1)(r-2) = 0$，$y_h = c_1 e^x + c_2 e^{2x}$。猜 $y_p = Ae^{5x}$。代入：$25A - 15A + 2A = 1$，$12A = 1$，$A = 1/12$。通解：$y = c_1 e^x + c_2 e^{2x} + \frac{1}{12}e^{5x}$。

---

**练习2.** 求解 $y'' + 4y = \cos 2x$。注意这是一个共振情况——检查你的猜测是否与齐次解冲突！

**提示：** 齐次解：$y_h = c_1\cos 2x + c_2 \sin 2x$。正常猜测 $A\cos 2x + B\sin 2x$ 与齐次解完全重叠！改猜 $y_p = x(A\cos 2x + B\sin 2x)$。代入化简后得 $-4A\sin 2x + 4B\cos 2x = \cos 2x$，所以 $A = 0$，$B = 1/4$。特解 $y_p = \frac{1}{4}x\sin 2x$——振幅随 $x$ 线性增长，共振！

---

**练习3.** 用叠加原理求解 $y'' + y = e^{2x} + 3\sin x$。

**提示：** 分别处理两个驱动项。(i) $y'' + y = e^{2x}$：猜 $y_{p_1} = Ae^{2x}$，代入得 $5A = 1$，$A = 1/5$。(ii) $y'' + y = 3\sin x$：猜 $A\cos x + B\sin x$ 与齐次解冲突，改猜 $x(A\cos x + B\sin x)$，解得 $y_{p_2} = -\frac{3}{2}x\cos x$。通解：$y = c_1\cos x + c_2\sin x + \frac{1}{5}e^{2x} - \frac{3}{2}x\cos x$。

---

**练习4.** 对方程 $y'' + y = \sec x$（$0 < x < \pi/2$），解释为什么待定系数法不适用，然后用常数变易法写出求 $u_1', u_2'$ 的方程组（不需要完成最终的积分）。

**提示：** $\sec x$ 的导数是 $\sec x \tan x$，再求导会出现 $\sec^3 x$ 等，不断产生新类型——没有封闭的"族"。常数变易法：设 $y_1 = \sin x$，$y_2 = \cos x$，$W = -1$。方程组为 $u_1'\sin x + u_2'\cos x = 0$，$u_1'\cos x - u_2'\sin x = \sec x$。解得 $u_1' = 1$，$u_2' = -\tan x$。所以 $u_1 = x$，$u_2 = \ln|\cos x|$。特解 $y_p = x\sin x + \cos x \ln|\cos x|$。

**练习5（开放题）.** 一张膜内发生扩散、线性反应和外部周期性供给：

$$
D\,c'' - kc = -S_0 \cos(\omega x)
$$

其中 $D, k, S_0, \omega$ 都是正常数。写出通解的形式（不需要确定边界条件对应的常数），并讨论：当 $\omega$ 满足什么条件时，特解的振幅最大？这在物理上意味着什么？

**提示：** 齐次解涉及 $e^{\pm\sqrt{k/D}\,x}$。猜 $c_p = A\cos\omega x + B\sin\omega x$，代入得 $(-D\omega^2 - k)A = -S_0$，$B = 0$。所以 $A = \frac{S_0}{D\omega^2 + k}$。注意分母 $D\omega^2 + k > 0$ 恒成立——不会有"共振"（因为这不是振荡系统，特征根是实的），但当 $\omega \to 0$ 时振幅最大（等于 $S_0/k$，就是反应消耗恰好平衡供给的稳态浓度）。$\omega$ 越大，供给的空间变化越快，膜内扩散来不及响应，浓度波动被"抹平"——这就是扩散的低通滤波效应。
