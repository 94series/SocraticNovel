## 1. 为什么需要数值方法？一个诚实的开场白

在第34-36课里，我们用分离变量法和Fourier级数成功地求解了扩散方程、波动方程等经典PDE。那些解析解是美丽的——它们以无穷级数的形式揭示了物理过程的内在结构。

但让我们诚实地面对现实。

你的膜研究中遇到的实际问题，几乎没有一个能用分离变量法求解。原因很多：

- **几何形状不规则。** 你的中空纤维膜不是平板，它是弯曲的圆管。膜组件有复杂的进料通道。
- **系数不是常数。** 扩散系数可能随浓度变化。渗透率可能随位置变化（膜不均匀）。
- **边界条件复杂。** 膜面上的浓度边界条件取决于传质系数，传质系数又取决于流场……
- **方程是非线性的。** 浓差极化模型、渗透压模型、结垢动力学——几乎都是非线性的。

面对这些问题，解析方法就像一把精致但太小的钥匙——它只能开特定的锁。我们需要一把万能钥匙。

**有限差分法**就是这样一把万能钥匙。它的思路朴素得令人吃惊：既然连续的导数太难处理，那就把它替换成离散的差商。既然整个空间和时间上的方程太难解，那就只在有限多个网格点上列方程。最终，一个偏微分方程变成了一组**代数方程**——矩阵方程——计算机对此驾轻就熟。

这不是近似的无奈。这是计算科学的基石。今天世界上绝大多数的工程计算——天气预报、飞机设计、半导体模拟、核反应堆分析——归根结底都在做同一件事：把PDE翻译成计算机能执行的加减乘除。

## 2. 差商：导数的离散化身

### 2.1 从极限定义说起

回忆导数的定义：

$$
f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h}
$$

这个极限要求 $h \to 0$。但如果我们**故意不取极限**——让 $h$ 停留在一个小但有限的值——我们就得到了导数的**近似**，叫做**差商**。

这正是有限差分的全部出发点。不同的差商形式对应不同的近似策略。

### 2.2 三种基本差分格式

设 $f(x)$ 是一个光滑函数，$h$ 是一个小的正数（叫做**步长**或**网格间距**）。

$$
f'(x) \approx \frac{f(x + h) - f(x)}{h}
$$

三种格式都近似同一个东西——$f'(x)$——但精度不同。怎么比较精度？**用Taylor展开。**

### 2.3 精度分析：Taylor展开揭示一切

将 $f(x + h)$ 在 $x$ 处Taylor展开：

$$
f(x + h) = f(x) + h f'(x) + \frac{h^2}{2} f''(x) + \frac{h^3}{6} f'''(x) + \cdots
$$

类似地：

$$
f(x - h) = f(x) - h f'(x) + \frac{h^2}{2} f''(x) - \frac{h^3}{6} f'''(x) + \cdots
$$

**前向差分的误差：**

$$
\frac{f(x+h) - f(x)}{h} = f'(x) + \frac{h}{2} f''(x) + O(h^2)
$$

截断误差的**主项**是 $\frac{h}{2} f''(x)$，是 $h$ 的一阶量。所以前向差分是**一阶精度**，记作 $O(h)$。步长减半，误差大约减半。

**中心差分的误差：** 用 $f(x+h) - f(x-h)$：

$$
f(x+h) - f(x-h) = 2h f'(x) + \frac{2h^3}{6} f'''(x) + O(h^5)
$$

除以 $2h$：

$$
\frac{f(x+h) - f(x-h)}{2h} = f'(x) + \frac{h^2}{6} f'''(x) + O(h^4)
$$

截断误差的主项是 $\frac{h^2}{6} f'''(x)$，是 $h$ 的**二阶**量。所以中心差分是**二阶精度**，$O(h^2)$。步长减半，误差大约减为原来的四分之一。精度翻倍！

> **直觉：** 中心差分之所以更好，是因为它"对称地"使用了 $x$ 两侧的信息。前向和后向差分只看一侧，自然少了一半信息。这就像称体重——左脚右脚各踩一半，总比单脚站更稳。

### 2.4 二阶导数的差分：构建核心武器

扩散方程中出现的是**二阶导数** $f''(x)$。怎么用差分近似它？

最直接的方法：对 $f'$ 再做一次差分。把中心差分"嵌套"一次：

$$
f''(x) \approx \frac{f'(x + h/2) - f'(x - h/2)}{h}
$$

其中 $f'(x + h/2) \approx \frac{f(x+h) - f(x)}{h}$，$f'(x - h/2) \approx \frac{f(x) - f(x-h)}{h}$。代入：

$$
f''(x) \approx \frac{f(x+h) - 2f(x) + f(x-h)}{h^2}
$$

这就是二阶导数的**中心差分公式**。它是二阶精度的（可以用Taylor展开验证）。

用Taylor展开验证：

$$
f(x+h) + f(x-h) = 2f(x) + h^2 f''(x) + \frac{h^4}{12} f^{(4)}(x) + O(h^6)
$$

所以：

$$
\frac{f(x+h) - 2f(x) + f(x-h)}{h^2} = f''(x) + \frac{h^2}{12} f^{(4)}(x) + O(h^4)
$$

误差是 $O(h^2)$。

> **停下来欣赏一下这个公式。** $f(x+h) - 2f(x) + f(x-h)$ 是什么？它度量的是"$f$ 在 $x$ 处偏离直线的程度"。如果 $f$ 是直线（$f'' = 0$），那么 $f(x+h)$ 和 $f(x-h)$ 的平均值恰好等于 $f(x)$，这个表达式为零。如果 $f$ 向上弯曲（$f'' > 0$），两端的平均值超过中间值，表达式为正。这正是"曲率"的离散版本。

## 3. 网格化：给时空画上棋盘格

### 3.1 建立网格

现在让我们来"离散化"扩散方程。考虑一维扩散：

$$
\frac{\partial u}{\partial t} = D \frac{\partial^2 u}{\partial x^2}, \quad 0 \leq x \leq L, \quad t \geq 0
$$

带有初始条件 $u(x, 0) = f(x)$ 和边界条件（比如 $u(0, t) = u(L, t) = 0$）。

第一步：在空间和时间上建立**离散网格**。

把空间 $[0, L]$ 分成 $N$ 等份，每份宽度 $\Delta x = L/N$。空间网格点为：

$$
x_i = i \cdot \Delta x, \quad i = 0, 1, 2, \ldots, N
$$

把时间分成步长为 $\Delta t$ 的离散时刻：

$$
t_n = n \cdot \Delta t, \quad n = 0, 1, 2, \ldots
$$

于是时空被画成了一张棋盘格：

```

t
^
|
t₃ ──●──●──●──●──●──●──
|  |  |  |  |  |
t₂ ──●──●──●──●──●──●──
|  |  |  |  |  |
t₁ ──●──●──●──●──●──●──
|  |  |  |  |  |
t₀ ──●──●──●──●──●──●──→ x
x₀ x₁ x₂ x₃ x₄ x₅

```

我们用 $u_i^n$ 来表示 $u(x_i, t_n)$ 的**近似值**——即在网格点 $(x_i, t_n)$ 上解的数值估计。

上标是时间指标，下标是空间指标。这个记号需要习惯一下。

### 3.2 将PDE翻译成代数方程

现在在每一个**内部网格点** $(x_i, t_n)$ 上用差分公式替换PDE中的导数。

时间导数 $\frac{\partial u}{\partial t}$——用**前向差分**：

$$
\frac{\partial u}{\partial t}\bigg|_{(x_i, t_n)} \approx \frac{u_i^{n+1} - u_i^n}{\Delta t}
$$

空间二阶导数 $\frac{\partial^2 u}{\partial x^2}$——用**中心差分**：

$$
\frac{\partial^2 u}{\partial x^2}\bigg|_{(x_i, t_n)} \approx \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{(\Delta x)^2}
$$

把它们代入扩散方程：

$$
\frac{u_i^{n+1} - u_i^n}{\Delta t} = D \cdot \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{(\Delta x)^2}
$$

整理：

$$
\boxed{u_i^{n+1} = u_i^n + r \left( u_{i+1}^n - 2u_i^n + u_{i-1}^n \right)}
$$

其中我们定义了一个关键参数：

$$
r = \frac{D \cdot \Delta t}{(\Delta x)^2}
$$

这就是**FTCS格式**（Forward Time, Central Space——时间前向、空间中心差分）。这个公式有一个美妙的特点：**等号左边只有一个未知量 $u_i^{n+1}$，等号右边都是已知量**（来自第 $n$ 个时间层）。所以一旦知道了第 $n$ 层的所有值，就可以直接算出第 $n+1$ 层。

```

时间层 n+1:         ● u_i^{n+1} (要算的)
/ | 
时间层 n:    ● ─── ● ─── ●
u_{i-1}^n  u_i^n  u_{i+1}^n  (已知的)

```

这叫做**显式格式**（explicit scheme）——"显"在于每一步的计算都是显而易见的直接运算，不需要解方程组。

## 4. 精讲例题：手算扩散的前几步

### 4.1 问题设置

让我们做一个完整的小例子。

**物理问题：** 一根长 $L = 1$ 的均匀棒，两端温度固定为零，初始温度分布为一个三角形脉冲：中间热、两端冷。扩散系数 $D = 1$。

$$
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}, \quad 0 \leq x \leq 1, \quad t > 0
$$

边界条件：$u(0, t) = u(1, t) = 0$。

初始条件：

$$
u(x, 0) = \begin{cases} 2x & 0 \leq x \leq 0.5 \\ 2(1-x) & 0.5 < x \leq 1 \end{cases}
$$

### 4.2 建立网格

取 $N = 4$（空间分4份），所以 $\Delta x = 0.25$。

空间网格点：$x_0 = 0, \; x_1 = 0.25, \; x_2 = 0.50, \; x_3 = 0.75, \; x_4 = 1.00$。

取 $\Delta t = 0.01$，则 $r = D \cdot \Delta t / (\Delta x)^2 = 1 \times 0.01 / 0.0625 = 0.16$。

### 4.3 初始化

初始条件给出第 $n = 0$ 层的值：

$$
u_0^0 = 0, \quad u_1^0 = 0.50, \quad u_2^0 = 1.00, \quad u_3^0 = 0.50, \quad u_4^0 = 0
$$

边界条件永远不变：$u_0^n = 0$，$u_4^n = 0$（对所有 $n$）。

### 4.4 第一步迭代：$n = 0 \to n = 1$

对内部点 $i = 1, 2, 3$ 应用FTCS公式 $u_i^{n+1} = u_i^n + r(u_{i+1}^n - 2u_i^n + u_{i-1}^n)$。

**$i = 1$：**

$$
u_1^1 = u_1^0 + 0.16(u_2^0 - 2u_1^0 + u_0^0) = 0.50 + 0.16(1.00 - 1.00 + 0) = 0.50 + 0 = 0.500
$$

**$i = 2$：**

$$
u_2^1 = u_2^0 + 0.16(u_3^0 - 2u_2^0 + u_1^0) = 1.00 + 0.16(0.50 - 2.00 + 0.50) = 1.00 + 0.16(-1.00) = 0.840
$$

**$i = 3$：**

$$
u_3^1 = u_3^0 + 0.16(u_4^0 - 2u_3^0 + u_2^0) = 0.50 + 0.16(0 - 1.00 + 1.00) = 0.50 + 0 = 0.500
$$

第一个时间步之后：

$$
u^1 = [0, \; 0.500, \; 0.840, \; 0.500, \; 0]
$$

**观察：** 中间的峰值从1.000降到了0.840——热量正在从高温区向两端扩散。两侧的温度没变——因为对称性，热量流入和流出恰好平衡。

### 4.5 第二步迭代：$n = 1 \to n = 2$

**$i = 1$：**

$$
u_1^2 = 0.500 + 0.16(0.840 - 1.000 + 0) = 0.500 + 0.16(-0.160) = 0.500 - 0.0256 = 0.474
$$

**$i = 2$：**

$$
u_2^2 = 0.840 + 0.16(0.500 - 1.680 + 0.500) = 0.840 + 0.16(-0.680) = 0.840 - 0.109 = 0.731
$$

**$i = 3$：** 由对称性，$u_3^2 = u_1^2 = 0.474$。

第二步之后：

$$
u^2 = [0, \; 0.474, \; 0.731, \; 0.474, \; 0]
$$

峰值继续下降（0.840 → 0.731），两侧也开始降低（0.500 → 0.474）——热量持续向边界流失。

```

u

|    / | 
0.8 |   /  *  \             n=1
|  / / | \ 
0.6 | / /  *  \ \           n=2
|/ /  / \  \ 
0.4 |*/  /   \  *
|  /     \

|/         \   
0.0 *───────────*───→ x

````

这个粗糙的ASCII图展示了温度分布如何随时间变平——扩散方程在数值上的行为和我们的物理直觉完全吻合。

## 5. 矩阵形式：线性代数的华丽回归

### 5.1 从迭代公式到矩阵乘法

让我们用更优雅的方式写出FTCS格式。定义向量 $\mathbf{u}^n = [u_1^n, u_2^n, \ldots, u_{N-1}^n]^T$（只包含内部节点，边界值已知）。

FTCS公式可以写成矩阵形式：

$$
\mathbf{u}^{n+1} = A \, \mathbf{u}^n
$$

其中 $A$ 是一个**三对角矩阵**：

$$
A = \begin{bmatrix} 1-2r & r & 0 & \cdots & 0 \\ r & 1-2r & r & \cdots & 0 \\ 0 & r & 1-2r & \cdots & 0 \\ \vdots & & \ddots & \ddots & r \\ 0 & 0 & \cdots & r & 1-2r \end{bmatrix}
$$

对于上面 $N = 4$ 的例子（3个内部节点），这就是：

$$
A = \begin{bmatrix} 1 - 2(0.16) & 0.16 & 0 \\ 0.16 & 1 - 2(0.16) & 0.16 \\ 0 & 0.16 & 1 - 2(0.16) \end{bmatrix} = \begin{bmatrix} 0.68 & 0.16 & 0 \\ 0.16 & 0.68 & 0.16 \\ 0 & 0.16 & 0.68 \end{bmatrix}
$$

验证第一步：

$$
\mathbf{u}^1 = A \mathbf{u}^0 = \begin{bmatrix} 0.68 & 0.16 & 0 \\ 0.16 & 0.68 & 0.16 \\ 0 & 0.16 & 0.68 \end{bmatrix} \begin{bmatrix} 0.50 \\ 1.00 \\ 0.50 \end{bmatrix} = \begin{bmatrix} 0.500 \\ 0.840 \\ 0.500 \end{bmatrix}
$$

与手算一致。

**注意发生了什么：** 扩散方程的时间演化变成了**反复用同一个矩阵乘以一个向量**。$n$ 步之后：

$$
\mathbf{u}^n = A^n \, \mathbf{u}^0
$$

如果你还记得第21课关于矩阵幂和特征值的讨论，你会立刻想到：$A^n$ 的长期行为由 $A$ 的特征值决定。如果 $A$ 的所有特征值的绝对值都不超过1，那么 $\|\mathbf{u}^n\|$ 不会增长——数值解是**稳定的**。如果某个特征值的绝对值超过1，$\|\mathbf{u}^n\|$ 会指数增长——数值解**爆炸**了。

这就把我们带向了稳定性分析。

### 5.2 线性代数的回响

请在这里停下来，感受一下各部分知识如何汇合：

- **第3课**的导数定义 → 差商
- **第9课**的Taylor展开 → 截断误差分析

## 6. 显式格式 vs 隐式格式

### 6.1 显式格式的优势与软肋

FTCS格式是**显式**的：$u_i^{n+1}$ 可以从第 $n$ 层的值直接算出来，不需要解方程组。这使得编程极其简单——一个循环就搞定。

但它有一个致命的弱点：**稳定性条件**。我们很快会看到，必须满足 $r \leq 1/2$，即：

$$
\Delta t \leq \frac{(\Delta x)^2}{2D}
$$

如果空间网格很细（$\Delta x$ 很小），时间步长必须**极其微小**。比如 $\Delta x = 0.001$，$D = 1$ 时，$\Delta t \leq 5 \times 10^{-7}$。要模拟1秒的扩散过程，需要两百万个时间步。这在计算上代价极高。

### 6.2 隐式格式：用解方程组换取自由

有没有办法避开这个限制？有。关键是把空间差分**评估在第 $n+1$ 层**而不是第 $n$ 层。

**后向Euler格式**（BTCS: Backward Time, Central Space）：

$$
\frac{u_i^{n+1} - u_i^n}{\Delta t} = D \cdot \frac{u_{i+1}^{n+1} - 2u_i^{n+1} + u_{i-1}^{n+1}}{(\Delta x)^2}
$$

整理：

$$
-r \, u_{i-1}^{n+1} + (1 + 2r) \, u_i^{n+1} - r \, u_{i+1}^{n+1} = u_i^n
$$

注意：**等号左边有三个未知量**——都在第 $n+1$ 层。要求出第 $n+1$ 层的所有值，必须同时求解一个**线性方程组**。

矩阵形式：

$$
B \, \mathbf{u}^{n+1} = \mathbf{u}^n
$$

其中：

$$
B = \begin{bmatrix} 1+2r & -r & 0 & \cdots & 0 \\ -r & 1+2r & -r & \cdots & 0 \\ 0 & -r & 1+2r & \cdots & 0 \\ \vdots & & \ddots & \ddots & -r \\ 0 & 0 & \cdots & -r & 1+2r \end{bmatrix}
$$

每个时间步都需要解一个三对角线性方程组 $B \mathbf{u}^{n+1} = \mathbf{u}^n$。

**这是否太昂贵了？** 不！三对角方程组有一个高效的算法——**Thomas算法**（LU分解的特殊形式），计算量只是 $O(N)$，和显式格式一样快。

而且——隐式格式是**无条件稳定**的。$r$ 取多大都不会爆炸。你可以用大时间步长，粗糙但稳定地推进。

### 6.3 Crank-Nicolson格式：兼得鱼与熊掌

能不能同时享受高精度和无条件稳定？可以。**Crank-Nicolson格式**在第 $n$ 层和第 $n+1$ 层的空间差分之间取平均：

$$
\frac{u_i^{n+1} - u_i^n}{\Delta t} = \frac{D}{2} \left[ \frac{u_{i+1}^{n+1} - 2u_i^{n+1} + u_{i-1}^{n+1}}{(\Delta x)^2} + \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{(\Delta x)^2} \right]
$$

这个格式：
- 时间精度：$O(\Delta t^2)$（比前向/后向Euler的 $O(\Delta t)$ 好一阶）
- 空间精度：$O(\Delta x^2)$
- 无条件稳定
- 每步需要解三对角方程组

Crank-Nicolson是扩散方程数值求解的"黄金标准"——在膜科学的数值模拟中，大量的传质模型就是用它或它的变体来求解的。

````

三种格式的"信息流"对比：

显式 (FTCS):           隐式 (BTCS):          Crank-Nicolson:

n+1: ● (直接算出)    n+1: ●──●──● (需解方程)  n+1: ●──●──● (需解方程)
/|\                  |  |  |                 /|\ |  |  |
n:  ● ● ● (已知)     n:  ● (已知)           n:  ● ● ● (已知+已知)

```

## 7. CFL条件：信息不能跑得比网格快

### 7.1 稳定性灾难

如果你在显式格式中选了一个太大的 $r$——比如 $r = 0.6$——会发生什么？

让我们用 $r = 0.6$ 重新计算上面的例子。FTCS公式变成：

$$
u_i^{n+1} = u_i^n + 0.6(u_{i+1}^n - 2u_i^n + u_{i-1}^n) = -0.2 \, u_i^n + 0.6 \, u_{i+1}^n + 0.6 \, u_{i-1}^n
$$

注意 $u_i^n$ 前面的系数变成了 **$-0.2$**——负数！

这意味着如果 $u_i^n$ 很大而邻居较小，新值 $u_i^{n+1}$ 可能变成**负的**。温度变负？物理上毫无意义。几步之后，正负值剧烈交替，振幅指数增长——数值解彻底崩溃。

```

稳定 (r=0.16):                    不稳定 (r=0.6):
u                                  u
|   \                              |     *
|    \  (逐渐衰减)                  |   * | *
|     \                            | *   |   *  (剧烈振荡!)
|      ___                        |*    |    *
+--------→ t                       +--*--+--*--→ t

```

### 7.2 稳定性条件的推导

**方法一：系数非负原则。** 观察FTCS公式：

$$
u_i^{n+1} = r \cdot u_{i-1}^n + (1 - 2r) \cdot u_i^n + r \cdot u_{i+1}^n
$$

$u_i^{n+1}$ 是三个旧值的**加权平均**——前提是所有权重都非负。$r > 0$ 保证了前后两项的系数为正。中间项的系数 $1 - 2r$ 必须也非负：

$$
1 - 2r \geq 0 \implies r \leq \frac{1}{2}
$$

即 $\frac{D \Delta t}{(\Delta x)^2} \leq \frac{1}{2}$。

如果所有权重非负且总和为1（这里 $r + (1-2r) + r = 1$，确实如此），那么新值一定落在旧值的范围内——不可能出现超出原有范围的振荡。这叫做**极值原理**的离散版本，而扩散方程的连续解也满足极值原理。

**方法二：特征值分析。** 矩阵 $A$ 的特征值可以精确计算（利用三对角Toeplitz矩阵的公式，这与第22课的离散Fourier分析有关）：

$$
\lambda_k = 1 - 4r \sin^2\left(\frac{k\pi}{2N}\right), \quad k = 1, 2, \ldots, N-1
$$

稳定性要求所有特征值满足 $|\lambda_k| \leq 1$。最"危险"的是 $k = N-1$（最高频模式），此时 $\sin^2(\cdot)$ 接近1：

$$
\lambda_{N-1} \approx 1 - 4r
$$

要求 $|1 - 4r| \leq 1$，即 $0 \leq r \leq 1/2$。

与系数非负原则给出的结论完全一致。

### 7.3 CFL条件的物理直觉

稳定性条件 $r = D\Delta t / (\Delta x)^2 \leq 1/2$ 有一个深刻的物理含义。

扩散过程中，信息（浓度扰动）在时间 $\Delta t$ 内传播的典型距离是 $\sqrt{D \cdot \Delta t}$（扩散长度，回忆第34课）。这个距离不应该超过网格间距 $\Delta x$——否则信息在一个时间步内"跳过"了中间的网格点，差分格式就无法正确追踪它了。

$$
\sqrt{D \cdot \Delta t} \lesssim \Delta x \implies \Delta t \lesssim \frac{(\Delta x)^2}{D}
$$

这正是稳定性条件（差一个常数因子）。

> **信息不能跑得比网格快。** 这就是CFL条件（Courant-Friedrichs-Lewy条件）的物理精髓。它最初由Richard Courant和他的两位学生在1928年发表——那时候连电子计算机都还没有。他们用纯粹的数学洞察预见到：**任何离散化方案，如果允许物理信息在一步内传播超过一个网格间距，就注定是不稳定的。**

这个条件在更一般的情况下（对流方程、波动方程）有不同的具体形式，但核心思想始终是同一个：**网格速度必须跟上物理速度。**

## 8. 精讲例题2：从伪代码到完整实现

让我们用伪代码写出一个完整的FTCS求解器，然后讨论每一步。

```

输入: L, N, D, t_final
计算: dx = L/N
选择: dt 使得 r = D*dt/dx^2 <= 0.5（取 r = 0.4 留安全余量）
dt = r * dx^2 / D
n_steps = ceil(t_final / dt)

初始化: x[0..N], u[0..N]  # 空间网格和初始条件
对 i = 0 到 N:
x[i] = i * dx
u[i] = 初始条件 f(x[i])

时间推进:
对 n = 1 到 n_steps:
u_new[0] = 左边界值    # 边界条件
u_new[N] = 右边界值    # 边界条件

```
    对 i = 1 到 N-1:      # 内部节点
        u_new[i] = u[i] + r * (u[i+1] - 2*u[i] + u[i-1])
    
    u = u_new              # 更新
```

输出: u（最终时刻的数值解）

```

这段伪代码的核心——内层循环——只有一行公式。**整个有限差分法的精髓就浓缩在这一行里。** 外面的一切（网格设置、边界条件处理、时间循环）都只是脚手架。

> **给膜科学研究生的实际建议：** 在实际研究中，你不必从头写有限差分求解器。Python的 `scipy.integrate.solve_ivp` 可以处理空间半离散化后的ODE系统；或者使用现成的PDE求解器如 FiPy、FEniCS。但**理解底层原理**是不可替代的——当模拟结果出现异常时（振荡、发散、不守恒），只有理解了差分格式和稳定性条件的人才能诊断问题。

## 9. 超越基础：边界条件的处理

### 9.1 Dirichlet边界条件

上面的例子用的是最简单的情况：边界值固定（$u = 0$）。这叫**Dirichlet边界条件**。处理方法很直接——在每个时间步强制设定边界点的值。

### 9.2 Neumann边界条件——膜科学中更常见

在膜科学中，你经常遇到的不是"边界上的浓度已知"，而是"边界上的通量已知"。比如膜面上的溶质通量由渗透行为决定：

$$
-D \frac{\partial C}{\partial x}\bigg|_{x=0} = J_s
$$

这是**Neumann边界条件**——给定的是导数，不是函数值本身。

怎么离散化？用**虚拟节点**（ghost node）。在边界外增加一个虚拟网格点 $x_{-1}$，用中心差分近似边界处的导数：

$$
\frac{\partial C}{\partial x}\bigg|_{x=0} \approx \frac{C_1 - C_{-1}}{2\Delta x} = -\frac{J_s}{D}
$$

从中解出虚拟节点的值 $C_{-1} = C_1 + \frac{2\Delta x \, J_s}{D}$，然后在差分公式中用这个值替代 $C_{-1}$。

### 9.3 Robin边界条件

有时边界条件是函数值和导数的线性组合：

$$
D \frac{\partial C}{\partial x}\bigg|_{x=L} + k(C - C_b) = 0
$$

这叫**Robin边界条件**（或混合边界条件）。在膜科学中，它出现在浓差极化的边界层模型中：膜面处的传质通量等于对流通量减去扩散通量。处理方法类似——用差分近似导数，代入边界条件，得到一个关于边界点值的代数方程。

## 10. 第四部分全景回顾：偏微分方程的三条道路

在过去五课中，我们走过了PDE的一段长长的旅程。让我们站在山顶回望。

### 解析方法（第34-36课）

**分离变量 + Fourier级数：** 把PDE分解成一系列ODE，用特征函数展开解。

- **优势：** 给出封闭形式的解，揭示物理结构（模态、衰减率、波速）
- **局限：** 只适用于线性、常系数、规则几何、简单边界条件

### 数值方法（第37课）

**有限差分法：** 把连续问题离散化为代数方程组，用计算机求解。

- **优势：** 适用于任意几何、变系数、非线性、复杂边界条件
- **局限：** 只给出数值（不是公式），需要关注稳定性和精度

### 第三条路：两者结合

在实际研究中，最有力的策略往往是把两者结合：

1. 先用解析方法理解问题的基本结构和参数依赖关系
2. 再用数值方法处理解析方法无法应对的复杂性
3. 用解析解作为数值代码的验证基准（benchmark）

比如：你用分离变量法得到平板膜扩散问题的解析解，然后用有限差分代码求解同一个问题，比较结果。如果吻合得很好，你就可以信任你的代码，然后放心地用它去求解那些没有解析解的复杂问题（比如加上浓差极化、结垢动力学、非均匀膜结构……）。

```

```
     PDE问题
    /       \
   /         \
```

简单几何        复杂几何
线性、常系数     非线性、变系数
规则边界        不规则边界
|              |
解析方法         数值方法
(Fourier级数)   (有限差分/有限元)
|              |
物理洞察         定量预测
参数依赖         设计优化
\              /

```

├── 无条件稳定
└── 实际计算的"黄金标准"

B · u^{n+1} = u^n (隐式)
→ 稳定性 ↔ 矩阵特征值 ↔ 线性代数！

CFL条件的物理直觉:
信息传播速度 ≤ 网格传播速度
√(D·Δt) ≤ Δx

```

### 练习5（综合题）：膜中的扩散

一张厚度 $L = 100$ μm 的聚合物膜，溶质在膜中的扩散系数 $D = 10^{-12}$ m²/s。初始时刻膜内溶质浓度为零，$t = 0$ 时膜的进料侧表面浓度突变为 $C_0 = 1$ mol/m³（透过侧浓度保持为零）。

(a) 这是什么类型的边界条件？写出完整的PDE和边界/初始条件。

(b) 如果你用 $N = 50$ 个网格点对膜厚度进行离散化，$\Delta x$ 是多少？用FTCS格式时，$\Delta t$ 的最大允许值是多少？

(c) 估算溶质"扩散穿透"整个膜所需的特征时间 $\tau \sim L^2 / D$。你需要多少个时间步才能模拟到这个时间？

> **答案提示：** (a) 左边界Dirichlet $C(0,t) = C_0$，右边界Dirichlet $C(L,t) = 0$，初始条件 $C(x,0) = 0$。(b) $\Delta x = 2 \times 10^{-6}$ m，$\Delta t_{\max} = (\Delta x)^2/(2D) = (4 \times 10^{-12})/(2 \times 10^{-12}) = 2$ s。(c) $\tau = L^2/D = (10^{-4})^2/10^{-12} = 10^4$ s ≈ 2.8小时。时间步数 ≈ $10^4/2 = 5000$ 步——完全可行。
