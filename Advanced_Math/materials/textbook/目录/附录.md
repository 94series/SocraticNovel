## 附录B：常用积分表

> 以下公式中，\(C\) 为任意积分常数；\(a, b, n\) 为常数且满足注明的约束条件。

### B.1 基本积分

\[
\int k\,\mathrm{d}x = kx + C
\]

\[
\int x^n\,\mathrm{d}x = \frac{x^{n+1}}{n+1} + C, \quad n \neq -1
\]

\[
\int \frac{1}{x}\,\mathrm{d}x = \ln|x| + C
\]

\[
\int e^x\,\mathrm{d}x = e^x + C
\]

\[
\int a^x\,\mathrm{d}x = \frac{a^x}{\ln a} + C, \quad a > 0,\; a \neq 1
\]

### B.2 三角函数积分

\[
\int \sin x\,\mathrm{d}x = -\cos x + C
\]

\[
\int \cos x\,\mathrm{d}x = \sin x + C
\]

\[
\int \tan x\,\mathrm{d}x = -\ln|\cos x| + C = \ln|\sec x| + C
\]

\[
\int \sec^2 x\,\mathrm{d}x = \tan x + C
\]

\[
\int \csc^2 x\,\mathrm{d}x = -\cot x + C
\]

\[
\int \sec x \tan x\,\mathrm{d}x = \sec x + C
\]

\[
\int \csc x \cot x\,\mathrm{d}x = -\csc x + C
\]

\[
\int \sec x\,\mathrm{d}x = \ln|\sec x + \tan x| + C
\]

\[
\int \sin^2 x\,\mathrm{d}x = \frac{x}{2} - \frac{\sin 2x}{4} + C
\]

\[
\int \cos^2 x\,\mathrm{d}x = \frac{x}{2} + \frac{\sin 2x}{4} + C
\]

### B.3 含根式/反三角函数的积分

\[
\int \frac{1}{\sqrt{a^2 - x^2}}\,\mathrm{d}x = \arcsin\frac{x}{a} + C, \quad a > 0
\]

\[
\int \frac{1}{a^2 + x^2}\,\mathrm{d}x = \frac{1}{a}\arctan\frac{x}{a} + C, \quad a \neq 0
\]

\[
\int \frac{1}{x\sqrt{x^2 - a^2}}\,\mathrm{d}x = \frac{1}{a}\operatorname{arcsec}\frac{|x|}{a} + C, \quad a > 0
\]

\[
\int \frac{1}{\sqrt{x^2 \pm a^2}}\,\mathrm{d}x = \ln\left|x + \sqrt{x^2 \pm a^2}\right| + C
\]

### B.4 指数与对数积分

\[
\int x e^{ax}\,\mathrm{d}x = \frac{e^{ax}}{a^2}(ax - 1) + C
\]

\[
\int x^2 e^{ax}\,\mathrm{d}x = e^{ax}\left(\frac{x^2}{a} - \frac{2x}{a^2} + \frac{2}{a^3}\right) + C
\]

\[
\int e^{ax}\sin bx\,\mathrm{d}x = \frac{e^{ax}(a\sin bx - b\cos bx)}{a^2 + b^2} + C
\]

\[
\int e^{ax}\cos bx\,\mathrm{d}x = \frac{e^{ax}(a\cos bx + b\sin bx)}{a^2 + b^2} + C
\]

\[
\int \ln x\,\mathrm{d}x = x\ln x - x + C
\]

\[
\int x\ln x\,\mathrm{d}x = \frac{x^2}{2}\ln x - \frac{x^2}{4} + C
\]

\[
\int \frac{\ln x}{x}\,\mathrm{d}x = \frac{(\ln x)^2}{2} + C
\]

### B.5 分部积分与递推（常用技巧提示）

分部积分公式：

\[
\int u\,\mathrm{d}v = uv - \int v\,\mathrm{d}u
\]

递推公式示例——\(\sin^n x\) 的积分：

\[
\int \sin^n x\,\mathrm{d}x = -\frac{\sin^{n-1}x\cos x}{n} + \frac{n-1}{n}\int \sin^{n-2}x\,\mathrm{d}x
\]

### B.6 膜科学中常遇到的定积分

高斯积分：

\[
\int_0^{\infty} e^{-x^2}\,\mathrm{d}x = \frac{\sqrt{\pi}}{2}
\]

\[
\int_0^{\infty} x\,e^{-ax^2}\,\mathrm{d}x = \frac{1}{2a}, \quad a > 0
\]

\[
\int_0^{\infty} x^2\,e^{-ax^2}\,\mathrm{d}x = \frac{\sqrt{\pi}}{4a^{3/2}}, \quad a > 0
\]

误差函数的定义：

\[
\operatorname{erf}(z) = \frac{2}{\sqrt{\pi}}\int_0^z e^{-t^2}\,\mathrm{d}t
\]

补误差函数：

\[
\operatorname{erfc}(z) = 1 - \operatorname{erf}(z) = \frac{2}{\sqrt{\pi}}\int_z^{\infty} e^{-t^2}\,\mathrm{d}t
\]

## 附录C：Laplace 变换表

### C.1 定义

Laplace 变换的定义为：

\[
F(s) = \mathcal{L}\{f(t)\} = \int_0^{\infty} e^{-st} f(t)\,\mathrm{d}t, \quad s > 0
\]

逆变换记为 \(f(t) = \mathcal{L}^{-1}\{F(s)\}\)。

### C.2 基本性质

| 性质 | 时域 \(f(t)\) | 频域 \(F(s)\) |
|:---|:---:|:---:|
| 线性 | \(af(t) + bg(t)\) | \(aF(s) + bG(s)\) |
| 时域微分 | \(f'(t)\) | \(sF(s) - f(0)\) |
| 二阶微分 | \(f''(t)\) | \(s^2F(s) - sf(0) - f'(0)\) |
| \(n\) 阶微分 | \(f^{(n)}(t)\) | \(s^nF(s) - \displaystyle\sum_{k=0}^{n-1}s^{n-1-k}f^{(k)}(0)\) |
| 时域积分 | \(\displaystyle\int_0^t f(\tau)\,\mathrm{d}\tau\) | \(\dfrac{F(s)}{s}\) |
| 频移 | \(e^{at}f(t)\) | \(F(s-a)\) |
| 时移 | \(f(t-a)\,u(t-a)\) | \(e^{-as}F(s)\) |
| 尺度变换 | \(f(at)\) | \(\dfrac{1}{a}F\!\left(\dfrac{s}{a}\right)\) |
| 卷积 | \((f * g)(t) = \displaystyle\int_0^t f(\tau)g(t-\tau)\,\mathrm{d}\tau\) | \(F(s)\cdot G(s)\) |
| 频域微分 | \(t\,f(t)\) | \(-F'(s)\) |
| 初值定理 | \(f(0^+)\) | \(\displaystyle\lim_{s\to\infty} sF(s)\) |
| 终值定理 | \(\displaystyle\lim_{t\to\infty}f(t)\) | \(\displaystyle\lim_{s\to 0} sF(s)\) |

### C.3 常用变换对

| 序号 | \(f(t)\)（\(t \ge 0\)） | \(F(s) = \mathcal{L}\{f(t)\}\) |
|:---:|:---:|:---:|
| 1 | \(1\) | \(\dfrac{1}{s}\) |
| 2 | \(t\) | \(\dfrac{1}{s^2}\) |
| 3 | \(t^n\)（\(n = 1,2,3,\ldots\)） | \(\dfrac{n!}{s^{n+1}}\) |
| 4 | \(t^{1/2}\) | \(\dfrac{\sqrt{\pi}}{2s^{3/2}}\) |
| 5 | \(t^{n-1/2}\)（\(n = 1,2,\ldots\)） | \(\dfrac{(2n)!\,\sqrt{\pi}}{n!\,2^{2n}\,s^{n+1/2}}\) |
| 6 | \(e^{at}\) | \(\dfrac{1}{s-a}\) |
| 7 | \(t\,e^{at}\) | \(\dfrac{1}{(s-a)^2}\) |
| 8 | \(t^n e^{at}\) | \(\dfrac{n!}{(s-a)^{n+1}}\) |
| 9 | \(\sin(bt)\) | \(\dfrac{b}{s^2+b^2}\) |
| 10 | \(\cos(bt)\) | \(\dfrac{s}{s^2+b^2}\) |
| 11 | \(e^{at}\sin(bt)\) | \(\dfrac{b}{(s-a)^2+b^2}\) |
| 12 | \(e^{at}\cos(bt)\) | \(\dfrac{s-a}{(s-a)^2+b^2}\) |
| 13 | \(t\sin(bt)\) | \(\dfrac{2bs}{(s^2+b^2)^2}\) |
| 14 | \(t\cos(bt)\) | \(\dfrac{s^2-b^2}{(s^2+b^2)^2}\) |
| 15 | \(\sinh(bt)\) | \(\dfrac{b}{s^2-b^2}\) |
| 16 | \(\cosh(bt)\) | \(\dfrac{s}{s^2-b^2}\) |
| 17 | \(u(t-a)\)（单位阶跃） | \(\dfrac{e^{-as}}{s}\) |
| 18 | \(\delta(t)\)（Dirac δ 函数） | \(1\) |
| 19 | \(\delta(t-a)\) | \(e^{-as}\) |
| 20 | \(\operatorname{erfc}\!\left(\dfrac{a}{2\sqrt{t}}\right)\) | \(\dfrac{1}{s}\,e^{-a\sqrt{s}}\) |
| 21 | \(\dfrac{1}{\sqrt{\pi t}}\,\exp\!\left(-\dfrac{a^2}{4t}\right)\) | \(\dfrac{e^{-a\sqrt{s}}}{\sqrt{s}}\) |
| 22 | \(\dfrac{a}{2\sqrt{\pi t^3}}\,\exp\!\left(-\dfrac{a^2}{4t}\right)\) | \(e^{-a\sqrt{s}}\) |

> **提示**：变换对 20–22 在非稳态膜扩散（半无限平板 Fick 第二定律求解）中尤为常用。

## 附录E：推荐进阶阅读（按主题分类，附简短评语）

### E.1 数学基础

1. **G. Strang,《Linear Algebra and Its Applications》（第 4 版）**
   经典线性代数入门。对理解膜传质中的矩阵方法与特征值问题极有帮助；叙述清晰，例题丰富。

2. **D. C. Lay, S. R. Lay & J. J. McDonald,《Linear Algebra and Its Applications》（第 6 版，全球版）**
   本书配套教材之一。强调概念与应用并重，"数值笔记"栏目对计算实现有指导意义。

3. **J. Stewart,《Calculus: Early Transcendentals》（第 8 版）**
   本书配套教材之一。多元微积分、级数与微分方程章节为膜传质建模提供了完整的数学工具链。

4. **E. Kreyszig,《Advanced Engineering Mathematics》（第 10 版）**
   覆盖 ODE、PDE、Laplace 变换、Fourier 分析与数值方法，是工程研究生的标准参考书。PDE 章节的分离变量法对非稳态扩散问题尤为实用。

5. **R. Haberman,《Applied Partial Differential Equations》（第 5 版）**
   对热传导方程（即扩散方程）进行了极为详尽的推导，适合希望深入理解 Fick 第二定律求解的读者。

### E.2 膜科学与技术

6. **R. W. Baker,《Membrane Technology and Applications》（第 3 版）**
   当代膜技术的百科全书级参考。从基础传质理论到工业应用均有覆盖，图文并茂，适合作为案头常备。

7. **M. Mulder,《Basic Principles of Membrane Technology》（第 2 版）**
   经典教材，对溶解-扩散模型和孔流模型的物理图像讲解得非常到位。数学推导适中，适合本科高年级/研究生入门。

8. **J. G. Wijmans & R. W. Baker, "The solution-diffusion model: a review"，*J. Membr. Sci.* 107 (1995) 1–21**
   溶解-扩散模型的经典综述论文，将RO、气体分离与渗透蒸发统一在同一框架下。几乎所有膜科学课程的必读文献。

9. **K. S. Spiegler & O. Kedem, "Thermodynamics of hyperfiltration (reverse osmosis)"，*Desalination* 1 (1966) 311–326**
   不可逆热力学方法处理 RO 的开创性论文。对理解反射系数 \(\sigma\) 的物理意义很有帮助。

## 附录F：全书公式索引（按课次排列）

> 此索引列出各课次核心公式。每条公式标注公式编号、名称及简要说明，方便复习与交叉查阅。

### 第一课：膜过程概论与数学预备

**(F-1.1)** 通量的基本定义：

\[
J = \frac{1}{A_m}\frac{\mathrm{d}V}{\mathrm{d}t}
\]

其中 \(A_m\) 为有效膜面积，\(V\) 为渗透液体积，\(t\) 为时间。

**(F-1.2)** 截留率定义：

\[
R = 1 - \frac{c_p}{c_f}
\]

**(F-1.3)** 回收率定义：

\[
Y = \frac{Q_p}{Q_f}
\]

**(F-1.4)** 质量守恒（单级系统）：

\[
Q_f c_f = Q_p c_p + Q_r c_r
\]

### 第二课：Fick 扩散定律与稳态膜传质

**(F-2.1)** Fick 第一定律（一维）：

\[
J_i = -D_i \frac{\mathrm{d}c_i}{\mathrm{d}x}
\]

**(F-2.2)** 稳态均匀膜中的通量：

\[
J_i = \frac{D_i}{l}(c_{i,0} - c_{i,l}) = \frac{P_i}{l}(p_{i,f} - p_{i,p})
\]

其中 \(P_i = D_i S_i\) 为渗透率，\(S_i\) 为溶解度系数。

**(F-2.3)** 理想选择性：

\[
\alpha_{A/B}^* = \frac{P_A}{P_B} = \frac{D_A}{D_B} \cdot \frac{S_A}{S_B}
\]

### 第三课：非稳态扩散——Fick 第二定律与求解

**(F-3.1)** Fick 第二定律（一维）：

\[
\frac{\partial c}{\partial t} = D\frac{\partial^2 c}{\partial x^2}
\]

**(F-3.2)** 半无限介质解（恒定表面浓度）：

\[
\frac{c(x,t) - c_0}{c_s - c_0} = \operatorname{erfc}\!\left(\frac{x}{2\sqrt{Dt}}\right)
\]

**(F-3.3)** 有限厚度平板解（分离变量法）：

\[
\frac{c(x,t) - c_s}{c_0 - c_s} = \frac{4}{\pi}\sum_{n=0}^{\infty}\frac{(-1)^n}{2n+1}\exp\!\left(-\frac{(2n+1)^2\pi^2 D t}{l^2}\right)\cos\!\left(\frac{(2n+1)\pi x}{l}\right)
\]

**(F-3.4)** 扩散时滞（time lag）：

\[
\theta = \frac{l^2}{6D}
\]

### 第四课：溶解-扩散模型

**(F-4.1)** 溶解-扩散模型——水通量（RO）：

\[
J_w = A(\Delta p - \Delta\Pi)
\]

其中 \(A\) 为水的渗透系数。

**(F-4.2)** 溶质通量：

\[
J_s = B(c_f - c_p)
\]

其中 \(B\) 为溶质渗透系数。

**(F-4.3)** 截留率与操作参数的关系：

\[
R = 1 - \frac{B}{A(\Delta p - \Delta\Pi) + B}
\]

**(F-4.4)** 渗透率 Arrhenius 温度依赖：

\[
P = P_0 \exp\!\left(-\frac{E_p}{RT}\right)
\]

### 第五课：孔流模型与微滤/超滤

**(F-5.1)** Hagen–Poiseuille 方程：

\[
J = \frac{\varepsilon\, r_p^2}{8\mu\tau l}\,\Delta p
\]

其中 \(\varepsilon\) 为孔隙率，\(r_p\) 为孔半径，\(\tau\) 为曲折因子。

**(F-5.2)** Kozeny–Carman 方程（多孔介质）：

\[
J = \frac{\varepsilon^3}{K\mu S_v^2(1-\varepsilon)^2 l}\,\Delta p
\]

**(F-5.3)** 筛分截留（硬球模型）：

\[
\sigma = \left(1 - \frac{r_s}{r_p}\right)^2, \quad r_s < r_p
\]

**(F-5.4)** Kedem–Katchalsky 方程：

\[
J_s = P_s(c_f - c_p) + (1 - \sigma)\bar{c}\,J_v
\]

### 第六课：浓差极化

**(F-6.1)** 边界层浓差极化模型（薄膜理论）：

\[
\frac{c_m - c_p}{c_b - c_p} = \exp\!\left(\frac{J}{k}\right)
\]

其中 \(k\) 为传质系数。

**(F-6.2)** 传质系数的经验关联（层流，Lévêque 近似）：

\[
Sh = 1.85\left(Re\,Sc\,\frac{D_h}{L}\right)^{1/3}
\]

**(F-6.3)** 临界通量概念（简化表达）：

\[
J_{\mathrm{crit}} = k\,\ln\!\left(\frac{c_g}{c_b}\right)
\]

其中 \(c_g\) 为凝胶浓度。

### 第七课：膜污染动力学

**(F-7.1)** 阻力串联模型：

\[
J = \frac{\Delta p}{\mu(R_m + R_f + R_c)}
\]

其中 \(R_m\) 为本征膜阻力，\(R_f\) 为污染层阻力，\(R_c\) 为滤饼层阻力。

**(F-7.2)** 恒压死端过滤的 \(t/V\) 线性化：

\[
\frac{t}{V} = \frac{\mu R_m}{A_m \Delta p} + \frac{\mu \alpha c_b}{2 A_m^2 \Delta p}\,V
\]

其中 \(\alpha\) 为比滤饼阻力。

**(F-7.3)** 堵塞定律的统一形式（Hermia 模型）：

\[
\frac{\mathrm{d}^2 t}{\mathrm{d}V^2} = k_n \left(\frac{\mathrm{d}t}{\mathrm{d}V}\right)^n
\]

| \(n\) 值 | 堵塞机理 |
|:---:|:---|
| 2 | 完全堵塞 |
| 3/2 | 标准堵塞 |
| 1 | 中间堵塞 |
| 0 | 滤饼过滤 |

### 第八课：渗透蒸发与膜蒸馏

**(F-8.1)** 渗透蒸发通量（溶解-扩散）：

\[
J_i = \frac{P_i}{l}\left(x_i\gamma_i p_i^{\mathrm{sat}} - y_i p_p\right)
\]

**(F-8.2)** 膜蒸馏（直接接触式，DCMD）通量：

\[
J = C_m (p_f^* - p_p^*)
\]

其中 \(p^* = p^{\mathrm{sat}}(T)\) 通过 Antoine 方程或 Clausius–Clapeyron 方程计算。

**(F-8.3)** Antoine 方程：

\[
\log_{10} p^{\mathrm{sat}} = A - \frac{B}{C + T}
\]

### 第九课：气体分离膜

**(F-9.1)** 单组分气体通量：

\[
J_i = \frac{P_i}{l}(p_{i,f} - p_{i,p})
\]

**(F-9.2)** 二元气体分离——级切割率与分离因子：

\[
\theta = \frac{Q_p}{Q_f}, \qquad \alpha^* = \frac{P_A}{P_B}
\]

**(F-9.3)** Robeson 上界（经验关系）：

\[
P_A = k\,\alpha_{A/B}^{\,n}
\]

其中 \(k, n\) 为依赖于气体对的经验常数（\(n < 0\)）。

**(F-9.4)** 二元混合气的实际分离因子：

\[
\alpha_{\mathrm{actual}} = \frac{y_A / y_B}{x_A / x_B}
\]

### 第十课：电膜过程——电渗析与离子交换膜

**(F-10.1)** Nernst–Planck 方程：

\[
J_i = -D_i\frac{\mathrm{d}c_i}{\mathrm{d}x} - \frac{z_i F D_i c_i}{RT}\frac{\mathrm{d}\phi}{\mathrm{d}x}
\]

**(F-10.2)** Donnan 平衡：

\[
\frac{c_i^m}{c_i^s} = \left(\frac{c_j^s}{c_j^m}\right)^{z_j/z_i}
\]

**(F-10.3)** 极限电流密度（Peers 方程）：

\[
i_{\lim} = \frac{z_i F D_i c_b}{\delta(t_i^m - t_i^s)}
\]

其中 \(t_i^m, t_i^s\) 分别为离子在膜中和溶液中的迁移数。

### 第十一课：正渗透与压力延迟渗透

**(F-11.1)** FO 水通量（无 ICP）：

\[
J_w = A(\Delta\Pi - \Delta p)
\]

**(F-11.2)** 内浓差极化（ICP）修正——活性层面向进料（AL-FS）：

\[
J_w = A\left[\pi_D \exp\!\left(-\frac{J_w S}{D}\right) - \pi_F \exp\!\left(\frac{J_w}{k}\right)\right]
\]

其中 \(S = t_s \tau / \varepsilon\) 为结构参数。

**(F-11.3)** PRO 功率密度：

\[
W = J_w \Delta p = A(\Delta\Pi - \Delta p)\Delta p
\]

最大功率密度在 \(\Delta p = \Delta\Pi / 2\) 时取得。

### 第十二课：膜反应器

**(F-12.1)** 管式膜反应器的一维质量守恒（反应物 A）：

\[
\frac{\mathrm{d}(Q c_A)}{\mathrm{d}z} = -r_A A_c - J_A P_m
\]

其中 \(r_A\) 为体积反应速率，\(A_c\) 为截面积，\(P_m\) 为膜的湿润周长。

**(F-12.2)** 平衡移位效应（Le Chatelier 原理的定量表达）——以脱氢反应为例：

\[
K_{eq} = \frac{c_C c_D}{c_A c_B}; \quad \text{移除 } D \Rightarrow Q < K_{eq} \Rightarrow \text{正反应加速}
\]

### 第十三课：膜组件设计与过程优化中的线性代数方法

**(F-13.1)** 多级膜系统的矩阵质量平衡：

\[
\mathbf{A}\mathbf{x} = \mathbf{b}
\]

其中 \(\mathbf{A}\) 为系数矩阵（由各级回收率、截留率构成），\(\mathbf{x}\) 为未知流量/浓度向量，\(\mathbf{b}\) 为已知条件向量。

**(F-13.2)** 灵敏度分析——条件数：

\[
\kappa(\mathbf{A}) = \|\mathbf{A}\|\cdot\|\mathbf{A}^{-1}\|
\]

\(\kappa \gg 1\) 表示系统对参数扰动高度敏感。

**(F-13.3)** 最小二乘参数拟合（正规方程）：

\[
\hat{\mathbf{x}} = (\mathbf{A}^\mathsf{T}\mathbf{A})^{-1}\mathbf{A}^\mathsf{T}\mathbf{b}
\]

### 第十四课：非稳态过程的数值方法

**(F-14.1)** 显式有限差分（FTCS 格式）——Fick 第二定律离散化：

\[
c_j^{n+1} = c_j^n + \frac{D\,\Delta t}{(\Delta x)^2}\left(c_{j+1}^n - 2c_j^n + c_{j-1}^n\right)
\]

稳定性条件：

\[
\frac{D\,\Delta t}{(\Delta x)^2} \le \frac{1}{2}
\]

**(F-14.2)** 隐式格式（Crank–Nicolson）：

\[
-r\,c_{j-1}^{n+1} + (1+2r)\,c_j^{n+1} - r\,c_{j+1}^{n+1} = r\,c_{j-1}^n + (1-2r)\,c_j^n + r\,c_{j+1}^n
\]

其中 \(r = \dfrac{D\,\Delta t}{2(\Delta x)^2}\)。该格式无条件稳定。

**(F-14.3)** 上述隐式格式导出三对角线性方程组 \(\mathbf{T}\mathbf{c}^{n+1} = \mathbf{d}^n\)，可用 Thomas 算法（LU 分解的特例）以 \(O(N)\) 复杂度高效求解。

### 第十五课：多组分传质与 Maxwell–Stefan 方程

**(F-15.1)** Maxwell–Stefan 方程（\(n\) 组分）：

\[
\frac{\mathrm{d}x_i}{\mathrm{d}z} = -\sum_{\substack{j=1\\j\neq i}}^{n}\frac{x_i J_j - x_j J_i}{c_t \mathcal{D}_{ij}}
\]

其中 \(\mathcal{D}_{ij}\) 为 Maxwell–Stefan 扩散系数。

**(F-15.2)** 矩阵形式：

\[
\frac{\mathrm{d}\mathbf{x}}{\mathrm{d}z} = -[\mathbf{B}]^{-1}\frac{\mathbf{J}}{c_t}
\]

其中 \([\mathbf{B}]\) 的元素为：

\[
B_{ii} = \frac{x_i}{\mathcal{D}_{in}} + \sum_{\substack{j=1\\j\neq i}}^{n}\frac{x_j}{\mathcal{D}_{ij}}, \qquad B_{ij} = -x_i\left(\frac{1}{\mathcal{D}_{ij}} - \frac{1}{\mathcal{D}_{in}}\right)
\]

### 第十六课：过程集成与系统级优化

**(F-16.1)** 单级系统的比能耗（RO）：

\[
\mathrm{SEC} = \frac{\Delta p}{Y \eta_p}
\]

其中 \(\eta_p\) 为泵效率。

**(F-16.2)** 热力学最小比能耗（零回收率极限）：

\[
\mathrm{SEC}_{\min} = \frac{\Delta\Pi_f}{\eta_p}
\]

**(F-16.3)** 多目标优化问题的一般形式：

\[
\min_{\mathbf{x}} \quad [f_1(\mathbf{x}),\; f_2(\mathbf{x}),\; \ldots,\; f_k(\mathbf{x})]
\]

\[
\text{s.t.} \quad g_j(\mathbf{x}) \le 0, \quad h_l(\mathbf{x}) = 0
\]

其中 \(f_1\) 可为成本函数，\(f_2\) 可为环境影响指标，等等。Pareto 前沿给出非支配解集。
