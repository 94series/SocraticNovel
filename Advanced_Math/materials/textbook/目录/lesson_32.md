## 前置知识清单

这节课是整本手册的一个重大转折点。从这里开始，我们离开常微分方程（ODE）的世界，进入偏微分方程（PDE）的疆域——那是膜科学理论真正栖居的地方。在跨过这道门槛之前：

- **偏导数**（第15课）：你能对 $f(x, t)$ 分别求 $\partial f/\partial x$ 和 $\partial f/\partial t$，并且理解"固定一个变量对另一个求导"的含义。
- **Fick 第一定律**（你在膜科学课程中一定学过）：扩散通量 $J = -D\,\partial c / \partial x$，物质从高浓度向低浓度方向扩散。如果你没学过，不要紧——我们马上要从头推导它。
- **一阶线性ODE与积分因子**（第26课）：你已经建立了"守恒律 → 微分方程"的推导直觉。
- **稳态扩散方程与边界值问题**（第29课）：你见过 $d^2c/dx^2 = 0$ 和 $d^2c/dx^2 - \phi^2 c = 0$，知道这些ODE描述的是空间中的浓度分布。

这节课要做的事情是：**把时间重新放回来。** 稳态方程是时间被冻结后的简化。真实的膜传质过程——浓差极化的建立、膜内扩散的瞬态、污染物浓度对阶跃扰动的响应——全部需要同时考虑时间和空间。这就是PDE的领地。

## 一、一个微元体的独白——从物质守恒到方程

### 1.1 场景设定

让我们从最基本的物理原则出发。

想象一根细长的管道——或者一张膜的横截面——其中溶质在做一维扩散。$x$ 是空间坐标，$t$ 是时间。$c(x,t)$ 是浓度：在位置 $x$、时刻 $t$ 处每单位体积含有多少克溶质。

我们的目标是推导一个描述 $c(x,t)$ 的方程。我们不打算从天上掉下一个方程——我们要**从零开始建造它**。

### 1.2 取一个微元体

在 $x$ 轴上取一个无穷小的切片：从 $x$ 到 $x + \Delta x$。假设截面积为 $A$（常数）。这个微元体的体积是 $A\Delta x$。

```
          J(x,t)              J(x+Δx, t)
          ───►  │              │  ───►
    通量流入 ──► │   微元体      │ ──► 通量流出
                │  体积 AΔx    │
                │  浓度 c(x,t) │
                │              │
               x              x+Δx
```

现在我们来做这个微元体的**物质守恒（质量平衡）**。

在任意时刻 $t$，微元体内的溶质总量是：

$$
\text{微元体内溶质质量} = c(x,t) \cdot A\Delta x
$$

（浓度 × 体积。我们把浓度近似为在整个 $\Delta x$ 范围内均匀的——当 $\Delta x$ 足够小时，这个近似是精确的。）

### 1.3 进 = 出 + 累积

物质守恒的核心法则：

$$
\text{单位时间内流入的质量} - \text{单位时间内流出的质量} = \text{单位时间内的累积}
$$

或者写成速率的形式：

$$
\text{流入速率} - \text{流出速率} = \text{累积速率}
$$

**流入速率：** 在左界面 $x$ 处，扩散通量为 $J(x,t)$（单位面积单位时间流过的质量）。流经截面积 $A$ 的质量速率是 $J(x,t) \cdot A$。

**流出速率：** 在右界面 $x+\Delta x$ 处，通量为 $J(x+\Delta x, t)$。流出的质量速率是 $J(x+\Delta x, t) \cdot A$。

**累积速率：** 微元体内溶质质量随时间的变化率是 $\frac{\partial}{\partial t}[c(x,t) \cdot A\Delta x] = A\Delta x \frac{\partial c}{\partial t}$。

把它们组装起来：

$$
J(x,t) \cdot A - J(x+\Delta x, t) \cdot A = A\Delta x\frac{\partial c}{\partial t}
$$

两边除以 $A\Delta x$：

$$
\frac{J(x,t) - J(x+\Delta x,t)}{\Delta x} = \frac{\partial c}{\partial t}
$$

左边——看到了吗？左边是一个差商，只不过多了一个负号。如果你把它写成：

$$
-\frac{J(x+\Delta x,t) - J(x,t)}{\Delta x} = \frac{\partial c}{\partial t}
$$

### 1.4 让微元体趋于零——PDE 诞生

现在，让 $\Delta x \to 0$。左边的差商变成导数：

$$
-\frac{\partial J}{\partial x} = \frac{\partial c}{\partial t}
$$

或者重新排列：

$$
\frac{\partial c}{\partial t} = -\frac{\partial J}{\partial x}
$$

停。这个方程值得你凝视十秒钟。

$$
\boxed{\frac{\partial c}{\partial t} = -\frac{\partial J}{\partial x}}
$$

这叫做**连续性方程（continuity equation）**，或者**质量守恒的微分形式**。它什么都没假设——没有假设扩散定律、没有假设反应机制——它只是说：**浓度在某一点随时间变化的速率，等于通量在那一点的空间散度的负值。**

用人话说：**如果流进某一点的物质比流出的多（$\partial J/\partial x < 0$），浓度就在增加（$\partial c/\partial t > 0$）。如果流出的比流进的多，浓度就在减少。**

就这么简单。就这么基本。就这么不可动摇。

### 1.5 代入 Fick 第一定律

现在，我们需要一个"本构关系"——一个告诉我们"通量 $J$ 和浓度 $c$ 之间有什么关系"的物理定律。

**Fick 第一定律：**

$$
J = -D\frac{\partial c}{\partial x}
$$

物质沿浓度梯度的反方向扩散。$D$ 是扩散系数（单位：$\text{m}^2/\text{s}$），$\partial c / \partial x$ 是浓度梯度。负号说的是：浓度向右增大时，物质向左流动。

这个定律是经验性的（就像牛顿冷却定律、欧姆定律），由 Adolf Fick 在1855年类比傅里叶热传导定律提出。它在稀溶液中的简单扩散过程中是非常好的近似。

把 Fick 第一定律代入连续性方程：

$$
\frac{\partial c}{\partial t} = -\frac{\partial}{\partial x}\left(-D\frac{\partial c}{\partial x}\right)
$$

如果 $D$ 是常数（与位置和浓度无关——对稀溶液通常成立）：

$$
\frac{\partial c}{\partial t} = D\frac{\partial^2 c}{\partial x^2}
$$

**这就是 Fick 第二定律。** 它也叫扩散方程（diffusion equation），在热传导的语境下叫热方程（heat equation）——数学结构完全一样，只是 $c$ 换成温度 $T$，$D$ 换成热扩散系数 $\alpha$。

让我把推导的逻辑链再画一遍，确保你看到的是一幅完整的建筑蓝图：

```
物质守恒（基本公理）
    │
    ▼
微元体质量平衡：流入 - 流出 = 累积
    │
    ▼
Δx → 0：连续性方程 ∂c/∂t = -∂J/∂x
    │
    ▼
代入 Fick 第一定律：J = -D ∂c/∂x
    │
    ▼
Fick 第二定律：∂c/∂t = D ∂²c/∂x²
```

每一步都是清清楚楚的物理或数学。没有魔法。没有天降公式。

## 二、逐项解读——每一个符号都在说话

### 2.1 左边：$\partial c / \partial t$ ——累积速率

这是浓度在**固定位置 $x$**随时间变化的速率。

想象你在膜内的某一点放了一个极其微小的浓度传感器。传感器不动，它一直盯着同一个位置。它读数的变化速率就是 $\partial c / \partial t$。

- $\partial c / \partial t > 0$：该点浓度正在升高——溶质在此处累积。
- $\partial c / \partial t < 0$：该点浓度正在降低——溶质在此处被耗散。
- $\partial c / \partial t = 0$：该点浓度不随时间变化——稳态！

### 2.2 右边：$D\,\partial^2 c / \partial x^2$ ——扩散驱动的净通量变化

这是方程中最需要仔细理解的部分。

$\partial^2 c / \partial x^2$ 是浓度对位置的**二阶导数**。一阶导数 $\partial c / \partial x$ 告诉你浓度曲线的"斜率"；二阶导数告诉你浓度曲线的**"弯曲程度"**——曲线是向上凸还是向下凹。

让我通过三种情形来建立直觉：

**情形1：浓度分布是一条直线（$\partial^2 c / \partial x^2 = 0$）。**

```
浓度 c
  │  \
  │   \        斜率恒定 → 通量恒定
  │    \       流进某点的 = 流出某点的
  │     \      → 没有累积 → ∂c/∂t = 0
  │      \
  └──────────── x
```

梯度处处相等。流进微元体的通量和流出的通量一样——没有净累积。浓度不随时间变化。**线性浓度分布已经是稳态。**

**情形2：浓度分布向上凸（$\partial^2 c / \partial x^2 < 0$——像一座山丘）。**

```
浓度 c
  │     ╱╲
  │    ╱  ╲     山顶处：左右两边都比你低
  │   ╱    ╲    物质向两边扩散流走
  │  ╱      ╲   → 浓度下降 → ∂c/∂t < 0
  └──────────── x
```

山丘顶部的浓度高于两侧。物质从高浓度处向两侧扩散流走。流出比流入多。浓度下降。**山丘在被"铲平"。**

**情形3：浓度分布向下凹（$\partial^2 c / \partial x^2 > 0$——像一个碗）。**

```
浓度 c
  │ ╲      ╱
  │  ╲    ╱     碗底处：两边都比你高
  │   ╲  ╱      物质从两边扩散流入
  │    ╲╱       → 浓度上升 → ∂c/∂t > 0
  └──────────── x
```

碗底的浓度低于两侧。物质从两侧向碗底扩散聚集。流入比流出多。浓度升高。**碗在被"填平"。**

这就是 Fick 第二定律的灵魂：

> **扩散的本质作用是"抹平"浓度分布。哪里凸出就削平它，哪里凹陷就填满它。最终一切趋于均匀。**

浓度曲线的弯曲程度（二阶导数）**精确地度量了扩散在那一点产生的"抹平"力度**。弯曲越厉害，抹平越快。

### 2.3 扩散系数 $D$ 的角色

$D$ 是一个正的常数，量纲为 $\text{m}^2/\text{s}$（或 $\text{cm}^2/\text{s}$）。它衡量分子扩散的"快慢"。

- $D$ 越大：扩散越快，"抹平"的速度越快。小分子在水中的 $D$ 大约是 $10^{-9}\,\text{m}^2/\text{s}$，蛋白质大约是 $10^{-11}\,\text{m}^2/\text{s}$——蛋白质扩散比小分子慢两个数量级。
- $D$ 的量纲 $[\text{长度}^2/\text{时间}]$ 暗示了扩散的标度律：扩散距离 $\sim \sqrt{Dt}$。要扩散两倍远，需要四倍的时间。这就是为什么膜越厚，传质越慢——而且是**平方**关系，不是线性关系。

### 2.4 用一段话总结整个方程

$$
\underbrace{\frac{\partial c}{\partial t}}_{\text{浓度在某一点的时间变化率}} = \underbrace{D}_{\text{分子扩散的快慢}} \cdot \underbrace{\frac{\partial^2 c}{\partial x^2}}_{\text{浓度曲线在该点的弯曲程度}}
$$

浓度曲线越"弯"（偏离直线越远），变化就越快。扩散系数越大，变化就越快。方向总是"抹平弯曲"——大自然讨厌浓度不均匀。

## 三、这是什么方程？——PDE 的定义

### 3.1 从 ODE 到 PDE

让我们停下来，正式认识一下我们刚刚遇到的新物种。

**常微分方程（ODE）** 包含一个自变量的未知函数及其导数。比如 $y'(t) + ky(t) = q$——$y$ 只依赖于 $t$。

**偏微分方程（PDE）** 包含**多个**自变量的未知函数及其**偏导数**。比如 $\frac{\partial c}{\partial t} = D\frac{\partial^2 c}{\partial x^2}$——$c$ 同时依赖于 $x$ 和 $t$。

这个差别意味着什么？

在 ODE 的世界里，你追踪的是一个量沿**一条路径**的演化——一根时间轴，或一条空间轴。解是一条曲线。

在 PDE 的世界里，你追踪的是一个量在**整个区域**上的分布如何演化——浓度在整张膜内部、在所有时刻的值。解是一个**曲面**（如果有两个自变量的话）。

```
ODE:                          PDE:
y                              c(x,t)
│                              │  ╱╲
│   ╱──                        │ ╱  ╲     ← 浓度随 x 的分布
│  ╱                           │╱    ╲        在不同时刻的快照
│ ╱  ← 一条曲线               ├───────── x
├────── t                      │  t₁  t₂  t₃
                               t
      一维的故事                   二维的故事
```

### 3.2 PDE 方程的阶

和 ODE 一样，PDE 的**阶（order）** 是方程中出现的最高偏导数的阶数。

Fick 第二定律含有 $\partial^2 c / \partial x^2$（二阶偏导数），所以它是**二阶PDE**。

连续性方程 $\partial c / \partial t = -\partial J / \partial x$ 只含一阶偏导数——它是一阶PDE（但如果代入 $J = -D\,\partial c/\partial x$ 后就变成二阶了）。

## 四、PDE 的三大家族——扩散、波、平衡

### 4.1 三种方程，三种物理

二阶线性PDE有三大经典类型，每一种对应一大类物理现象。你需要知道它们，因为它们代表了大自然中三种截然不同的"性格"。

**（一）抛物型（Parabolic）——扩散/热传导**

$$
\frac{\partial u}{\partial t} = D\frac{\partial^2 u}{\partial x^2}
$$

代表方程：**扩散方程 / 热方程**。

物理性格：**弥散、平滑、不可逆。** 初始的尖锐特征随时间被抹平。信息扩散开来，细节逐渐丢失。浓度高峰被削平，温度差异被拉平。时间有方向——你可以区分过去和未来（热力学第二定律！）。

膜科学中的出现：膜内溶质扩散的瞬态过程、浓差极化的建立过程、膜污染初期的扩散吸附。

**（二）双曲型（Hyperbolic）——波动/传播**

$$
\frac{\partial^2 u}{\partial t^2} = a^2\frac{\partial^2 u}{\partial x^2}
$$

代表方程：**波动方程**。

物理性格：**传播、振荡、可逆。** 信息以有限速度传播。扰动不会弥散——它保持形状向前跑。如果你把时间反过来，物理规律仍然成立（至少在理论上）。

膜科学中偶尔出现：压力波在流体管路中的传播、膜振动（用于膜污染控制的超声波/脉冲流）。但对大多数膜传质问题不是主角。

**（三）椭圆型（Elliptic）——稳态/平衡**

$$
\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0
$$

代表方程：**Laplace 方程**。

物理性格：**平衡、光滑、无时间。** 方程里根本没有 $t$！它描述的是已经达到稳态的系统——所有的瞬态过程已经结束，一切不再变化。解极其光滑（无穷次可微），内部值完全由边界条件决定。

膜科学中的出现：二维膜内的稳态浓度分布、平板膜片的稳态传质模拟。

### 4.2 一张表格

| 类型 | 代表方程 | 物理原型 | 时间行为 | 膜科学角色 |
|---|---|---|---|---|
| 抛物型 | $u_t = Du_{xx}$ | 扩散 / 热传导 | 单调趋近平衡 | **核心** |
| 双曲型 | $u_{tt} = a^2u_{xx}$ | 波动 / 振动 | 振荡传播 | 次要 |
| 椭圆型 | $u_{xx} + u_{yy} = 0$ | 稳态 / 平衡 | 无时间演化 | 稳态分析 |

### 4.3 分类的数学判据（简述）

对于一般的二阶线性PDE

$$
Au_{xx} + 2Bu_{xy} + Cu_{yy} + \text{低阶项} = 0
$$

分类取决于判别式 $\Delta = B^2 - AC$：

- $\Delta > 0$：双曲型
- $\Delta = 0$：抛物型
- $\Delta < 0$：椭圆型

对于扩散方程 $u_t = Du_{xx}$，可以写成 $Du_{xx} - u_t = 0$。这里 $A = D$，$C = 0$（没有 $u_{tt}$），$B^2 - AC = 0$：抛物型。

对于波动方程 $u_{tt} = a^2 u_{xx}$，即 $a^2 u_{xx} - u_{tt} = 0$，$A = a^2$，$C = -1$，$B = 0$，$\Delta = 0 - a^2(-1) = a^2 > 0$：双曲型。

对于 Laplace 方程 $u_{xx} + u_{yy} = 0$，$A = 1$，$C = 1$，$B = 0$，$\Delta = -1 < 0$：椭圆型。

你不需要记住这个判据——但知道它存在是有用的。分类的核心意义不在于数学形式，而在于它预测了**解的行为特征**：是弥散、是传播、还是平衡。

## 五、PDE 需要什么信息？——初始条件和边界条件

### 5.1 从"两个常数"到"两个函数"

还记得吗？一个二阶ODE的通解包含两个待定常数。你需要两个条件（两个初始值，或两个边界值）来确定它们。

PDE 的情况远比这复杂。Fick 第二定律 $c_t = Dc_{xx}$ 的"通解"不是包含两个常数——它包含**任意函数**。确定解需要的不是几个数字，而是**整条曲线**的信息。

具体来说，要唯一确定扩散方程在区域 $0 \le x \le L$，$t \ge 0$ 上的解，你通常需要：

**（A）一个初始条件（Initial Condition, IC）：** 在 $t = 0$ 时整个空间域上的浓度分布。

$$
c(x, 0) = f(x), \quad 0 \le x \le L
$$

$f(x)$ 是一个**函数**（不是一个数！）。它是系统在一切开始之前的完整快照。

**（B）两个边界条件（Boundary Conditions, BCs）：** 在每一端（$x = 0$ 和 $x = L$），对所有时间 $t > 0$ 的约束。

$$
\text{在 } x = 0: \quad \text{某种关于 } c \text{ 或 } \frac{\partial c}{\partial x} \text{ 的条件}
$$

$$
\text{在 } x = L: \quad \text{某种关于 } c \text{ 或 } \frac{\partial c}{\partial x} \text{ 的条件}
$$

这些可以是 Dirichlet、Neumann 或 Robin 型（第29课的内容——一切都衔接上了！）。

### 5.2 为什么 PDE 比 ODE 需要更多信息？

因为 PDE 有**两个独立变量**。

在一个纯时间的 ODE 问题中，你只需要告诉它"从哪里出发"——一个时间点的信息就够了，物理定律接管剩下的一切。

在一个时间-空间的 PDE 问题中，你需要告诉它两样东西：

1. **你从哪里出发**（初始条件——$t = 0$ 时整个空间域的状态）
2. **你的空间边界发生了什么**（边界条件——整段时间内两端的约束）

如果你只给初始条件而不给边界条件——物质会扩散到边界上然后怎么办？反弹回来？消失？继续往外走？你不说，方程不知道。

如果你只给边界条件而不给初始条件——系统从什么状态开始？一团浓度峰值？均匀分布？你不说，方程也不知道。

**IC 和 BC 合在一起，才构成一个"适定问题（well-posed problem）"——有唯一解的问题。**

### 5.3 ASCII 全景图

```
               t
               ▲
               │   IC + BC → 唯一解 c(x,t)
               │
    BC at x=0  │╔════════════════════╗  BC at x=L
    (所有t>0)  │║                    ║  (所有t>0)
               │║   求解区域         ║
               │║   c(x,t) = ?      ║
               │║                    ║
               │╚════════════════════╝
               └──────────────────────► x
               0    IC: c(x,0) = f(x)    L
```

底边是初始条件（一条完整的浓度分布曲线），两侧是边界条件（贯穿整个时间的约束），求解区域是这个矩形内部所有 $(x,t)$ 点处的 $c$ 值。

## 六、一个完整的例题——膜内扩散的瞬态响应

### 6.1 问题设定

一张厚度为 $L$ 的致密膜。初始时刻膜内溶质浓度为零：$c(x,0) = 0$。在 $t = 0$ 时刻，突然将膜的左表面浓度升高到 $c_0$（阶跃扰动），右表面维持在零浓度。

$$
\frac{\partial c}{\partial t} = D\frac{\partial^2 c}{\partial x^2}, \quad 0 < x < L, \quad t > 0
$$

$$
c(x, 0) = 0 \quad (\text{IC})
$$

$$
c(0, t) = c_0, \quad c(L, t) = 0 \quad (\text{BCs, Dirichlet-Dirichlet})
$$

这正是膜科学中"膜从干态启动"的数学模型。

### 6.2 分解策略：稳态 + 瞬态

我们在第26课学到了一个强大的思想：通解 = 稳态解 + 瞬态解。它在 PDE 中同样适用。

**稳态解 $c_s(x)$** 是 $t \to \infty$ 时的极限——令 $\partial c / \partial t = 0$：

$$
D\frac{d^2c_s}{dx^2} = 0, \quad c_s(0) = c_0, \quad c_s(L) = 0
$$

这是第29课的例题1！解是：

$$
c_s(x) = c_0\left(1 - \frac{x}{L}\right)
$$

一条从 $c_0$ 到 $0$ 的直线。

**瞬态偏差 $v(x,t) = c(x,t) - c_s(x)$** 满足：

$$
\frac{\partial v}{\partial t} = D\frac{\partial^2 v}{\partial x^2}
$$

$$
v(0,t) = 0, \quad v(L,t) = 0 \quad (\text{齐次边界条件!})
$$

$$
v(x,0) = c(x,0) - c_s(x) = 0 - c_0\left(1 - \frac{x}{L}\right) = -c_0\left(1 - \frac{x}{L}\right)
$$

瞬态部分的边界条件变成了**齐次的**（两端都是零）——这大大简化了问题，因为齐次边界条件允许我们使用分离变量法（下一课的主题！）。

### 6.3 最终解的形式（预告）

在下一课中，你将学到用分离变量法和傅里叶级数来解这个瞬态部分。最终结果是：

$$
c(x,t) = c_0\left(1 - \frac{x}{L}\right) - \sum_{n=1}^{\infty}\frac{2c_0}{n\pi}\sin\left(\frac{n\pi x}{L}\right)\exp\left(-\frac{n^2\pi^2 Dt}{L^2}\right)
$$

第一项是稳态解。求和项是瞬态——由无穷多个衰减的正弦波叠加而成。每个正弦波以指数速率衰减，衰减速率正比于 $n^2$——高频分量（小尺度波动）消失得最快，低频分量（大尺度趋势）最后消失。

这就是扩散方程的典型行为：**快速抹平细节，缓慢消除大尺度不均匀。**

虽然我们还没有推导这个解，但你可以先用 Python 看看它长什么样子：

```python
import numpy as np
import matplotlib.pyplot as plt

L, D, c0 = 1.0, 1e-2, 1.0
x = np.linspace(0, L, 200)
N_terms = 50  # 傅里叶级数截断

fig, ax = plt.subplots(figsize=(9, 5))
for t_val in [0, 0.5, 2, 5, 10, 50]:
    c = c0 * (1 - x/L)   # 稳态部分
    for n in range(1, N_terms+1):
        coeff = 2*c0 / (n*np.pi)
        c -= coeff * np.sin(n*np.pi*x/L) * np.exp(-n**2*np.pi**2*D*t_val/L**2)
    ax.plot(x/L, c/c0, label=f't = {t_val}')

ax.set_xlabel('x / L'); ax.set_ylabel('c / c₀')
ax.set_title('膜内扩散瞬态响应：从零到稳态')
ax.legend(); ax.grid(alpha=0.3)
plt.tight_layout(); plt.show()
```

你会看到：$t = 0$ 时膜内浓度为零（平坦的底线）。随着时间推移，溶质从左侧渗入，浓度分布逐渐从零"长"成一条斜线。最终趋近于稳态直线 $c_0(1 - x/L)$。整个过程就像墨水从纸的一端慢慢浸透到另一端。

## 七、扩散方程的标度分析——一个无价的直觉工具

### 7.1 扩散时间尺度

不需要解方程，单凭量纲分析就能得到一个极其重要的结论。

方程 $c_t = Dc_{xx}$ 中，$[c_t] = [c]/[t]$，$[c_{xx}] = [c]/[x]^2$。量纲平衡要求：

$$
\frac{[c]}{[t]} = D \cdot \frac{[c]}{[x]^2} \quad \Longrightarrow \quad [t] = \frac{[x]^2}{D}
$$

如果特征长度是 $L$，那么扩散的**特征时间**是：

$$
\tau_{\text{diff}} = \frac{L^2}{D}
$$

这就是膜传质中的**扩散时间尺度**。它告诉你：溶质扩散穿过厚度为 $L$ 的膜大约需要多长时间。

数值感受：对于一张 $L = 100\,\mu\text{m} = 10^{-4}\,\text{m}$ 的膜，小分子 $D = 10^{-9}\,\text{m}^2/\text{s}$：

$$
\tau_{\text{diff}} = \frac{(10^{-4})^2}{10^{-9}} = \frac{10^{-8}}{10^{-9}} = 10\,\text{s}
$$

十秒。这意味着在阶跃扰动后大约几十秒（几个 $\tau_{\text{diff}}$），膜内浓度分布基本达到稳态。

如果膜厚 $L = 1\,\text{mm} = 10^{-3}\,\text{m}$：

$$
\tau_{\text{diff}} = \frac{(10^{-3})^2}{10^{-9}} = 1000\,\text{s} \approx 17\,\text{min}
$$

膜厚增加10倍，达到稳态的时间增加**100倍**（因为是 $L^2$）！这就是为什么超薄复合膜（活性层只有几十纳米）在工业上如此重要——薄意味着快。

### 7.2 与搅拌罐时间常数的对比

还记得第26课中搅拌罐的时间常数 $\tau = V/Q$ 吗？那是一阶ODE的时间尺度——线性地依赖于"系统大小"$V$。

扩散方程的时间尺度 $\tau = L^2/D$ 是**二次方**地依赖于长度 $L$。这种平方关系是扩散过程的标志性特征——它反映了布朗运动的随机行走本质。

| | 搅拌罐（ODE） | 扩散（PDE） |
|---|---|---|
| 时间尺度 | $\tau = V/Q$ | $\tau = L^2/D$ |
| 对"大小"的依赖 | 线性 | 二次方 |
| 物理原因 | 对流冲刷 | 随机行走扩散 |

## 八、源项的加入——扩散-反应方程的瞬态版

### 8.1 如果微元体内有化学反应

回到我们推导 Fick 第二定律的微元体。如果微元体内发生化学反应（消耗溶质），质量平衡变成：

$$
\text{流入} - \text{流出} - \text{反应消耗} = \text{累积}
$$

如果反应是一阶的（速率 = $kc$），那么微元体内单位时间消耗的质量是 $kc \cdot A\Delta x$。重新走一遍推导：

$$
\frac{\partial c}{\partial t} = D\frac{\partial^2 c}{\partial x^2} - kc
$$

这就是**扩散-反应方程的瞬态版**。第29课中的 $Dc'' - kc = 0$ 是它的稳态特例（$\partial c / \partial t = 0$）。

更一般地，如果有源项 $S(x,t,c)$（反应、产生、外部注入等），方程变成：

$$
\frac{\partial c}{\partial t} = D\frac{\partial^2 c}{\partial x^2} + S(x,t,c)
$$

物理含义清澈如水：

$$
\text{累积} = \text{扩散带来的净输入} + \text{源/汇}
$$

### 8.2 稳态 vs. 瞬态——一张全景图

```
                ┌─────────────────────────────────┐
                │   ∂c/∂t = D ∂²c/∂x² + S        │  ← 最一般的扩散-反应 PDE
                │                                 │
                │         │                       │
                │         ▼                       │
                │   令 ∂c/∂t = 0                  │
                │         │                       │
                │         ▼                       │
                │   D d²c/dx² + S = 0             │  ← 稳态 BVP (ODE!)
                │   (第29课的世界)                 │
                └─────────────────────────────────┘
```

稳态 ODE 是 PDE 的"终极状态"——所有瞬态耗散完毕之后的极限。PDE 包含了 ODE 作为特例，同时描述了从初始条件到稳态的**完整旅程**。

## 九、一点历史与哲学

### 9.1 Fick、Fourier、与类比的力量

Adolf Fick 在1855年发表扩散定律时，他是一位26岁的德国生理学家。他的灵感来源非常直接：他直接类比了 Joseph Fourier 在1822年发表的热传导理论，把"温度"换成"浓度"，把"热导率"换成"扩散系数"。

这个类比至今仍然是科学中最成功的类比之一。扩散方程和热方程在数学上完全相同——同一个方程，同一类解，同一套方法。如果你学会了解扩散方程，你就自动学会了解热方程；如果你学会了热传导的有限元模拟，你就可以直接用来做膜传质的有限元模拟。

这是数学抽象的力量：**一旦你把物理抽象成方程，不同物理的方程可能是同一个方程。解同一个方程就是同时解决了所有的物理。**

### 9.2 PDE 是一个更大的世界

从 ODE 到 PDE 的跳跃，在难度上是非线性的。ODE 有一套完整的通解理论——对于线性ODE，你总能写出通解。PDE 没有这样的奢侈。PDE 的"通解"通常不存在或没有实际用处。取而代之的是一系列强大但各有局限的方法：分离变量、傅里叶级数、格林函数、拉普拉斯变换、数值方法。

但不要被吓到。你需要掌握的 PDE 远比你想象的少。对于膜科学研究者来说，90% 的情况你只需要掌握**一个方程**——扩散方程（及其变体）——和**一种方法**——分离变量法。这正是下一课要教你的。

## 十、精讲例题：验证一个解

**例题.** 验证函数

$$
c(x,t) = \frac{A}{\sqrt{t}}\exp\left(-\frac{x^2}{4Dt}\right)
$$

是扩散方程 $c_t = Dc_{xx}$ 的解（其中 $A$ 是常数，$t > 0$）。讨论这个解的物理含义。

**解：**

先算 $\partial c / \partial t$。令 $u = x^2/(4Dt)$，则 $c = At^{-1/2}e^{-u}$。

$$
\frac{\partial c}{\partial t} = A\left(-\frac{1}{2}t^{-3/2}\right)e^{-u} + At^{-1/2}e^{-u}\left(\frac{x^2}{4Dt^2}\right)
$$

$$
= \frac{A e^{-u}}{t^{3/2}}\left(-\frac{1}{2} + \frac{x^2}{4Dt}\right)
$$

再算 $\partial c / \partial x$ 和 $\partial^2 c / \partial x^2$。

$$
\frac{\partial c}{\partial x} = At^{-1/2}e^{-u}\left(-\frac{2x}{4Dt}\right) = -\frac{Ax}{2Dt^{3/2}}e^{-u}
$$

$$
\frac{\partial^2 c}{\partial x^2} = -\frac{A}{2Dt^{3/2}}e^{-u} + \frac{Ax}{2Dt^{3/2}}e^{-u}\cdot\frac{x}{2Dt} = \frac{A e^{-u}}{2Dt^{3/2}}\left(-1 + \frac{x^2}{2Dt}\right)
$$

现在验证 $c_t = Dc_{xx}$：

$$
Dc_{xx} = D \cdot \frac{Ae^{-u}}{2Dt^{3/2}}\left(-1 + \frac{x^2}{2Dt}\right) = \frac{Ae^{-u}}{2t^{3/2}}\left(-1 + \frac{x^2}{2Dt}\right) = \frac{Ae^{-u}}{t^{3/2}}\left(-\frac{1}{2} + \frac{x^2}{4Dt}\right)
$$

和 $c_t$ 完全一致。$\checkmark$

**物理含义：** 这个解叫做**基本解（fundamental solution）**或**高斯核（Gaussian kernel）**。它描述的是：如果在 $t = 0$ 时刻，在 $x = 0$ 处瞬间释放一团溶质（$\delta$ 函数初始条件——一个无穷窄、无穷高的浓度尖峰），那么溶质随后的扩散行为就是这个高斯钟形曲线，随着时间的推移越来越宽、越来越矮。

宽度正比于 $\sqrt{Dt}$——这就是 $\tau \sim L^2/D$ 标度律的直观来源。高度正比于 $1/\sqrt{t}$——越来越平缓。面积守恒——溶质总量不变，只是越铺越开。

这是扩散过程最纯粹的数学肖像。

## 十一、练习

**练习1（微元体推导练习）.** 从微元体质量平衡出发，推导**带有对流项**的扩散方程。假设除了 Fick 扩散外，溶质还以速度 $v$（常数）被流体"冲走"（对流）。即通量为：

$$
J = -D\frac{\partial c}{\partial x} + vc
$$

推导出**对流-扩散方程**：

$$
\frac{\partial c}{\partial t} + v\frac{\partial c}{\partial x} = D\frac{\partial^2 c}{\partial x^2}
$$

> **提示：** 和正文一样取微元体做质量平衡。连续性方程仍然是 $c_t = -\partial J/\partial x$。把 $J = -Dc_x + vc$ 代入，注意 $v$ 是常数，$\partial(vc)/\partial x = v\,\partial c/\partial x$。

**练习2（量纲分析）.** 一种蛋白质的扩散系数 $D = 6 \times 10^{-11}\,\text{m}^2/\text{s}$。

(a) 蛋白质扩散穿过 $L = 50\,\mu\text{m}$ 的膜需要大约多长时间？

(b) 扩散穿过 $L = 500\,\mu\text{m}$ 的膜需要多长时间？与(a)比较。

(c) 如果你希望蛋白质在10秒内扩散穿过膜，膜最厚不能超过多少？

> **提示：** (a) $\tau = L^2/D = (50 \times 10^{-6})^2 / (6 \times 10^{-11}) = 2.5 \times 10^{-9} / 6 \times 10^{-11} \approx 41.7$ s。 (b) $L$ 增大10倍，$\tau$ 增大100倍，约4170 s ≈ 70 min。 (c) $L = \sqrt{D\tau} = \sqrt{6 \times 10^{-11} \times 10} \approx 2.4 \times 10^{-5}$ m = 24 μm。

---

**练习3（验证解）.** 验证 $c(x,t) = e^{-\alpha^2 Dt}\sin(\alpha x)$ 是扩散方程 $c_t = Dc_{xx}$ 的解（$\alpha$ 是常数）。这个解在什么边界条件下有物理意义？

> **提示：** $c_t = -\alpha^2 D e^{-\alpha^2 Dt}\sin(\alpha x)$。$c_{xx} = -\alpha^2 e^{-\alpha^2 Dt}\sin(\alpha x)$。$Dc_{xx} = -\alpha^2 D e^{-\alpha^2 Dt}\sin(\alpha x) = c_t$。✓ 如果 $\alpha = n\pi/L$，那么 $c(0,t) = \sin(0) = 0$ 且 $c(L,t) = \sin(n\pi) = 0$——齐次 Dirichlet 边界条件。这就是分离变量法中出现的单个"模态"。

---

**练习4（PDE 分类）.** 将以下方程分类为抛物型、双曲型或椭圆型：

(a) $u_t = 3u_{xx}$

(b) $u_{tt} = 4u_{xx}$

(c) $u_{xx} + u_{yy} = 0$

(d) $u_t = u_{xx} - 5u$

(e) $u_{xx} - u_{yy} = 0$

> **提示：** (a) 抛物型（扩散型，$D = 3$）。 (b) 双曲型（波动型，$a = 2$）。 (c) 椭圆型（Laplace 方程）。 (d) 抛物型——这是带反应项的扩散方程，低阶项 $-5u$ 不影响分类（分类只看最高阶导数的系数结构）。 (e) 双曲型（$A = 1$，$C = -1$，$\Delta = 0 - (1)(-1) = 1 > 0$）。这其实就是波动方程，只是把时间变量叫成了 $y$。

**练习5（物理直觉）.** 不需要解方程——仅凭 Fick 第二定律的物理含义判断：

以下三种初始浓度分布，在之后的瞬间（$t = 0^+$），浓度在 $x = L/2$ 处是上升、下降、还是不变？

(a) $c(x,0) = c_0$（均匀分布），边界条件 $c(0) = c(L) = c_0$。

(b) $c(x,0) = c_0 \sin(\pi x / L)$（中间高、两端为零的拱形），边界 $c(0) = c(L) = 0$。

(c) $c(x,0) = 0$（膜内初始为零），边界 $c(0) = c_0$，$c(L) = 0$（阶跃扰动）。

> **提示：** 看 $\partial^2 c/\partial x^2$ 的符号。(a) 均匀分布 → $c_{xx} = 0$ → 不变。已经是稳态。(b) 拱形 = 向上凸 → $c_{xx} < 0$ → $c_t < 0$ → 下降。山丘在被削平。(c) 在 $x = L/2$ 处 $c = 0$ 而附近也基本为零……但左侧有 $c_0$ 的来源。此刻 $c_{xx}$ 实际上难以判断（初始条件在 $x = 0$ 处有间断）。物理上，溶质正从左侧涌入，但还没来得及到达中点。$c_t \ge 0$——浓度将开始上升，只是最初在中点处极为缓慢。真正的"前锋"需要扩散时间 $\sim (L/2)^2/D$ 才能到达。

## 课末沉思

这节课我们做了三件意义深远的事。

**第一，我们从一个微元体的质量平衡出发，通过让微元体趋于零的极限过程，让 Fick 第二定律自然诞生。** 你现在知道这个方程不是一个假设——它是物质守恒（公理）加 Fick 第一定律（经验律）的必然推论。每当有人说"用扩散方程建模"，你知道他们实际上只是在说"物质守恒 + 扩散正比于浓度梯度"。

**第二，我们理解了方程的每一项的物理含义。** $c_t$ 是累积速率，$Dc_{xx}$ 是扩散驱动力——浓度曲线的"弯曲程度"决定了浓度在那一点是增加还是减少。扩散的本质是抹平不均匀。

**第三，我们认识了 PDE 与 ODE 的根本区别。** PDE 有两个独立变量，需要初始条件和边界条件的联合约束才能确定唯一解。我们还了解了三大类 PDE 的分类及其物理性格。

下一课，我们将学习**分离变量法**——PDE 世界的第一把钥匙。它的核心思想惊人地简单：猜测 $c(x,t) = X(x)T(t)$，把一个 PDE 拆成两个 ODE。你已经知道怎么解 ODE——所以你实际上已经知道怎么解 PDE 了，只是还不知道而已。

> *"每一个偏微分方程的背后，都是一位物理学家对着一个微元体做的加减法——然后让微元体小到消失。方程从消失中诞生。"*
