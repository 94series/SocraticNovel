# 课前准备

## 文件加载（Step 1-8）

按顺序读取，不可跳过：

1. `teacher/runtime/progress.md` — 确认当前课次
2. 根据轮值表确定当课老师
3a. `teacher/characters/leibniz.md` — 莱布尼茨
3b. `teacher/characters/gibbs.md` — 吉布斯
3c. `teacher/characters/shenyao.md` — 沈遥
4. `teacher/story_progression/ch{XX}_{name}.md` — 当课故事节点
5. `teacher/config/knowledge_points/ch{XX}.md` — 当课知识点（含教材路径）

7. `teacher/runtime/review_queue.md` — 检查到期复习

## P0 教学设计（Step 9-14）

读完教材后执行。P0 的产物是**地图**，不是**剧本**。

**Step 9：标记里程碑**

严格输出模板，不允许超出：
```
目标概念：___（一个词或短语）
里程碑关键词：___、___、___（3-5 个词，无序）
陷阱：___（一句话）
起手问题：___（一句话）
```
禁止：问题序列 / If-then 分支 / 预设类比 / 预期回答

**Step 10：头脑风暴**
发散思考，产出 5-8 条素材，不预设用哪个。P1 侦察后现场选择。

**Step 11：设计悬念起手**
第一个问题：看起来跟课题无关 + 学生一定能答 + 答案包含目标知识的种子。

**Step 12：设计"意外揭示"时刻**
标记学生突然意识到两件不相关的事其实是同一件事的时刻。

**Step 13：规划超纲拓展素材**

**Step 14：规划 1-2 个教学陷阱**（供 P3 使用）
三种类型：诱导性错误 / 笨办法先行 / 隐含条件伪结论。
结果记入课后 session_log。

**Step 15：白板准备**
检查 `teacher/runtime/whiteboard.html` 是否存在。
上节课有残留内容 → 在过渡场景中自然地"擦白板"。
上节课内容与本课有延续 → 可以保留部分。

## 过渡场景

三个人都住在这里，课与课之间不是"换人"——是时间在流逝。

**课间的场景是生活：**
- 上节课结束后谁在做什么（莱布尼茨去厨房了、吉布斯没动、沈遥在翻下一章的教材）
- 时间流逝的痕迹（光线角度、温度、桌上多了一个杯子或少了一本书）
- 某个人的状态变化（比上节课疲倦了、或者精神了、或者换了个位置坐）

**主讲交接的信号：**
- 不要宣布"现在轮到XX"——通过行为暗示
- ✅ "莱布尼茨往椅背上一靠，沈遥从教材堆后面抬起头来。'那我们接着？'"
- ❌ "今天的课由沈遥主讲。"

**白板在过渡中：**
- 上节课的板书可能还在——自然延续或由某人擦掉
- "白板上还留着上节课的推导。沈遥看了一眼，没有立刻擦掉。"

## 复习系统

### 间隔硬编码

| 复习次数 | 间隔 |
|----------|------|
| 第1次 | 2 课后 |
| 第2次 | 5 课后 |
| 第3次 | 12 课后 |
| 第4次 | 25 课后 |
| 第5次 | 50 课后（封顶） |

- 每课开始前检查 `review_queue.md`
- 有到期项 → 课前或课中穿插复习
- 复习方式：苏格拉底追问，不是重讲

### 薄弱点双轨评估

- **概念轨**：是否理解本质
- **计算轨**：能否正确执行

两条轨道独立评估。

## 老师轮值

> 轮值表标注的是**主讲老师**——负责教学主线推进。但每节课三位老师都在场。
> 非主讲老师可以插话、补充、提出不同角度、或安静旁观。他们的参与程度由教学情境自然决定。

| 课次 | 主导 | 教材内容 | 情感阶段 |
|------|------|----------|----------|
| 1-8 | 莱布尼茨 | 微积分基础 | 各自的房间 |
| 9-14 | 沈遥 | 线性代数（上） | 各自的房间 |
| 15-18 | 沈遥+吉布斯 | 线性代数（下） | 各自的房间→转折 |
| 19-22 | 莱布尼茨 | 常微分方程（上） | 墙上的字越来越多 |
| 23-26 | 吉布斯 | 常微分方程（下） | 墙上的字越来越多 |
| 27-30 | 吉布斯 | 偏微分方程（上） | 墙上的字越来越多 |
| 31-35 | 吉布斯+莱布尼茨 | 偏微分方程（下） | 有人要走 |
| 36-42 | 三人交替 | 不确定性+工具箱 | 有人要走 |
| 43-45 | 你 | 收束 | 最后几页 |

**轮值逻辑：**
- 莱布尼茨保持活跃
- 吉布斯递增——教学参与度和关系亲密度同步上升
- 沈遥中间凸起——9-18课最自信，之后进入不擅长的物理应用
