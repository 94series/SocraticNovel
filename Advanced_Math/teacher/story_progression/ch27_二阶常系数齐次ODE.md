**对应 ODE 教材的二阶常系数齐次线性方程章节。吉布斯接管教学的第二周，节奏已经稳定。**

角色状态：
吉布斯教学。他的黑板字迹比莱布尼茨更克制，每行推导之间的间距一样宽。沈遥已经习惯了他的风格，不再试图抢话。你偶尔会回头看客厅墙上的旧推导——莱布尼茨画的那些图还在。

**必然节点：**

1. **课前 — 为什么从二阶开始**
吉布斯不会寒暄。他在黑板上写 ay'' + by' + cy = 0，然后说："一阶你们已经会了。真实系统——振动、衰减、扩散——几乎都是二阶或更高。"他用弹簧振子和RC电路举例，但不展开。

2. **课中 — 特征方程的三种情形**
假设 y = e^(rt)，代入得到特征方程 ar² + br + c = 0。判别式 Δ = b² - 4ac 决定三种情形：Δ > 0 过阻尼（两个不同实根，指数衰减之和），Δ = 0 临界阻尼（重根，te^(rt)项），Δ < 0 欠阻尼（共轭复根，振荡衰减）。复根情形回扣 ch23 的欧拉公式——e^(αt)(C₁cos βt + C₂sin βt)。吉布斯在这里停了一下，说："特征方程把微积分变成代数。这就是线性的力量。"

3. **课后 — 线性背后的代价**
你问他非线性系统怎么办。吉布斯看了你一眼："不能叠加。不能分解。每一步都要单独算。"他没有展开，但那个语气像是在说：等你们准备好了再说。
