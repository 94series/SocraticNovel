**对应 PDE 教材的分离变量法章节。所有知识的汇合点——吉布斯和莱布尼茨都知道这节课是高潮。**

角色状态：
吉布斯主导教学，莱布尼茨在旁边。今天两个人都提前到了。吉布斯在黑板上画了一个膜的示意图，莱布尼茨在旁边补上了坐标轴和边界标注。没有商量——像是已经排练过。

**必然节点：**

1. **课前 — 所有线索汇合**
"今天把之前的一切串起来。"吉布斯指黑板："Fick 第二定律——PDE。边界条件——Dirichlet。Fourier 级数——展开工具。现在用它们来解。"

2. **课中 — 分离变量法完整流程**
假设 c(x,t) = X(x)T(t)，代入扩散方程，分离出两个ODE：T'/T = D·X''/X = -λ（常数）。空间方程 X'' + (λ/D)X = 0 带 Dirichlet 边界条件 → 特征值问题 → 正弦特征函数 sin(nπx/L)。时间方程 T' = -λDT → 指数衰减。通解为无穷级数：c(x,t) = Σ Bₙ sin(nπx/L) e^(-D(nπ/L)²t)。初值条件决定 Fourier 系数 Bₙ。吉布斯在这里写满了一整面黑板，然后放下粉笔："这就是扩散方程的完整解。"

3. **课后 — 解的物理意义**
莱布尼茨课后在黑板前站了一会儿。他说："高阶模衰减得快——大的 n 对应短波长，扩散把短波长先磨掉。最后剩下的只有基模 sin(πx/L)。长期来看，所有细节都会消失，只剩下最简单的那个形状。"吉布斯在旁边点了点头，然后说："这就是为什么边界条件重要。最终形状是边界条件决定的。"

**机会节点：**
- 如果你在分离常数取正取负之间犹豫，莱布尼茨会说："物理要求浓度有界。取负，否则指数增长。"
- 如果你问"无穷级数真的能收敛吗"，吉布斯会说"满足初值条件的平方可积函数——Fourier 保证收敛"，然后建议你自己查 Dirichlet 条件
